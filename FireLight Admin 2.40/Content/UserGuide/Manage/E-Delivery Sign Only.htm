E-Delivery Sign Only [User Instructions]

E-Delivery Sign Only workflow is available if your organization has implemented the E-Delivery Sign Only feature.

E-Delivery Sign Only workflow includes e-signing of activity/application documents by the clients in the E-Delivery Sign Only parent case and associated child cases.

FireLight® supports one E-Delivery Sign Only process that drives what FireLight does with the case when it is received. Currently, cases can be client directed only. Client directed cases go directly to the clients and do not require an agent to approve the case.

Client Directed E-Delivery Sign Only Case

Client Directed E-Delivery Sign Only Case

  1. Clients in the parent and child cases receive the E-Delivery Sign Only email that indicates their documents are ready to be signed. The clients receive separate emails with the login Passcode if the passcode option is enabled. The E-Delivery Sign Only case is in Pending Signatures status.

    If the agent opens the case in the agent's view, the Pending Requests dialog shows the E-Delivery Sign Only pending requests for the parent and child cases. Optionally, the agent can click the Send Reminder and/or Send Passcode links to send reminders or passcodes (if passcode enabled) to the clients. The agent clicks Close.

  2. The client enters the Last 4 Digits of SSN/Government ID and their Birth Date or a Passcode sent to them via a separate email (if the passcode option is enabled), and clicks Enter to log in to FireLight.

  3. On the Electronic Signatures page, the client clicks one of the following:

    • Review & Sign button to review and sign the E-Delivery documents.

    • Contact Agent button to send a message to the agent.

    The client can click the Contact Agent button to open a message, enter message text for the agent and click Send to send the message to the agent.


  4. On the Signature page, client clicks each document to review and then selects the checkbox for I have reviewed and agree with the terms expressed within this document.

  5. On the Capture Electronic Signature page, client enters their Signer Full Name and City and signs by clicking the I Consent button, signing on the signature pad with a mouse or stylus, or signing with the Topaz signature tablet.

    NOTE: If the client clicks the I Decline button. The e-Delivery case is finished and in a status that went back to the carrier.

  6. On the Congratulations page, client clicks a button.

    • Client clicks Document Download button to download the E-Delivery Sign Only documents.

    • Client clicks OK button to exit FireLight. Client receives an email to download or print the completed E-Delivery Sign Only documents.

      Client clicks the URL and enters the Last 4 Digits of SSN/Government ID and Birth Date on the Welcome page and clicks Enter.

      Client clicks the Download All link to download all of the activity documents in a single PDF file.

      On the notification bar, client clicks an option to Open, Save menu, or Cancel the downloaded PDF documents.

  7. In the Download PDF Documents dialog, client clicks the Download All link.

    When download is complete, client clicks an option on the notification bar at the end of the screen to Open, Save menu, or Cancel the PDF documents. The notification is browser dependent.

  8. (Optional) The carrier has an option in FireLight Admin to show or hide the Request Print button. When the button is shown, the client can click the Request Print button on the top navigation bar to send a request to the carrier to provide a paper copy of the activity documents. This can be used instead of downloading the documents or in addition to downloading the documents. The following message appears. Client closes the browser window to exit FireLight.

  9. If the client does not Request Print, the client clicks the Log Off button on the upper right to log off FireLight.

  10. (Optional for Agent) In All Activities, agent clicks the link icon to view the statuses of the child cases.

    Agent clicks the View History link to view the audit History for the E-Delivery Sign Only case. Additional audits are in the child cases History.

    After the clients complete the case, the agent's view of the E-Delivery Sign Only case in FireLight displays the following message for the case. The agent can click Display/Print PDF on the Other Actions menu to print and save the activity documents.

Cancel E-Delivery

Cancel E-Delivery

Use this option when the client wants to cancel the electronic delivery. This is a terminal status and prevents further action by the agent or the clients on the case.

  1. The agent logs in to FireLight and opens the E-Delivery case the client wants to cancel.

  2. On the E-Delivery toolbar, the agent clicks Cancel E-Delivery to cancel the E-Delivery case.

  3. To proceed with the cancellation, the agent clicks Cancel E-Delivery on the Cancel E-Delivery Case dialog.

  4. The following message appears on the open E-Delivery case.

  5. FireLight sends a status back to the carrier letting them know that the client has elected to cancel the E-Delivery case.
