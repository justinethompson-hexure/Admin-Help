Share Activities [User Instructions]

This feature is available if your organization enabled user share in FireLight® Admin.

User share, all activities and single activity, allows you to share with a specific user.

Generally, an organization enables only one type of user share for a group.

  • Share all activities allows you to share all your activities with another user. The user is able to view and edit, if allowed, all activities of another user including creating an activity on behalf of the other user.
  • Single share one activity allows you to share one activity with another user. The user is able to view and edit, if allowed, a single activity of another user.

IMPORTANT! Each user must have access to FireLight and an email address in their Preferences.

Share Activities

  1. Click the Share Cases button on the toolbar to open the Share My Activities dialog.

  2. In the Search My Contacts field, type a user name with whom to share your activities and click the search button. To clear the user name search box, click the X to the right of the search button.

  3. To the right of a contact name, click a link for the level of case sharing with the contact:

    • Share Full Control gives the “share to” user full access to your activities, including the ability to create and save new activities for you if your group has share all activities enabled. If a single activity share, then the shared-to user cannot create an activity on your behalf.
    • Share Read Only gives the “share to” user read-only access to your existing activities. The user cannot edit a shared activity and is also unable to save the activity, send client fill and sign requests, and perform activity processes, such as locking, unlocking, checking out, and submitting activities. When the read-only access user views the case, any changes to data items that were initiated by a rule being triggered are not saved.

  4. FireLight sends an email message to the selected contact (user). The user clicks a link to accept the case sharing.

    The shared-to user receives a confirmation message that thanks them for using case share. The user can access their shared activities by clicking their user ID in the User list (see Filter Activities - User List for instructions) or by selecting a single shared activity in the Shared Activity Tab on the FireLight Home page.

Remove Share

  1. Click the Share Cases button on the toolbar to open the Share My Activities dialog.

  2. On the far right of a user’s name, click the Remove Share link to remove all activities case share with the selected user.

  3. Click OK on the confirmation dialog.

  4. The shared-to user receives an email similar to one of the following.

    • Single activity share: “[Your Name] has revoked your (Full Control or Read Only Control) access on the [activity name] for the user share request originally sent on [Date].”
    • All activities share: “[Your Name] has revoked your (Full Control or Read Only Control) access for all activities originally sent on [Date].”
