Export [User Instructions]

Users can export data in All Activities into a Comma Separated File (CSV), which can be used for further research. Users can export data from the default view or after performing a search.

On the toolbar in All Activities, click the Export button to download the data for all the activities to a .csv file.

Sample view of the downloaded .csv file from the information bar in the Chrome browser at the lower left of the screen:

The .csv file contains data in the following columns:

  • User

  • Activity Name

  • Product

  • Activity Type

  • Status

  • Last Action

  • Last Audit Entry

  • Created (date/time)
