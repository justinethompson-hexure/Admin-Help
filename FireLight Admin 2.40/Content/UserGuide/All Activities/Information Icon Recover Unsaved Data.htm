Information Icon – Recover Unsaved Data

In an activity, click the View button on an activity with an information icon in the title bar to open the Recover Unsaved Data – Action Required dialog. This occurs when an activity closes before saving changes.

On the dialog:

  • Click the Recover button to have the system recover the unsaved data from the last actions in the activity.
  • Click the Discard button to have the system discard the unsaved data from the last actions in the activity.
