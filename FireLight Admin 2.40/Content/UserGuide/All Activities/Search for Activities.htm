Search for Activities [User Instructions]

Search

In the activity list, the activity title bar displays the activity name, product, and status.

Do any of the following to find a saved activity.

  1. On the toolbar, click the Week, Month, Quarter, YTD, or All link to show only activities within that timeframe.

    NOTE: If you have visibility to multiple agents’ activities, click the search button next to the User list to open the User List dialog and select a user to find activities for the selected user. See Filter Activities - User List for instructions.

  2. In the search field, type a full or partial name of the activity and click the search button. To clear the search, delete the text in the search box and click the search button.

    NOTE: When you open an activity from the filtered search list, you can return to the filtered search list by closing the activity and returning to the All Activities page. FireLight® retains your search criteria for the remainder of the session or until you clear your search criteria.

  3. Click the Export button on the toolbar to download the search results to a .csv file. See Export.

Advanced Search

  1. Click the Advanced Search button to open the Advanced Search dialog.

  2. Select a date range: Week, Month, Quarter, YTD, or All.

  3. Select search options in the lists:

    User list includes the users' activities to which you have access. My Activities is the default User. If user sharing is enabled, the User list includes yourself (My Activities) and other users with whom you have shared activities (e.g., All Users, My Activities, User 1, etc.).

    Activity Type list includes activity types for which the selected user has access (e.g., All, Application, Illustration, Multiple Activities, etc.).

    Status list includes the various statuses for the activities (Any, Data Entry, Signatures, PreSubmit Review, Review Queue, Submit Requested).

    Sort list includes options to sort the search results by Date, Activity Name, or Product Name.

    Signature Type list includes options: All, e-Signature, or Wet Signature. This list provides the ability to search for cases with the FLI_DECLINED_ESIGN tag, or by accepting or declining e-Sign in FireLight. Declining e-signature in the FireLight UI causes the application to show up as "Wet Signature" in the search results.

  4. Enter search parameters in the applicable fields: Activity Name, Issue State, Policy Number, Product Name, Company ID, Environment, etc.

    NOTE: Search parameters are specific to the organization and may differ from the ones listed here. Any of the search items that are set within the List Profile in the Provider will display in the Advanced Search.

  5. Optionally enter a search date range in Begin Date and End Date fields.

  6. Click one of the following buttons:

    • Save – opens a Save Search dialog. Enter a Name and click Save to save the search options for future use. You need to save the search options before initiating the search. Click Cancel to cancel saving the search.

    • Search – initiates the search process and displays the search results on the right.

    • Cancel – cancels the search and closes the Advanced Search dialog.

    NOTE: When you open an activity from the filtered Advanced Search list, you can return to the filtered search list by closing the activity and returning to the All Activities page. FireLight retains your search criteria for the remainder of the session or until you click the Clear button on the right of the Advanced Search button on the toolbar. The Clear button become available after an Advanced Search.

    Click the Export button on the toolbar to export and download the search results to a .csv file. See Export.

Saved Search

  1. At the top of the page in the Saved Search list, select a saved search.

  2. In the Advanced Search dialog, click the Search button to initiate the search based on the saved settings. Click Cancel to cancel the search and close the dialog.

  3. Delete a saved search:

    1. Select a saved search in the Saved Search list.

    2. Click the Advanced Search button to open the saved search in the Advanced Search dialog.

    3. Click the Delete button and click Delete on the deletion confirmation dialog.
