Print Forms [User Instructions]

Print Forms activity is available if your organization has implemented the Print Forms feature.

The Print Forms activity is available only in Wizard format.

This feature allows a group of FireLight users to select and print forms within a package to a fillable PDF file. It also allows non-FireLight users to enter a URL in a browser to access and download a fillable PDF file.

NOTE: The Print Forms activity PDF files are fillable PDF files, which means the forms are editable outside of FireLight.

Print Forms Activity Users in FireLight

  1. On the Home page under Start New, click Print Forms or on the navigation bar, and then click New Activity > Print Forms.

  2. Select a Jurisdiction, Locale, Product Type\, and Organization (as applicable) from the drop-down lists. Print Forms is a Non-Insurance Product.

  3. Click Print Forms.

  4. Click Create.

    A Print Forms wizard opens.

    On the Print Forms navigation bar, a green check mark next to Alerts indicates FireLight® does not detect any validation errors. If the check mark is red, click Alerts to view errors and then correct the errors. When no validation errors exist, FireLight enables the Print button.

  5. Select or enter applicable inputs on the wizard.

  6. On the right of the Print Forms navigation bar, click Print.

  7. In the Select Documents to Print dialog, select the documents to print to a PDF file. Click the up or down arrow to customize the print order of the files. Click the Print Selected Documents button to print the documents to a PDF file.

  8. If applicable, save the documents to a location on the network or PC.

  9. In the Select Documents to Print dialog, click Close to close the dialog.

    NOTE: Print Forms cases are hidden and do not appear on the Home page or on the All Activities page. When you close a Print Forms case, you cannot access it again.

Other Users Outside of FireLight

If the organization added Form URL paths to their Print Forms library forms, users outside of FireLight can also access the fillable PDF files.
