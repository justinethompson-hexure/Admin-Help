Tele-Sign [User Instructions]

If your organization implemented the Tele-Sign feature, do the following to capture client signatures.

  1. Enter the appropriate data into the following Tele-Sign fields on the form or wizard:

    • Enable Tele-Sign – enter True
    • Tele-Sign App ID – enter any number (recommended at least 4 digits – must be unique per application)
    • Tele-Sign PIN – enter any 4-digit number (does not need to be unique)
    • Tele-Sign DOB – enter the signer’s birth date

  2. When the activity is complete, click CONTINUE to proceed to signatures.

  3. On the Electronic Signatures page, click Use E-Signature.

  4. In the List of Required Signers, click the non-agent signer(s) and click Proceed.

    NOTE: If the non-agent signer is more than one signer type on the form and the organization enabled the multi-signer type feature in FireLight® Admin, click each signer type button that applies and then click Proceed to begin a single signing process for both signer types. For example, the Insured can also be the Owner.

  5. On the Client Signature Choice page, click Tele-Sign.

  6. On the Tele-Sign Signer Information page:

    1. Enter the signer information for Signer Full Name, City, and State in the upper section.

    2. If the signer will receive an email with a link to the signed documents, enter the information for the Signer Email, Signer Birth Date and Signer Last 4 Digits of SSN/Government ID in the lower section.

    3. Click Continue.

  7. On the Tele-Sign: Form Access page, select a Delivery Type in the list (Download, Generate Link, or Send via Email). On the Form Access section, choose a document delivery method.

    NOTE: Your organization may not have all options available. A delivery method does not need to be selected to complete the Tele-Sign signature.

    • Download: In the Delivery Type list, select Download and click the Download Forms button to download a copy of the PDF application forms that can be mailed to the client. Click Continue.

    • Generate Link: In the Delivery Type list, select Generate Link, enter the signer information for Last 4 Digits of SSN/Government ID and Signer Birth Date, and click the Generate Link button to generate a URL link to give to the signer so they can view and download the application PDF file on the Web. The link text appears below the Generate Link button as shown below. Click Continue.

    • Send via Email: In the Delivery Type list, select Send via Email. Enter the Signer Name, Signer Email, Signer Last 4 Digits of SSN Government ID, and Signer Birth Date. Click the Send Email button as shown below. The client receives an email and clicks the URL link in the email to go to the URL in a web browser. Client enters the Last 4 Digits of SSN Government ID and their Birth Date to view and download the PDF application documents. Click Continue.

  8. On the Agent Instructions page, read the Tele-Sign script and follow the instructions to verify the signer’s identity and gain their Tele-Sign voice signature.

  9. (Optional) Click Cancel Tele-Sign to clear the Tele-Sign signature and start over.

  10. Click the Done button to continue.

    • If the Tele-Sign signature was completed, but declined, FireLight displays a message that the Tele-Sign was declined and returns to the application in Data Entry mode.

    • If the signature has not yet been completed, FireLight returns to the List of Required Signers page. A "(Tele-Sign in Progress)" indicator appears next to the client signature name for any signer types using Tele-Sign.

      • To return to the Agent Instructions page, click the client Tele-Sign signature button. Click Done on the Agent Instructions page to return to the List of Required Signers page.

      • To cancel a Tele-Sign signature capture, click the Cancel Tele-Sign link under the client signer name and click OK on the confirmation dialog.

    • If the Tele-Sign signature was completed and approved, FireLight returns to the application view if Review has been enabled. If Review has not been enabled, FireLight returns to the list of signers if additional signatures need to be collected, or the application can be submitted, either manually or automatically if your organization uses auto submit.

      NOTE: The Agent can sign in a default order, first or last, depending on what is set in the organization settings.
