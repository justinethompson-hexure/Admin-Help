Optional Features for Creating a New Activity [User Instructions]

NOTE: The following examples use Application for the new activity type.

  • If the organization has the Enable Resident State and Use Resident State in Form Selection options enabled in the product Sales Agreement in FireLight Admin, the selection in the Residence list determines the available form/wizard selections based on the resident jurisdiction instead of the issue jurisdiction. If the Use Resident State in Form Selection option is not enabled, form/wizard selection is based on the issue jurisdiction.

  • If the organization has the Require Entry of Activity Name group option enabled for the applicable role code, you are required to enter an activity name (instead of using the default activity name). A “Required” indicator appears in the Create [ActivityName] dialog. Enter a unique Application Name and click the Create button.

  • If the organization has the Case Access by Firm ID and the Enable All Activities User Share group options enabled in FireLight Admin for the applicable role code, you can select a user as the owner of the new activity in the On Behalf of list and then click the Create button.

  • If the organization has the Case Access by Firm ID and the Enable All Activities User Share group options enabled in FireLight Admin for the applicable role code, and you work with multiple offices or firms, select a user in the On Behalf of list and an office code in the Office list to use for the activity and then click the Create button. This sets the owner of the new activity and the activity’s Firm ID to the Firm Registration Number of the selected office or firm.

  • If the organization requires inclusion of a supplemental document (e.g., illustration, etc.) with the activity, a dialog appears.

    In the following example, the user must add an illustration to the application documents.

    Upload the required supplemental document (Illustration with Signature in this example) from the Other Actions menu in Documents. Select a Document Type and click the Choose File button to select and add the supplemental document. Click the Upload button. When finished uploading the required supplemental document, the Application Documents list includes the uploaded supplemental document and the supplemental document is included in the list of documents in Data Entry.
