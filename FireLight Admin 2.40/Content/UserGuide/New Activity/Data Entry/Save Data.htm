Save Data [User Instructions]

If you do not have all of the information to complete the forms, quotes, or wizards, you or FireLight Auto Save can save your progress and then return to the quote or package as often as necessary to complete the activity. FireLight® stores saved activities. Click an activity in the Recent Activity list on the Home page or click All Activities on the navigation bar to access the saved activities on the All Activities page.

Auto Save

The FireLight Auto Save functionality occurs behind the scenes but does not save the activity to the dashboard/Home page without user interaction where automatic save does not currently exist (Illustration and Quote). All other activities are saved automatically upon creation and do not prompt the user with a save dialog upon exiting as the auto save has captured all user inputs and selections. The auto save functionality allows the activity to be accessed from the FireLight Recent Activity list on the Home page and from All Activities. The activity audits reflect the Auto Save (username, number of changes made/saved within the duration) and Last Updated timestamp.

For activities that are auto saved, the navigation bar displays "Auto Save Enabled" in red font to the left of the buttons. In tablet mode, “Auto Save Enabled” will display to the right of the activity name.

NOTE: To copy an activity, open the Other Actions menu and click the Copy Activity option.

Save an Illustration or a Quote

To save an illustration, click Save on the navigation bar, enter the illustration Name in the Save Illustration dialog, and click Save.

To save a quote, click Save on the navigation bar, update the quote Name in the Save Quote dialog, and click Save.

The saved activity is added to the Recent Activity list on the Home page and to the All Activities page.

Close an Illustration or Quote Without Saving Data

If you close an illustration or quote activity without saving data, a dialog opens. Click Yes to save the activity, click Discard to discard the activity, or click Cancel to exit the dialog without saving changes.

If you close an illustration or quote activity from the Open Activities menu (if enabled for your organization in Admin), click Save to save the activity, click Discard to discard the activity, or click Cancel to exit the dialog without saving changes.

Save Guidelines

A hard save (permanently saving the currently filled session data with the activity) occurs when:

  • The Disable Save Activities option is disabled (cleared) in FireLight® Admin. For all activities other than Illustration and Quote, the FireLight Auto Save functionality automatically saves the activity upon input and/or selection while editing an activity (Print, Continue, Custom Action, app transfer, initiate Fill &Sign; Sign, uploading a document, when % complete gets to 100, switching between multiple open activities, etc.). "Auto Save Enabled" appears to the left of the top navigation bar in FireLight. The activity audits reflect the Auto Save (username, number of changes made/saved within the duration) and Last Updated timestamp.

    NOTE: Auto Save is disabled if the Disable Save Activities option is enabled in FireLight Admin.

  • User clicks Save on an Illustration or Quote activity.
  • User clicks Yes when FireLight prompts the user to save changes when leaving an unsaved Illustration or Quote activity (i.e., go Home, Logoff, etc.).
