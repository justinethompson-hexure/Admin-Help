Share [User Instructions]

If this feature is enabled for your user group in FireLight® Admin, you can share your active (unlocked) activity with one or more other users who have product access to the activity. Each user must have access to FireLight and an email address added to their Preferences. If Case Access By FirmID is turned on in FireLight Admin, only users who match the FirmID filtering is available for the share.

Guidelines

  • The Share option is available until the activity is locked.
  • For an Illustration activity, the illustration is locked when the illustration is linked to an application and the application is locked to begin signatures. An unsaved illustration cannot be shared.
  • For an e-Delivery parent activity, the Share option is available unless the parent activity is in a terminal status (Change Requested by Agent, Change Requested by Client [client directed workflow], Policy Canceled, Expired, or Complete).
  • The Share option is not available for e-Delivery child activities.
  • Only the original activity owner can share an activity with other users. The shared-to user cannot re-share the activity.
  • The shared-to user cannot transfer (give owner rights to) an activity; only the owner of the activity can transfer it.
  • The shared-to user can create a copy of the shared activity.
  • The user can remove the user share from an activity.
  • With autosave share with another user in FireLight, the message about another user editing is displayed only to the other user until the authentication timeout is reached. Once this time is reached, the user no longer sees a message about the other user potentially making edits/changes.

  • If the user gave “Share Full Control” to the shared-to user, the shared-to user can create an activity “On Behalf Of” the user by selecting the user’s name from the On Behalf of list during the creation of a new activity.

  • Selecting the Disable Share Full Control checkbox disables the user from sharing the activity with full control. When this setting is selected, the Full Control link is not available when the user shares an activity.

  • Selecting the Allow Share Limited Control checkbox enables the user to provide limited share options while sharing an activity with another user/agent. An admin can check both the group settings to allow full control and limited control to a user group. All four limited share options are available to the user, while only one of theoptions can be selected for an activity per user, and an Agent is able to revoke limited share access from another user.

Share an Activity

  1. Open the activity to be shared with one or more other users.
  2. On the Other Actions menu, click Share to open the Share My Activities dialog.

  3. In the Search My Contacts field, type a user name with whom to share your activities and click the search button. To clear the user name from the search box, click the X to the right of the search button.

  4. Click a link for the level of case sharing for the contact name in the list:

    • Share Full Control gives the “shared to” user the ability to edit and complete the activity.

    • Share Read Only gives the “shared to” user read only access to your existing activity.

  5. FireLight sends a user share request email message to the selected user. The user clicks a link to accept the case sharing.

  6. Share My Activities dialog displays the existing user shares and user share requests that are pending acceptance.

  7. To remove a user share, on the Other Actions menu click Share and click the Remove Share link in the Share My Activities dialog.

View an Activity Shared to You

  1. When a user shares an activity to you, you receive a user share request email message. Click the URL link within the message to accept the share request.

    FireLight displays a confirmation message that thanks you for using case share.

  2. Go to All Activities in FireLight and open the User List. Click a user name to view an activity that was shared to you by the user. In the following example, click mdavid (User Share) to view the activity that user “mdavid” shared with you. The User List dialog displays the five most recent users that you can select.
