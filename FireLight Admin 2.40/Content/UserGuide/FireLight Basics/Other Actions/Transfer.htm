Transfer [User Instructions]

If this feature is enabled for your user group, you can transfer any activities that are in Data Entry or can be unlocked, excluding e-Delivery, to other users. The other user will then own the activity.

  • Each user must have access to FireLight® and an email address.

  • If Case Access By FirmID is turned on in FireLight Admin, only users that match the FirmID filtering is available for the transfer.

  • E-Delivery activities cannot be transferred.

NOTE: Only the owner of the activity can transfer the activity to another user.

IMPORTANT!

  • If transferring to another agent after the activity is locked for signatures, users need to make sure that the transferred Agent information is entered on the activity to not cause any NIGO signatures issues.

  • FireLight disables the Transfer functionality when users click "Initiate DocuSign" for the signing ceremony. The Other Actions menu still shows "Transfer" as an option, and if users click Transfer from the menu, the Transfer dialog opens with a message stating: "Application cannot be transferred while signatures are in DocuSign. Unlock the application to enable application transfer."

  • With auto save transfer with another user in FireLight, the message about another user editing is only displayed to the other user until the authentication timeout is reached. Once this time is reached, the user no longer sees a message about the other user potentially making edits/changes.

  1. On the Home page in the Recent Activity grid click the activity to transfer, or on the All Activities page, locate the activity to transfer and click View.
  2. On the Other Actions menu, click Transfer to open the Transfer Activity dialog.

  3. In Search My Contacts, type the name of the user who will receive the activity and click the search button. NOTE: The User list is filtered based on product access. It displays only users that have product access to the product being transferred.

    Click a username in the list to begin the transfer process. The Transfer dialog displays a message: "[Activity Type] Transfer has started and is completed when the new owner receives the [Activity Type]." Click Close to close the dialog.

  4. In All Activities, the Last Audit Entry section displays “Activity Transfer to [user] initiated by [YourUserID]”. An audit entry also appears in the receiving user’s activity audit history.

  5. The activity will remain in your All Activities list until the transfer of ownership is complete.

    Until the transfer of ownership is complete, you can click the View button in All Activities to view the activity, but not edit it. The following message appears when you open the activity before the transfer completes.
