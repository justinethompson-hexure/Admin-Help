Recent Activity Tab [User Instructions]

NOTE: When a user is associated with multiple firms, the user has visibility and access to activities that were created under firms other than the firm for the current session. If the current session Firm ID/Firm Name is not the same as the Firm ID/Firm Name of an open activity, the system may present a popup message to the user, stating "This application was created and is submitted under firm XYZ and you are currently logged in under firm 123." The actual language is determined by each organization.

Activity Status and Last Update information appear below the Recent Activity name in the left pane. See Activity Display Statuses [User Instructions] for descriptions of the various statuses.

  1. On the left of the Home page in the Recent Activity tab, click an activity name to open an activity that you recently accessed.

    NOTE: If no activities exist yet, the Recent Activity tab displays a Create an Activity button instead of a list of activities.

  2. (Optional) In the View list, click an activity type to filter the list by activity.

  3. Click the numbered navigation buttons below the Recent Activity list to move through the list of recent activities.

NOTE: If the Disable View Activities option is enabled in FireLight® Admin, the Recent Activity list on the home page will provide only the status and date of recent activities in a read-only view. Clicking an activity in the list will not open the activity.

Post-Submission and Product Notification Icon [User Instructions]

Post-Submission Notification

In the Recent Activity list, for organizations that have post-submission polling services set up in the Provider (Back Office), when a polling service occurs and a notification is received for an activity, a post submission status "envelope"  icon appears for the activity on the Recent Activity list or on the right of an activity in the All Activities list. Click the icon to view the message in the message center. If you have the Send Message Center Emails checkbox selected in Preferences, you also receive the message via email.

The post-submission message contains the agent name, case name, status, and carrier along with any Provider-specific information, such as instructions for the agent. NOTE: Any references to the Provider are listed as "Back Office." Post-submission statuses include Carrier Review Complete, Approved (Carrier status), Declined (Carrier status), Further Review (Carrier status), and In Carrier Review. Once you open a post-submission status message, the envelope icon changes to , which indicates that no unacknowledged post-submission messages exist for the activity.

Product Notification

In the Recent Activity list, for products that have a product notification set up in FireLight Admin, when a product notification message is received for a locked activity, a notification “envelope” icon appears for the activity on the Recent Activity list or on the right of an activity in the All Activities list. Click the icon to view the message in the message center. If you have the Send Message Center Emails checkbox selected in Preferences, you also receive the message via email. When you open the activity, the Product Notification message appears. Click [Close] to close the message.

Once you open a product notification message, the envelope icon changes to , which indicates that no unacknowledged product notification messages exist for the activity.
