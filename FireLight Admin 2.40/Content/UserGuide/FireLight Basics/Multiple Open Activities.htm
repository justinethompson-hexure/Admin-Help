Multiple Open Activities [User Instructions]

If your organization enabled the Multiple Open Activities feature, you can have multiple activities open at the same time in the same session. An Open Activities (#) menu appears on the navigation bar. Create or open multiple activities and then go to a different open activity by selecting the name of another activity on the Open Actitivities (#) menu. (The number next to the menu name indicates the number of open activities.) When switching to another open activity, FireLight® auto-saves the current activity, except for Illustration and Quote activities, which need to be saved by using the Save number.

To close an open activity, click the X to the right of the open activity name on the Open Activities (#) menu. FireLight closes open activities when the session ends.
