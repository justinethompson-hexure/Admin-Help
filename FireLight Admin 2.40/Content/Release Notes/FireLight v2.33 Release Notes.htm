FireLight v2.33 Release Note

January 2024 Release—1/19/2024 Production

Enhancements

Review Queue – Reviewer Notification When Attachment Added to Locked Activity: DIST-1986

This feature will notify a reviewer who has locked an activity for review if the agent adds/removes an attachment/document to the app. The notification is sent via email.

Impact

  • Create a new email template to notify reviewer about document update in the activity. This email template is added under the email category “Review Queue” the “Email Template” page under the Org Config menu.
  • Send email to reviewer when document is updated for the activity locked in review.

Firelight Administration Tool - Changes for Ease-of-Use DIST-1896

This enhancement updates commonly used administration tool functionality to improve usability. This will enhance efficiency for users of the FireLight Administration tool.

Impact

  • Add links to directly access forms/wizards on the list boxes on page - FLAdmin > Packages > Forms/Wizards in Package (tab)
  • Add links to directly access rule sets on the list boxes on page - FLAdmin > Packages > Rule Sets in Package (tab)
  • Add a column for ACORD value in the grid on page - FLAdmin > Packages > Package Detail (tab)
  • Add links to directly access packages on the list boxes on page - FLAdmin > Sales Agreements > Sales Agreement Details (tab) > Packages (list box)
  • Add a column for CUSIP value in the grid on page - FLAdmin > Sales Agreements > Sales Agreement Details (tab) > Product (grid)
  • Add a column for CUSIP value in the grid on page - FLAdmin > Products > Product Details (tab) > Product (grid)
  • Display alert when duplicate package is created using Duplicate link - FLAdmin > Packages > Package Detail (tab)
  • Display alert when duplicate rule set is created using Duplicate link - FLAdmin > Rule Sets

Review Queue - Add New Filter Option to List All Unlocked Activities DIST-2072

Provide ability to filter review queues on two more options, such as, activities that are unlocked currently and activities that were not locked by any reviewer yet. Both these options/filters is available on all the queues and on Advanced Search dialog.

Impact

Added two more filters on the statuses dropdown on Review Queue and Advanced Search dialog to allow searching:

  • Unassigned Orders - Landed in the queue and not locked by any reviewer yet
  • Unlocked Orders - Was locked by a reviewer at least once and not locked currently

DTCC Fall Update Enhancements CR-906

These enhancements update APP/SUB to add new fields and cycles in accordance with the 2023 DTCC Fall Update.

Impact

  • Add additional PSE output cycle to APP/SUB
  • APP/SUB Record Layout updated to match September update.
  • Add new fields to match ACORD 951e form.

LexisNexis Instant ID Enhancements CR-1226

These enhancements allow LexisNexis Instant ID and Instant ID Q&A integrations to be called independently and to be included in the same workflow.

Impact

  • Add unique data group properties/response types for Wizards/Forms Designers.
  • Add custom action parameter for OTP confirmation button.

Map User First/Middle/Last Name in SSO to Agent Preference DIST-1216

This enhancement will map agent first, middle, and last name value from SSO login to the agent preference fields. Currently, when agent logs in using SSO, only the full name is mapped. With this enhancement agent first, middle, and last name will also be mapped to the system data items.

Impact

  • Agent name fields on preferences section in the FL App.
  • Fields with the values from 1228 file is populated in below data items:
    • Agent_FirstName
    • Agent_MiddleName
    • Agent_LastName

Activity Reports - Changes to Abandoned Activity Admin Report DIST-879

Four new columns will now be available to the existing admin report "Abandoned Activity Report." These new columns can be enabled for clients who request them.

Impact:

  • Below columns can now be made available to Abandoned Activity Admin Report, in the CSV format.
    • Agent Carrier ID (Data Item ID - Agent_CarrierID)
    • Agent Full Name (Data Item ID - Agent_Full_Name)
    • Percent Complete (Data Item ID - FLI_PERCENT_COMPLETE)
    • Is Deleted

Visualization

The new Visualization tool interface optimizes the use of viewable areas for the graphical presentation of policy values in individual product and product comparison scenarios to enhance communication in the sales process and facilitate client decision-making.

  • Provide the ability to compare a set of user-selected products graphically based on various presentations of Premium, Cash Value, Death Benefit, IRR, etc.
  • Continue to allow clients to leverage existing product presentations. When an individual product is selected, any proprietary presentation materials/graphics is accessible in the new visualization interface.
  • Provide for an easy transition to comparisons with competing/ complementary products.

Firelight Administration Tool -Package Effective Date EAES-2111

The Package Detail screen includes an ability to specify the start and end date of the package. It will behave like the start and end dates for Products, Rule Sets, Forms, Wizards, and Sales Agreements. If nothing is entered for the start date or end dates, the defaults is the same as the defaults for their respective dates on Products (e.g., 1Jan1900 / 31Dec9999)

Embedded - Case Sharing EAES-1043

Firelight embedded experience is enhanced so that the features for Case Sharing are implemented as well. This will include the following:

  • Single Activity Share
  • Share Request Users
  • Share Control Request
  • Delete Share Control
  • Share All Control

Embedded - Copy Activities EAES-1117

Firelight embedded experience is enhanced to include the Copy As Is / Copy with Changes features that are available in native FireLight. Copy As Is / Copy With Changes is implemented via FireLight’s REST API.

Embedded - Optional Forms EAES-1045

Firelight embedded experience is enhanced to include the Optional Forms features that are available in native FireLight. Copy As Is / Copy With Changes is implemented via FireLight’s REST API.

Embedded - Pre-Submit Review Queue EAES-1046

Firelight embedded experience is enhanced to include an option, if enabled, to allow Cases to be automatically submitted to a Pre-Submit Review Queue prior to submission. Functionality includes the ability to specify the reviewer.

Passcode Updates EAES - 811

Several enhancements to Passcodes

  • Passcodes will show an 8-digit code
  • Passcodes will show numbers only
  • All Passcodes is the same, regardless of delivery medium.
