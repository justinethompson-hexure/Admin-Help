Export a Form

Export (save) a form with all of its metadata from FireLight® Admin if the form is used in other regions within your environment, such as Staging, Production, Quality Engineering, etc., or if planning to use the same form multiple times; for example, when requiring a proof of citizenship form for joint or additional insured.

NOTE: Ensure pop-ups are not blocked for Admin in the web browser. Otherwise, the export feature will not work.

  1. On the Home page or navigation bar, click Designer/Form Library.
  2. Select a form in the form grid.

  3. In the lower center of the Form Detail tab, click Export.

  4. The form opens in a separate web browser window in PDF format. If a notification bar appears at the bottom of the page instead, click the Save button arrow, click Save As, select a location to save the file and click Save.

  5. Save a copy of the form to a location on the PC or network:

    1. Click the PDF viewer Save button to open a Save As dialog.

    2. Browse to the folder in which to store the form.

    3. Enter a form name and click Save.

    4. Close the form display browser tab or window when finished.
