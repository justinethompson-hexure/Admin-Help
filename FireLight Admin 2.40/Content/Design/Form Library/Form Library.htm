Form Library

On the navigation bar, click Designer/Form Library to go to the Form Library page. Use the Form Library page to create and manage forms.

Begin by uploading forms and adding form detail. Then lock the form for editing and go to the Form Designer to add form fields and rules. Return to the form in the Form Library and unlock the form so other administrative users can edit it.
