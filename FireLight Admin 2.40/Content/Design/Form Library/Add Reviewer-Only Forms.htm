Add Reviewer-Only Forms

Add required and optional reviewer forms to be filled in by reviewers in the Review Queue.

  1. Upload the reviewer forms to the Form Library page and fill in the applicable Form Detail.

  2. On the Form Designer, open the form.

    • In Form Properties, set Protection to Public.
    • In every field on the form where form data from matching fields are imported from the application, open the Field Properties and select the Is read only checkbox. This pre-fills those fields on the reviewer form with form data from the application and set those fields to display only.
    • Save the form.

  3. In Packages, create a new package for the reviewer forms.

    1. In Area Type: select Reviewer if reviewers are required to fill in the forms, or select Reviewer Optional if reviewers can fill in the optional reviewer forms at their discretion.
    2. Add only the reviewer forms to the package. (Reviewer forms should not be included in the application package or broker package.)
    3. Set the reviewer forms in package to Required.
    4. Save the package.
  4. On the Sales Agreements page, add the Reviewer package to the applicable product sales agreements.
  5. On the Deploy Data page, deploy the forms, package, and sales agreement to the applicable environments.
  6. When a user completes and submits an application, the application along with any required or optional reviewer-only forms are sent to the Review Queue.
  7. When a reviewer clicks the Review button on an application order that has reviewer-only forms included, a Reviewer Documents button appears on the order review item. If the reviewer documents are required, the Approve button is disabled until the reviewer fills in and completes the required reviewer documents and returns to the order review item. If the reviewer documents are optional, the Approve button is enabled and the reviewer can determine whether or not to fill in the optional reviewer documents during the review process.
