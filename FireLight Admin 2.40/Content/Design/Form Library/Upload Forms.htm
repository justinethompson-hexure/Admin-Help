Upload Forms

Upload existing forms into FireLight® Admin. Admin supports form files that are in fillable or non-fillable Adobe .pdf file format or in Word .xps file format. Admin does not support Adobe proprietary file formats, such as .fdf. It does, however, fully support forms with existing tags, and they persist if present after upload in Admin. So if you already have existing forms that are tagged for anything else, you can upload them into Admin.

Important!

FireLight removes any interactive form elements from the PDF before stamping the data entry values onto form fields.

PDFs with interactive form elements (e.g., checkbox with an "x" already added) should be flattened before uploading into FireLight so that form fields and annotations become permanent and non-fillable and are not removed by FireLight.

You can find information on how to flatten files on the Internet. Another way to accomplish this is to print to PDF using something like Microsoft Print to PDF.

Upload Forms

  1. On the navigation bar, click Designer/Form Library.

  2. On the upper left of the Form Library page, click Upload Forms.

  3. If you previously exported forms (see Export a Form) and you want to upload those forms along with all of the metadata, select the Use exported definitions checkbox to import the forms along with the form field definitions on the forms. On the upper left, click the Add Files button.

    Click the View supported browsers link on the upper right for a list of supported web browsers.

  4. In the Choose File to Upload dialog, browse to the folder in which the forms reside, select the forms to upload for the new product, and click Open to begin uploading the files.

    Press and hold the Ctrl key while clicking file names to select multiple files from a folder.

  5. Click Start Upload to upload the files to Admin, or do any of the following:

    • Click Cancel on the right of a file name to remove the file from the file upload list.
    • Click Start on the right of a file name to start uploading the file to .
    • Click the Back button to return to the Form Library page without uploading the files.

  6. The upload forms grid displays the file name, file size, and status of the uploaded forms. There is one row for each form upload.

  7. When finished uploading forms, click the Back button to return to the Form Library page.
