Forms Grid and Associated Fields

The grid contains a row of information for each form with the following columns. Forms can be sorted alphabetically by clicking on Form Number, Form Name, or Edition in the header row.

Form Number displays the form number.

Form Name displays the form name.

Edition displays the form edition number as entered in the Form Detail tab on the lower left.

Lock/Unlock column contains a lock button that is used to allow or prevent other admin users from editing the form.

Delete column contains a red X that is used to delete the form from the system.

The following associated fields are below the form grid.

Exclude Expired Select this checkbox to exclude expired forms from the form grid. This filters the list to show only active forms in the grid.

Total displays the total number of forms that are currently in the grid.
