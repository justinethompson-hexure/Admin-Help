Check Box

Purpose

Provides selection of a single value within a group of checkbox fields or provides a single checkbox for a true (yes) or false (no) value.

  1. Drag a Check Box field from the right pane to the wizard page.
  2. On the page, double-click the Check Box to open the Check Box Properties dialog.

  3. Enter the field properties and click OK to save your changes.

  4. On the upper left, click SAVE WIZARD to save the wizard, which unlocks the wizard for editing by other users.

Caption The display label text for the checkbox. It is enabled to enter the caption when no Common Tag Name is associated with the field. It is disabled when there is a Common Tag Name associated with the field. To fill the correct value, click the Edit List Name Values link.
Export Value

The export value for the control. This can be a common tag list value or a custom export value. It is enabled to enter the export value when no Common Tag Name is associated with the field. It is disabled when there is a Common Tag Name associated with the field. To fill the correct value, click the Edit List Name Values link.
Edit List Name Values

NOTE: A list only appears in a checkbox control if a common tag name has been added to the field.

To add a caption and export value when a Common Tag Name is present that is associated with a list, click the Edit List Name Values link to open the List Name Values dialog. Only one row can be selected for a checkbox control. This can be a row provided by the selected Common Tag Name or a user-created row.

The following tooltips are available in the List Name Values dialog title row:

The following examples are for the Owner_Gender common tag. Do one of the following:

  • If you want a row provided by the selected Common Tag Name, select the checkbox for the applicable common tag row for a List Value and Caption/Rules Value for the checkbox control. Click OK to save. When the dialog is closed, the Caption/Rules Value appears in both the Caption and Export Value fields if no Rules Override was entered in the row. If a Rules Override was entered, it appears in the Export Value field.

  • If you want to create a new row with different values, scroll down to the last row, select the checkbox for the row, and enter the Caption/Rules Value and optional Rules Override value for the export value. Tab off the Caption/Rules Value column and click OK to save. When the dialog is closed, the Caption/Rules Value appears in both the Caption and Export Value fields if no Rules Override was entered in the row. If a Rules Override was entered, it appears in the Export Value field.

Example: Group of Two Check Boxes for Joint Owner

  1. Add a Check Box control with the Field Name and Common Tag Name set to JointOwner_Add, the Group Name set to JointOwner, and the Export Value of True. The user selects this checkbox to indicate there is a Joint Owner.

  2. Add another Check Box control with the Field Name and Common Tag Name set to Joint_Owner_Add, the Group Name set to JointOwner, and the Export Value set to False (No) and click the Update button. The user selects this checkbox to indicate there is no joint owner.

  3. Click the SAVE WIZARD button on the upper left of the Wizard Designer page to save the wizard, and unlock the wizard for editing by other users.
