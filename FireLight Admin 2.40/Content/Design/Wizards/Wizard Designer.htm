Maximum Wizard Size: The maximum size for a single wizard or form is 10MB.

Tip: The maximum size for a single wizard is 10MB. A wizard can be separated into multiple wizards instead of having dozens of pages in the same wizard to reduce the load/save time for an individual wizard.
