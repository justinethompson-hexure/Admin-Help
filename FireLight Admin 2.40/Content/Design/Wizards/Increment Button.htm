Increment Button

Purpose

Provides a button that will increment a data item’s value by a specified amount when clicked. Commonly used when you wish to add and remove multiple items to and from a page, such as additional Beneficiaries.

  1. Drag an Increment Button control from the right pane to the wizard page.
  2. On the page, double-click the Increment Button to open the Increment Button Properties dialog.
  4. Enter the remaining field properties and click OK to save your changes.
  5. On the upper left, click SAVE WIZARD to save the wizard, which unlocks the wizard for editing by other users.

Properties

Increment Button Properties
Property Description

Help Text
Icon Position The position of the icon in relation to the label text. Valid options are Left and Right.
Icon The icon displays next to the label text. This element is for display purposes only. Click Choose and select an icon from the available icons. Simply click the desired icon to select it. Click Clear to remove it.
Text Alignment The alignment of the label text. Valid options are Left, Center and Right.
Increment The amount to add to the data item’s value when the button is clicked or tapped.
