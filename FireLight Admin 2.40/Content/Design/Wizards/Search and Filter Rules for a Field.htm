Search and Filter Rules for a Field

  1. On the navigation bar, click Designer/Wizard Designer.
  2. On the upper left of the wizard Designer page, select a Wizard and a Page in the lists.
  3. In the left pane, click EDIT RULES.
  4. In Filter Rule Sets, begin typing the name of the rule set to filter the list, then click the rule set name in the list.

  5. In the left pane in Filter Rules with Field, type a full or partial dataitem name to filter the list of rules.

    The first rule for the dataitem appears on the right.
