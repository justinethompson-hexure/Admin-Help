Text Box

Purpose

Provides user input of text values.

  1. Drag a Text Box control from the right pane to the wizard page.
  2. On the page, double-click the Text Box to open the Text Box Properties dialog.
  3. Use only letters, numbers, spaces, and brackets.
  4. Enter the remaining field properties and click OK to save your changes.
  5. On the upper left, click SAVE WIZARD to save the wizard, which unlocks the wizard for editing by other users.

SSN Format Select the level of masking to apply to the SSN, TIN, or Canadian SIN identification number during data entry: None (not a SSN, TIN or SIN field), Visible (all numbers are visible, e.g., 123456789), Partial (last four numbers are visible, e.g., XXXXX6789), or Full (no numbers are visible, e.g., XXXXXXXXX). The default value is None, which indicates the field is not a SSN, TIN or SIN field. The Text Mask field property cannot be used when the SSN Format option is other than None. To secure the value, SSN mask formatting is applied on the server so it is not visible in the browser.

Maximum Length Sets the maximum number of characters that can be entered in a field. For example, a field may display 20 characters but the user needs to be able to enter up to 35 characters. By setting the Maximum Length to 35 the field will automatically scroll after 20 characters up to the maximum. See "Show Character Count" below.

Show Character Count This field property becomes available when a Maximum Length is added. Select the Show Character Count checkbox to display the character countdown on the lower right of the Text Box field. In the wizard and combined wizards+forms UI (in React), a character countdown displays on the lower right of the Text Box field as the user types in the field. For example, "42/100" shows the text box contains 42 characters of a possible 100 characters per the 100 Max Length setting in the field properties. When a user hovers over the character countdown value, a tool tip is displayed as "Maximum characters allowed xxx." Requires a Maximum Length entry. NOTE: The legacy forms UI does not support this feature. It is only available in the wizards UI and the wizards+forms UI.
