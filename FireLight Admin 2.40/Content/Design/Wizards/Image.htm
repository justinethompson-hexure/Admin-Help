Image

Purpose

Displays an image.

  1. Drag an Image control from the right pane to the wizard page.
  2. On the page, double-click the Image to open the Image Properties dialog.
  4. Enter the remaining field properties and click OK to save your changes.
  5. On the upper left, click SAVE WIZARD to save the wizard, which unlocks the wizard for editing by other users.

Properties

Image Properties
Property Description
Place Field on a New Row
Field Name This does not need to be unique among all fields in the wizard.
Width
Mobile Width
Class Name
Help Text
Image The image to be displayed within the field. Click Choose and a browse dialog will open to search for an image. Simply click the desired image to select it. Click Clear to remove it. NOTE: only jpeg and png file types are supported.
