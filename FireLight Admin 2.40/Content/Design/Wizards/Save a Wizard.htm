Save a Wizard

  1. On the navigation bar, click Designer/Wizard Designer.
  2. On the upper left of the Wizard Designer page, select a wizard in the list and make updates to the wizard.
  3. Click the SAVE WIZARD button on the upper left of the Wizard Designer page to save any changes that were made to the wizard, and to unlock the wizard for editing by other users.
  5. If you do not click the Save Wizard button on the upper left of the Wizard Designer page before leaving the page, a Save Wizard dialog appears. Click Yes to save wizard changes and exit the wizard or click No to discard changes and exit the wizard.
