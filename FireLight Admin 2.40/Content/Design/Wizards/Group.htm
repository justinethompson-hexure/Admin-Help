Group

Purpose

Groups other fields. Similar to a Panel, but with no default styling.

  1. Drag a Group control from the right pane to the wizard page.
  2. On the page, double-click the Group to open the Group Properties dialog.
  4. Enter the remaining field properties and click OK to save your changes.
  5. On the upper left, click SAVE WIZARD to save the wizard, which unlocks the wizard for editing by other users.

Properties

Group Properties
Property Description
Place Field on a New Row Regardless of the width of the group and the width of the fields in the current row, force the group to be placed at the beginning of a new row. See Wizard Grid Layout for more information.
Field Name
Width The width of the group. This should be a number from 1 to 12. See Wizard Grid Layout in Wizard Designer for more information.
Mobile Width The width of the group, when displayed on a mobile device. Defaults to 12 unless otherwise specified. See "Grid Layout" for more information.
Error Placement
Class Name Use the following root site CSS classes to add 1 em (1 times the size of the font) padding to the individual sides of a Group as applicable: padTop, padBottom, padLeft, padRight. NOTE: These padding classes are available only for the Group container.
