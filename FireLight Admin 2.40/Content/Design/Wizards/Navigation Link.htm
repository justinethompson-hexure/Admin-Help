Navigation Link

Purpose

Provides a link, either internal within the wizard user interface or to an external site. The link can be displayed with a text label, or an icon, or both.

  1. Drag a Navigation Link control from the right pane to the wizard page.
  2. On the page, double-click the Navigation Link to open the Navigation Link Properties dialog.
  4. Enter the remaining field properties and click OK to save your changes.
  5. On the upper left, click SAVE WIZARD to save the wizard, which unlocks the wizard for editing by other users.

Link Type

Select a type of link:

  • Internal – Use this navigation link type to navigate to a page within the wizard. You must enter the form number located in the Form properties and associated wizard page number. Be sure to add the form to the package so the internal link will work.
  • External – Use this navigation link type to navigate to an external link; a URL must be provided.
  • Set in Rules – Use this navigation link type to navigate to a conditional external link; a URL must be provided in the rule set.

NOTE: When Link Type is Internal or Set in Rules and a page is disabled due to conditions of a section break, the Navigation Link will not jump to the disabled page and FireLight will display a message that the page is not available to the user. (See Add and Delete Pages and Section Breaks). An External link is not impacted by a section break.
URL Enter a URL for the External navigation link. Note, this option only applies to the External Link Type.
Open in a New Window: Indicates that the external navigation link location will open in a new window. Note, this option only applies to the External Link Type.
Text Alignment The alignment of the label text. Valid options are Left, Center and Right.
Font Size The font size for the field’s text. Valid options are Default, Small, Medium, Large, Extra Large and Extra Extra Large. Default is the default font size in the FireLight base CSS file. Note that the font size measurement is in pixels.
