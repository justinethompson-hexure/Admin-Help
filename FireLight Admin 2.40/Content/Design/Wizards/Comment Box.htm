Comment Box

NOTE: The Comment Box floating text box is only for wizards. It is not available in forms. The Comment Box applies only to the current wizard in which it is added. It does not apply to all wizards within a package. Only one Comment Box is allowed per wizard page.

Purpose

Adds a floating text box on a wizard in FireLight®, which provides FireLight users with the ability to add notes and comments. FireLight users can drag the floating text box anywhere on the FireLight page including the space to the right or left of the wizard page. The floating text box automatically scrolls up and down with the page so it is always visible.

  1. In the lower left pane of the wizard page, click Add Comment Box.

    NOTE: The comment box and properties are not visible on the wizard page until you click Edit Comment Box.

    >

  2. In the lower left pane, click Edit Comment Box to open the Comment Box Properties dialog.

  3. Type the Name for the control.
  4. Enter the remaining field properties and click OK to save your changes. Click CANCEL to cancel the edits to properties. Click REMOVE to remove (delete) the floating text box.
  5. On the upper left, click SAVE WIZARD to save the wizard, which unlocks the wizard for editing by other users.

Properties

Comment Box Properties
Property Description
Field Name This does not need to be unique among all fields in the wizard.
Text The label text for the Comments Box text box (appears on top of the field).
Class Name
Help Text
Position Vertical Settings Sets the vertical placement of the Comments Box on the wizard page. Select Top, Middle or Bottom.
Position Horizontal Settings Sets the horizontal placement of the Comments Box on the wizard page. Select Right or Left.
Maximum Length Sets the maximum number of characters that can be entered in the Comments Box. For example, a field may display 20 characters but the user needs to be able to enter up to 35 characters. By setting the Maximum Length to 35, the field automatically scrolls after 20 characters up to the maximum. This field property does not require an entry. The default is 100.
Apply to all pages of the Wizard Select the checkbox to apply the floating text box to all pages in the current wizard. Clear the checkbox to apply the floating text box to only the page on which the Comments Box was added in the wizard—it does not apply to any other pages in the wizard. NOTE: You can add a Comments Box to separate pages in the wizard or to all pages in the wizard.
