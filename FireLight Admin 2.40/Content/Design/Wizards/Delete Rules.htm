Delete Rules

  1. On the navigation bar, click Designer/Wizard Designer.

  2. On the upper left of the Wizard Designer page, select a wizard in the list.
  3. On the wizard, click in a field.

  4. In the left pane, click EDIT RULES to open the Rules window.

  5. In Filter Rule Sets, begin typing the name of the rule set to filter the list, then click the rule set name in the list.
  6. The List only rules for [dataitem] checkbox is selected by default. The rules list contains only rules for the selected dataitem on the wizard page.
  7. Locate the rule in the list and click the delete (trash can) button on the far right of the rule name to delete the rule.
  8. On the lower right, click SAVE RULE SET to save the rule set, and unlock the wizard for editing by other users.
  9. Click CLOSE to exit the Rules window.
