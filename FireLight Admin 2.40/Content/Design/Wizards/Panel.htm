Panel

Purpose

Provides a styled area on the page that contains other fields. This includes a header and border around all of the fields within the panel.

  1. Drag a Panel control from the right pane to the wizard page.
  2. On the page, double-click the Panel to open the Panel Properties dialog.
  3. Type the Field Name for the panel. Optionally, you may type in search text in the empty search box and select from the list of common tags to autocomplete the field name.
  4. Enter the remaining field properties and click OK to save your changes.
  5. On the upper left, click SAVE WIZARD to save the wizard, which unlocks the wizard for editing by other users.

Properties

Panel Properties
Property Description
Place Field on a New Row Regardless of the width of the panel and the width of the fields in the current row, force the panel to be placed at the beginning of a new row. See Wizard Grid Layout for more information.
Field Name
Text The panel header label text.
Width The width of the panel. This should be a number from 1 to 12. See "Wizard Grid Layout" in Wizard Designer for more information.
Mobile Width The width of the panel, when displayed on a mobile device. See "Wizard Grid Layout" for more information.
Class Name
Help Text If set, display a Help icon beside the panel that, when clicked, will display the specified text.
Image The image to be displayed in the header of the panel. Click Choose to browse to and select an image file. Click Clear to remove the image file.
