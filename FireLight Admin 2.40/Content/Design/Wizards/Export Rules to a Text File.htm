Export Rules to a Text File

  1. On the navigation bar, click Designer/Wizard Designer.
  2. In the left pane of the Wizard Designer page, click EDIT RULES to open the Rules window.
  3. In Filter Rule Sets, begin typing the name of the rule set to filter the list, then click the rule set name in the list.
  4. On the lower left of the Rules window, click EXPORT RULES.

  5. If a Notification bar appears, click the arrow on the right of the Save button and click one of the Save options to save the [RuleSetName].txt file to a location on the network or PC.

  6. Click CLOSE to exit the Rules window.
