Text

Purpose

Displays text on the screen.

  1. Drag a Text control from the right pane to the wizard page.
  2. On the page, double-click the Text to open the Text Properties dialog.
  4. Enter the remaining field properties and click OK to save your changes.
  5. On the upper left, click SAVE WIZARD to save the wizard, which unlocks the wizard for editing by other users.

Properties

Text Properties
Property Description
Place Field on a New Row
Field Name This does not need to be unique among all fields in the wizard.
Text The text to be displayed within the field.
Width
Mobile Width
Class Name
Help Text
Bullet Type The bullet to display before the text. Valid options are: Disc, Circle and Square.
Text Alignment The alignment of the text within the field. Valid options are Left, Center and Right.
Font Size The font size for the field’s text. Valid options are Small, Medium, Large, Extra Large and Extra Extra Large. Default is the default font size in the FireLight base CSS file. Note that the font size measurement is in pixels.
Bold Check to display the text in a bold format.
Text is HTML Indicates that the text in the field is in HTML format.
Office Use Only
