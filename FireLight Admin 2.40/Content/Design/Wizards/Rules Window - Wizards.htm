Rules Window

NOTE: See Locking of Wizards, Rule Sets, and Rules in the Wizard Designer for wizard designer edit lock information.

In the left pane of the wizard Designer page, click the EDIT RULES button to open the Rules window and add rules to wizard fields (data items) via the pre-built rule templates or by creating a new rule.

Node Help

  1. On the left of the Wizard Designer page, click EDIT RULES.
  2. On the upper right of the Rules Window, click Node Help to open a dialog.
  3. Select an option in the Node Help For Template Type list to access descriptions and examples of nodes used in Rules, Validation, BackOffice or Queues.
