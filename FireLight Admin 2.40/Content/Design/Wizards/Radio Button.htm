Radio Button

Use the Radio Button control to limit the number of selection options.

Purpose

Provides selection of a single value within a group of radio button fields.

  1. Drag a Radio Button control from the right pane to the wizard page.
  2. On the page, double-click the Radio Button to open the Radio Button Properties dialog.
  3. Enter the field properties and click OK to save your changes.

Caption

The display name text for the radio button. It is enabled to enter the caption when no Common Tag Name is associated with the field. It is disabled when there is a Common Tag Name associated with the field. To fill the correct value, click the Edit List Name Values link.

Export Value

The export value for the control. This can be a common tag list value or a custom export value. It is enabled to enter the export value when no Common Tag Name is associated with the field. It is disabled when there is a Common Tag Name associated with the field. To fill the correct value, click the Edit List Name Values link.
Edit List Name Values

NOTE: A list only appears in a radio button control if a common tag name has been added to the field.

To add a caption and export value when a Common Tag Name is present that is associated with a list, click the Edit List Name Values link to open the List Name Values dialog. Only one export value row can be selected for a radio button control. This can be a row provided by the selected Common Tag Name or a user-created row.

Example: Group of Radio Buttons for Gender

Add Radio Button controls for the owner gender.

  1. On the right, drag a Radio Button control to the applicable location on the wizard.
  2. On the wizard, double-click a Radio Button to open the Radio Button Properties dialog.

  3. In Field Name, enter the data item ID for the field. Click in the Common Tag Name property and enter or select the common tag name, Owner_Gender. The system automatically adds the Owner_Gender common tag list items to the List Name Values dialog. The Caption and Export Value fields are disabled when the Common Tag Name is selected.

  4. Enter OwnerGender in Group Name. The Group Name identifies the group of radio buttons.

  5. Click the Edit List Name Values link to open the List Name Values dialog.

    If a value in the common tag list is preferred for the checkbox (Female, Male, etc.), select the checkbox for the preferred value. For example, select Male. If applicable, add a Rules Override value. For example, enter M to be used in rules rather than the caption. Click OK to save changes and close the List Name Values dialog.

  6. The radio button now shows a Caption of Male and an Export Value of "M" on the Radio Button Properties dialog.

  7. Continue adding Radio Button controls for each owner gender option to be included in the OwnerGender group of radio buttons. Set the Field Name, Common Tag Name, and Group Name to the same values for each of the radio buttons in the group. Then select a unique row for the List Value and Caption/Rules Value in the List Name Values dialog for each radio button.
