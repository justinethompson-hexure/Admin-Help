Address Autocomplete

NOTE: This service is not Health Insurance Portability and Accountability Act (HIPAA) compliant. The integrations are configured so that we can disable them for an organization, which we do if we enter into a HIPAA Business Associate Agreement (BAA) with the organization.
Spotlight Video: For additional information, see the Address Autocomplete, Address Validation, and Dr. Lookup Spotlight Feature Video.

Purpose

Provides an Address Autocomplete field where the user begins typing an address and presses Enter to open a drop-down selection list of addresses to choose from provided by the Google Places database.

NOTE: You must set the Data Group for who it applies to (e.g., Owner) in the Address Autocomplete Properties and then select the same Data Group and the applicable Data Group Property in each of the associated address fields on the wizard.

  1. Drag an Address Autocomplete control from the right pane to the wizard page.
  2. On the page, double-click the Address Autocomplete field to open the Address Autocomplete Properties dialog.
  3. In the Field Name property, type the data item ID for the control.
  4. Enter the remaining field properties and click OK to save your changes.
  5. On the upper left, click SAVE WIZARD to save the wizard, which unlocks the wizard for editing by other users.

Data Group (Required) Select a Data Group. You must select a Data Group for who it applies to in the Address Autocomplete field and then select the same Data Group and the applicable Data Group Property in all of the associated address fields on the wizard that is pre-filled by the address autocomplete feature.

For example, in the Address Autocomplete Properties for the Address Autocomplete field, set Data Group to Owner. In the Text Box Properties for the Owner_ResidentialAddress1 field, set Data Group to Owner and set Data Group Property to Line1. In each of the remaining address fields, set Data Group to Owner and set Data Group Property to the applicable option (Line2, City, State, Zip, etc.).
