Export Rules to a Text File

You can export rules for the selected input control data item to a text file.

  1. On the navigation bar, click Designer/Form Designer.
  2. On the upper left of the Form Designer page, select a form in the list.
  3. Click an input control field on the form.
  4. On the left under Input Control, click Rules Window.

  5. In the pane on the lower left, verify that the data item assigned to the input control has at least one associated rule.

    On the Rules Window, click Export Rules on the upper right.

  6. In the Notification bar below the Rule Window, click Rules.txt to open the Rules.txt file.

  7. If applicable, save the Rules.txt file to a location on the network or PC.
