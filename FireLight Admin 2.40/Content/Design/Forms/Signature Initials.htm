Signature Initials

NOTE: Within FireLight®, most dates and times in the UI are displayed in the user's time zone (as configured in Preferences). Signatures are different. Signature timestamps on forms are shown in UTC. Signature dates and times elsewhere in the UI, including dates obtained from the Signature Date control, are converted to the time zone of the signer’s state of residence.

Use the Signature Initials control to add e-signature initials to the form.

  1. On the left under Add Control to Form, click Signature Initials.

  2. On the Adding an Initial dialog, click OK to proceed.

  3. Drag the Signature Initial control from the upper left of the form to the signature initials area on the form and re-size it to fit the field space.

  4. On the left under Input Control, click Field Properties.

  5. Enter or change settings in Field Properties.

    Width, Height

    Type the width and height of the signature box in pixels. A warning appears if the signature box is not at least 160 pixels wide. The practical minimum size is 160 wide by 30 high. See "Width Height Examples" below.
    Group Name

    (Optional) Type the name for the group of signatures. The Group Name is used by the system to assemble a collection of forms into a document set that can be signed together. For example, you could define one group named “1Consent” to include only the consent form, and another group called “Application” for the rest of the forms. FireLight arranges the document sets first by number and then alphabetically. To review the consent form first, the Group Name is set to "1Consent" and the Group Name fields for the forms in the application package are set to "Application." Doing this, FireLight leads the signer through two separate signing processes; starting with the E-Consent form and followed by the Application.
    Signer Type Select the type of signer in the list. The system uses Signer Type to determine who signs the form in the signature capture process.
    Control Type Select the control type of Initial (other options are Signature, City, Date, or State).

  6. Click the Update button to update the field properties and lock the form for editing.
  7. Click the Close button to close the Field Properties dialog.

  8. Click the SAVE button on the upper left of the Designer page to save the controls and field properties, which unlocks the form for editing by other users.

    NOTE: If at least one Signature field with the same Group Name and Signer Type as the Initial is not on the form, a "Please be sure to add at least one Signature field (with same Group Name and Signer Type as this Initial) on this form" message appears. Click Yes to close the message.
