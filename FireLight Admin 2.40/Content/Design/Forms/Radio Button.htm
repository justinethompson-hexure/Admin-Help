Radio Button

Add a Radio Button Control

Use the Radio Button control to limit the number of selection options, which provides selection of a value within a group of radio button fields.

  1. On the left under Add Control to Form, click Radio Button.
  2. Drag the control from the upper left of the form to an applicable location on the form.

  3. Double-click the Radio Button, or on the left under Input Control, click Field Properties to open the Field Properties dialog.

  4. Enter the field properties for the radio button.

    Radio Button Field Properties
    Field Property Description
    Field Name
    Common Tag Name

    When a Common Tag Name field is set to blank for a Check Box or Radio Button, the existing list is deleted.

    If a Common Tag Name is selected, the Edit List Name Values link appears below the Export Value field property. Otherwise, it does not apply.
    Export Value

    The export value for the control. This can be a common tag list value or a custom export value. The default value is "true." It is enabled to enter the export value when no Common Tag Name is associated with the field. It is disabled when there is a Common Tag Name associated with the field. To fill the correct value, click the Edit List Name Values link.
    Edit List Name Values

    NOTE: A list appears in a radio button control only if a common tag name has been added to the field.

    To add an export value when a Common Tag Name is present that is associated with a list, click the Edit List Name Values link to open the List Name Values dialog. Only one row can be selected for a radio button control. This can be a row provided by the selected Common Tag Name or a user-created row.

    The following tooltips are available in the List Name Values dialog title row:

    The following examples are for the Owner_Gender common tag. Do one of the following:

    • If you want a row provided by the selected Common Tag Name, select the checkbox for the applicable common tag row for a List Value and Caption/Rules Value for the radio button control. Click OK to save. When the dialog is closed, the Caption/Rules Value appears in the Export Value field if no Rules Override was entered in the row. If a Rules Override was entered, it appears in the Export Value field.

    • If you want to create a new row with different values, scroll down to the last row, select the checkbox for the row, and enter the Caption/Rules Value and optional Rules Override value for the new export value. Tab off the Caption/Rules Value column and click OK to save. When the dialog is closed, the Caption/Rules Value appears in the Export Value field if no Rules Override was entered in the row. If a Rules Override was entered, it appears in the Export Value field.

Example: Group of Radio Buttons for Gender

Add radio button controls for selecting a gender for the owner.

  1. On the left under Add Control to Form, click Radio Button.
  2. Drag the control from the upper left of the form to an applicable location on the form.

  3. Double-click the Radio Button, or on the left under Input Control, click Field Properties to open the Field Properties dialog.

  4. In Field Name, enter OwnerGender.

  5. Click in Common Tag Name and begin typing Owner in the text box at the top of the list of common tags and select Owner_Gender from the list of common tags. The Export Value is disabled when a Common Tag Name is selected.

  6. Click in Group Name and add OwnerGender as the group name.

  7. Click the Edit List Name Values link to open the List Name Values dialog. The last row with the default export value of true is checked by default.

  8. If a value in the common tag list is preferred for the checkbox (Female, Male, etc.), select the checkbox for the preferred value. For example, select Female. If applicable, add a Rules Override value. For example, enter F to be used in rules rather than the caption. Click OK to save changes and close the List Name Values dialog.

    The radio button now shows an Export Value of "F." On the Field Properties dialog, click Update.

  9. Continue adding Radio Button controls for each owner gender option to be included in the OwnerGender group of radio buttons. Set the Field Name, Common Tag Name, and Group Name to the same values for each of the radio buttons in the group. Then select a unique row for the List Value and Caption/Rules Value in the List Name Values dialog for each radio button.
