Delete Rules

  1. On the navigation bar, click Designer/Form Designer.
  2. On the upper left of the Form Designer page, select a form in the list.
  3. On the form, click an input control for which to delete a rule.
  4. On the left under Input Control, click Rules Window.

  5. On the left, locate the rule in the list and click the red X next to the rule name to delete the rule.

    Or, click the Delete All link above the rules list to delete all of the rules for the input control.

  6. Click Yes on the Delete Confirmation dialog.

  7. Click the Update button on the Rules Window to save the change and lock the rule set for editing.

  8. Click the X on the upper right to close the Rules Window.

  9. Click Yes on the Rule Set Changes dialog to save the rule set, which unlocks the rule set for editing by other users.
