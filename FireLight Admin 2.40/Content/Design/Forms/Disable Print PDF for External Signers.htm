Disable Print PDF for External Signers

Agents may want certain external signers to not have the option to print forms while reviewing them to remain in compliance and not need to report any incidents due an incorrect process being followed.

  1. In the Admin tool in a signature control in the form designer, select DisablePrintPDF.

  2. Verify that the signer type is not an agent and that the signature control is not an initial control.

  3. Create an application and enter into the signing process.

  4. Select the signer that had disable print PDF enabled.

  5. Choose to send the email to complete the signing as an external request.

  6. Enter the external signing process using the emailed URL link.

  7. Verify that the option to "Review Documents" does not appear on the initial screen. In addition, the option to "View Activity PDF" should not appear in the toolbar.

  8. Select Sign and enter the signing process.

  9. Verify that View Activity PDF does not appear on the toolbar.
