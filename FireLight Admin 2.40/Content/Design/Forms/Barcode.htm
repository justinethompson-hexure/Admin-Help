Barcode

Use the Barcode control to add a barcode to the form.

  1. On the left of the Form Designer page under Add Control to Form, click Barcode.

  2. Drag the Barcode control from the upper left of the form to the signature area on the form and re-size it to fit the field space.

  3. On the left under Input Control, click Field Properties.

  4. Enter or change settings in Field Properties.

    Barcode Field Properties
    Field Property Description
    Field Name

    Top, Left
    Width, Height
    Format Type the barcode format. This can be whatever value you type into the field or it can be dynamically set with a data item value. To set it dynamically with a data item, enclose the data item name in curly brackets, for example, {Transaction ID}.
    Barcode Type

    In the list, select the barcode type (Datamatrix or QRCode).

    NOTE: Datamatrix type is visible in the application process during all phases (data entry, e-sign, submission) and it is visible online and in the PDF/printed version. QRCode type is not visible in the application process and is visible online and in the PDF/printed version.
    Data Group

    Select a Data Group if applicable.

    Custom List

    In the list, select a Custom List if this control is associated with a Custom List Data Group.

    Data Group Property

    Select an option in Data Group Property when a field is associated with a Data Group. Select the type of data to pull in the Data Group Property for the selected Data Group and dataitem. When Data Group is a Custom List, select a custom list column from which to pull the data for the field.

    Global Scope
    Exclude from PDF

    By default the Barcode control is selected and locked, which means the control does not appear on the printed or submitted form.
    Passcode Only Validation
    Z Order

  5. Click the Update button to update the field properties and lock the form for editing.
  6. Click the Close button to close the Field Properties dialog.
  7. Click the SAVE button on the upper left of the Form Designer page to save the form, which unlocks the form for editing by other users.
