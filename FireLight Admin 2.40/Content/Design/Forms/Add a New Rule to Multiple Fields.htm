Add a New Rule to Multiple Fields

You can select multiple fields on a form and create a new custom rule or pre-built template rule for the multiple fields using the Create Multiple button on the Rules Window. This feature is for new rules only and cannot be used for updating rules.

The following example adds the pre-built template Required rule to the Owner_FirstName, Owner_MiddleName, and Owner_LastName controls.

  1. On the navigation bar, click Designer/Form Designer.
  2. On the upper left of the Form Designer page, select a form in the list.

  3. On the form, select the multiple controls (data items) that have the same rule.

    For example, select Owner_FirstName, Owner_MiddleName, and Owner_LastName controls.

  4. On the left in Input Control, click Rules Window.
  5. In Rules Window, click New.
  6. On the left of the New Rule Template dialog, click a pre-built rule template (or enter your own custom rule) (e.g., click the Required rule) and click OK.

  7. On the Rules Window, click Create Multiple.

    NOTE: If the Validate Rule Name Uniqueness checkbox is selected in Organization Configuration/Organization Settings on the Admin tab, a "Rule name must be unique!" message appears if the rule name is not unique. Click OK and change the rule names to be unique.

    8. The new rules names and details appear on the left of the Add Rule to Multiple Fields dialog. The rules XML appears on the right of the dialog with item number replacing the name of the dataitem name (e.g., {item1}).

  8. On the lower right, click Create Rules.

  9. On the Save Multiple Rules dialog, click OK.
  10. The new rules appear in the list of rules for the selected rule set on the Rules Window. Close the Rules Window.
  11. On the Rule Set Changes dialog, click Yes to save the rule set changes.
