Label

Use the Label control to add viewable text on screen, but the text does not print on the form itself.

You can set the Protection field property to Private so only the form designer can see the label and then add a rule containing a dataitem. For example, you could add a Label control to a form with a rule that sets the value of the FLI_REQUIREMANUALREVIEW dataitem to true so the application would automatically go to the PreSubmit review instead of giving the user the option to submit without review.

In this example, a label control is used to guide the user to select a plan type.

  1. On the left under Add Control to Form, click Label.
  2. Drag the control from the upper left of the form to an applicable location on the form.

  3. On the left under Input Control, click Field Properties.

  4. Enter or change settings in Field Properties.

  5. Click the Update button to update the field properties and lock the form for editing.
  6. Click the Close button to close the Field Properties dialog.
  7. Click the SAVE button on the upper left of the Form Designer page to save the form, which unlocks the form for editing by other users.
