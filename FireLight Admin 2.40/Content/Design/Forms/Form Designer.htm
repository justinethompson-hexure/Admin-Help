Form Designer

On the navigation bar, click Designer/Form Designer to go to the Form Designer page. Use this page to add input controls to new fillable forms from uploaded Word .xps or PDF files or to enhance existing fillable PDF forms. Use the linked Rules Window to add rules to the forms.

NOTE: See the "Common Tags" topic in Admin Help Common Tags for common tags to be used in mapping dataitems. Use of common tags ensures data, such as inbound data, input controls, CRMs, etc., is mapped correctly throughout the system.

Maximum Form Size: The maximum size for a single form or wizard is 10 MB.
