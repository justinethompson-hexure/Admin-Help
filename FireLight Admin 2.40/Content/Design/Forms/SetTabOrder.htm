Set Tab Order

NOTE: If implemented, the individual field properties Tab Index settings override the tab order sequence set here.

Use the Set Tab Order feature to have the system set the default tab order from left to right, top to bottom on the form page or to set the tab order vertically based on selected fields.

Set a Default Tab Order

Set the default tab order from left to right, top to bottom on the form page.

  1. On the navigation bar, click Designer/Form Designer.
  2. On the upper left, select a form in the list.
  3. On the left under Input Control, click Set Tab Order to open the Tab Order dialog.
  4. In Start From, select a starting tab order number.

  5. Click the Set Default Tab Order button, which locks the form for editing.

    The tab order is set from left to right, top to bottom on the form page and a “Setting Tab Order Completed Successfully” message appears below the button.

  6. Close the dialog by clicking the X on the upper right of the Tab Order dialog.
  7. Click SAVE on the upper left to save the form, which unlocks the form for editing by other users.

Set a Tab Order of Selected Fields

Set the tab order of vertical fields.

  1. On the navigation bar, click Designer/Form Designer.
  2. On the upper left, select a form in the list.
  3. Select a group of vertical fields on the form page.
  4. On the left under Input Control, click Set Tab Order to open the Tab Order dialog.
  5. In Start From, select a starting tab order number.

  6. Click the Set Tab Order Of Selected Fields button, which locks the form for editing.

    The tab order is set for the vertical field selections and a “Setting Tab Order Completed Successfully” message appears below the button.

  7. Close the dialog by clicking the X on the upper right of the Tab Order dialog.

  8. On the upper left, click the SAVE button to save the form, which unlocks the form for editing by other users.
