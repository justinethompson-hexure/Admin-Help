CheckBox (Check Box)

Add a CheckBox Control to a Form

  1. On the left under Add Control to Form, click CheckBox.

  2. Drag the CheckBox control from the upper left of the form to the checkbox location on the form.

  3. On the left under Input Control, click Field Properties.

  4. Enter or change settings in Field Properties.

    CheckBox Field Properties
    Field Property Description
    Field Name
    Common Tag Name

    When a Common Tag Name field is set to blank for a Check Box or Radio Button, the existing list is deleted.

    If a Common Tag Name is selected, the Edit List Name Values link appears below the Export Value field property. Otherwise, it does not apply.
    Export Value

    The export value for the control. This can be a common tag list value or a custom export value. It is enabled to enter the export value when no Common Tag Name is associated with the field. It is disabled when there is a Common Tag Name associated with the field. To fill the correct value, click the Edit List Name Values link.
    Edit List Name Values

    NOTE: A list appears in a checkbox control only if a common tag name has been added to the field.

    To add an export value when a Common Tag Name is present that is associated with a list, click the Edit List Name Values link to open the List Name Values dialog. Only one row can be selected for a checkbox control. This can be a row provided by the selected Common Tag Name or a user-created row.

    The following tooltips are available in the List Name Values dialog title row:

    The following examples are for the Owner_Gender common tag. Do one of the following:

    • If you want a row provided by the selected Common Tag Name, select the checkbox for the applicable common tag row for a List Value and Caption/Rules Value for the checkbox control. Click OK to save. When the dialog is closed, the Caption/Rules Value appear in the Export Value field if no Rules Override was entered in the row. If a Rules Override was entered, it appears in the Export Value field.

    • If you want to create a new row with different values, scroll down to the last row, select the checkbox for the row, and enter the Caption/Rules Value value and optional Rules Override value for the new export value. Tab off the Caption/Rules Value column and click OK to save. When the dialog is closed, the Caption/Rules Value appears in the Export Value field if no Rules Override was entered in the row. If a Rules Override was entered, it appears in the Export Value field.

    Click OK to save the entries and return to the Field Properties dialog.

  5. Click the Update button to update the field properties and lock the form for editing.
  6. Click the Close button to close the Field Properties dialog.
  7. Click the SAVE button on the upper left of the Form Designer page to save the form, which unlocks the form for editing by other users.

Example: Group of Check Boxes for Owner Marital Status

  1. Add a Check Box control with the Field Name and Common Tag Name set to Owner_MaritalStatus, the Group Name set to OwnerMarStat. The Export Value is disabled when a Common Tag Name is selected.

  2. Because the Common Tag Name has associated list name values, click the Edit List Name Values link to open the List Name Values dialog. The last row with the default export value of true is checked by default.

    If a value in the common tag list is preferred for the checkbox (Married, Single, etc.), select the checkbox for the preferred value. For example, select Single. Click OK to save changes and close the List Name Values dialog.

  3.  Click Update on the Field Properties dialog. The Export Value is now set to Single.

  4. Continue adding Check Box controls for each applicable owner marital status option. Set the Field Name to Owner_MaritalStatus and set the Group Name to OwnerMarStat for each checkbox in the group.

  5. Click the Save button on the upper left of the Form Designer page to save the form, which unlocks the form for editing by other users.
