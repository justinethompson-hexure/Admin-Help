Rules Window

On the Form Designer page, select a form in the list and click the Rules Window link. Use the Rules Window to add rules to input controls (data items) on the form via the pre-built rule templates available from the New Rule Template window or by creating a new rule, deleting rules, exporting rules to a text file, and searching and filtering rules for a field.

Node Help

  1. On the upper right of the Rules Window, click Node Help.
  2. On the Node Help dialog, select an option in the Node Help For Template Type list to access descriptions and examples of nodes used in rules.
  3. On the left, expand the lists to find Help for specific nodes.
