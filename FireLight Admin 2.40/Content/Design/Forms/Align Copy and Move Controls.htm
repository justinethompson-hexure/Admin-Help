Align, Copy, and Move Controls

Use the Toolbox feature or the mouse to align, copy, and move controls on a form.

Use the Toolbox to Align, Copy, and Move Controls

  1. From the upper left, select a form in the list.

  2. Select one or more controls on the form.

    If copying or moving controls to another page, go to the target page and go to step 4b or 4c below.

  3. On the left, click Toolbox.

  4. Do any of the following for the selected controls.

    Copy selected fields to the same page:

    1. Click Copy Selected Fields On Page to copy the selected controls (fields).
    2. Drag the new controls to the applicable locations on the page.

    Copy selected fields to another page:

    1. Go to the page to copy the fields to and click Copy Selected Fields On Page.
    2. Drag the copied controls to the applicable locations on the target page.

    Move controls to another page:

    1. Go to the page to copy the fields to and click Copy Selected Fields On Page.
    2. Go back to the page you copied the fields from and close the Toolbox by clicking the X on the upper right.
    3. Ensure the copied controls are still selected and click Delete Control on the left.

    Move controls on the same page:

    1. Select the applicable controls.

      Type the number of pixels to move the controls Left, Top, Right, or Bottom.

      Click Move Controls. This moves the selected controls the number of pixels in the designated direction.

    Align controls on a page:

    Select the applicable controls and then click the Left, Top, Bottom, or Right alignment button in which to align the selected controls.

  5. Click the X on the upper right to close the Toolbox.

  6. Click the SAVE button on the upper left to save the form changes, which unlocks the form for editing by other users.

Use the Mouse to Move Controls

Use the mouse to move a single control or multiple controls.

  1. On the upper left, select a form in the list.
  2. Click and hold the left mouse button, drag to select one or more controls, then release the mouse button.

  3. Click and hold the left mouse button on one control and move it in the direction all of the selected controls need to go. Release the mouse button when the controls are in place.
