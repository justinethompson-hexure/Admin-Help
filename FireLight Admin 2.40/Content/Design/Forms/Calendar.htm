Calendar

Use the Calendar control to add a calendar selection to the form instead of using a Text Box control for a date entry.

  1. On the left under Add Control to Form, click Calendar.
  2. Drag the control from the upper left of the form to an applicable location on the form.

  3. On the left under Input Control, click Field Properties.

  4. Enter or change settings in Field Properties.

    Data Group Property

    Select an option in Data Group Property when a calendar is associated with a custom action Button. When Action Type for the Button control is set to CRM, select the type of data to pull in the Data Group Property for the selected Data Group and dataitem (e.g., for the Owner_FirstName field, select Owner in Data Group and select FirstName in Data Group Property). When a custom action Button is set to Custom List, select a custom list column from which to pull the data for the field (e.g., for a field associated with a Medications custom list, select Custom List 1, etc., number in Data Group, and select Drug Name in Data Group Property as the column from which to pull data for the field).

  5. Click the Update button to update the field properties and lock the form for editing.
  6. Click the Close button to close the Field Properties dialog.
  7. Click the SAVE button on the upper left of the Form Designer page to save the form, which unlocks the form for editing by other users.
