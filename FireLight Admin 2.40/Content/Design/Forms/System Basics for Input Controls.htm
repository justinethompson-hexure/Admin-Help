System Basics for Input Controls

  • New Controls – By default, new controls are added to the form page in the upper left corner. Drag the control to the desired location. If the same type of control is selected, the new control is added adjacent to the selected control.
  • Automatic Name Assignment – The system automatically assigns a name to the new control, and that name appears in the Name field of the Input Control section. The system-assigned name has the following format: “<ControlType>_<incremental number>”. For example, the first instance of a radio button control is automatically named RadioButton_0, and the fifth instance of a given control is automatically named RadioButton_4.
  • Field Properties – Click a control and click Field Properties on the left under Input Control to open the Field Properties dialog for editing. See Add Controls to a Form for descriptions of all field properties for each input control type.
  • Re-size Controls – Point to the edge of a control to drag the handles on the control to re-size it or use the Field Properties dialog for resizing the field.

  • Show Tags – Show Tags checkbox is anchored at the top right of the main window. Select the Show Tags checkbox to display the names of the form fields on the page. Show Tags override Show Tab Order on the form page.


  • Show Tab Order – Show Tab Order checkbox is anchored at the top right of the main window. Select the Show Tab Order checkbox to display the tab order numbers at the beginning of the input control names as shown in the pink numbered boxes in the example below. Clear the Show Tab Order checkbox to hide the tab order numbers.


  • Save Form Changes – If you click a different tab on the navigation bar before saving changes, a Form Changes dialog appears. Click Yes to save the changes to the form or click No to discard the changes.
