Address Autocomplete

NOTE: This service is not Health Insurance Portability and Accountability Act (HIPAA) compliant. The integrations are configured so that we can disable them for an organization, which we do if we enter into a HIPAA Business Associate Agreement (BAA) with the organization.
Spotlight Video: For additional information, see the Address Autocomplete, Address Validation, and Dr. Lookup Spotlight Feature Video.

Use the Address Autocomplete control for an address autocomplete field where the user begins typing an address and presses Enter to open a drop-down selection list of addresses to choose from provided by the Google Places database.

NOTE: You must set the Data Group for who it applies to (e.g., Owner) in the Address Autocomplete Properties and then select the same Data Group and the applicable Data Group Property in each of the associated address fields on the form.

  1. On the left of the Form Designer page under Add Control to Form, click Address Autocomplete.
  2. Drag the Address Autocomplete control from the upper left of the form to an address autocomplete search box location on the form. For example, drag an address autocomplete control to an area near the address entry fields.
  3. Drag the right side of the Address Autocomplete control to fill the form field space.

  4. Double-click the Address Autocomplete control or, on the left under Input Control, click Field Properties.

  5. Enter or change settings in Field Properties.

    For example, in the Address Autocomplete Properties, set Data Group to Owner. In the Owner_ResidentialAddress1 text box field properties, set Data Group to Owner and set Data Group Property to Line1. In each of the remaining address fields, set Data Group to Owner and set Data Group Property to the applicable option (Line2, City, State, Zip, etc.)

  6. Click the Update button to update the field properties and lock the form for editing.

  7. Click the Close button to close the Field Properties dialog.
  8. Click the SAVE button on the upper left of the Designer page to save the controls and field properties, which unlocks the form for editing by other users.
