Input Control Visibility Field Properties

When adding input controls, use the following field properties to set the visibility for the control.

Protection field property sets the protection level; visibility for the control was never implemented and may be implemented in the future.

None follows the master setting of the form in the Form Properties dialog (see Set Form Properties for master form settings). Individual input control settings override the master form setting.

Public is visible to everyone.

Private is visible only to the form designer.

Client is visible only to the client. This is used for the client’s private information, such as medical information.

Agent is visible only to the agent. This is used for agent information, such as commission splits.

Signer is visible only to the signer.

Reviewer is visible only to the reviewer.

Exclude from PDF field property sets whether the control appears when the form is printed on paper. By default, every control appears on the PDF. When this checkbox is selected, the control does not appear on the printed or submitted form.

Read Only field property sets the field as a read only control. Use this for displaying calculated values on the form where editing is not allowed (for example, fund allocation total).
