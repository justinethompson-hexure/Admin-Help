Add Controls to a Form

NOTE: See the "Common Tags" topic in Admin Help Common Tags for common tags to be used in mapping dataitems. Use of common tags ensures data is mapped correctly throughout the system, for example, inbound data, input controls, CRMs, etc.

  1. On the navigation bar, click Designer/Form Designer to go to the Form Designer page.

  2. On the upper left of the form, select a form to design.

  3. Add the following input controls to the form.

    Address Autocomplete

    Barcode

    Button

    Calendar

    Check Box

    Combo Box

    Label

    Radio Button

    Signature Box

    Signature Initials

    Text Box

  4. Click the SAVE button on the upper left of the Form Designer page to save the form, which unlocks the form for editing by other users.
