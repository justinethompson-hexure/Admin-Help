Add a New Rule from a Pre-Built Rule Template

  1. On the navigation bar, click Designer/Form Designer.

  2. On the upper left of the Form Designer page, select a form in the list.

  3. On the form, click the control (data item) for the rule.

    For example, click the Owner_DOB control.

  4. On the left under Input Control, click Rules Window.
  5. In the Filter Rule Sets box, type the name of the rule set if known and click .

  6. On the lower right in Selected Control verify the name of the selected control for the new rule.

    For example, ensure the Owner_DOB control is the selected control in the Rules Window.

    To view a list of all the rules associated with the form controls, clear the List only rules for selected control checkbox.

  7. On the upper right, click New.

  8. On the left in the New Rule Template window, select a template for the new rule.

    For example, select Date Valid.

  9. In the New Rule Template window, enter or verify the dataitemid value is set to the correct data item. For example, enter Owner_DOB. Click the Ok button to close the New Rule Template window.

  10. On the Rules Window in Rule Name, add a field identifier to the rule. For example, add the field name to the rule, as in Date Valid: Owner_DOB.

  11. In Rule Scope, select an option to indicate when the rule is executed.

  12. On the Rules Window, click Update to update the new rule for the control, and lock the rule set for editing.

    For example, checks to see whether the owner’s date of birth is a valid date.

    Click the X on the upper right to close the Rules Window.

  13. Click Yes on the Rule Set Changes dialog to save the rule set changes, which unlocks the rule set for editing by other users.

    NOTE: If the Validate Rule Name Uniqueness checkbox is selected on the Admin tab in Organization Configuration/Organization Settings, the new rule cannot be saved unless it has a unique name.

  14. After adding and saving a rule set, add the rule set to the applicable Packages.
