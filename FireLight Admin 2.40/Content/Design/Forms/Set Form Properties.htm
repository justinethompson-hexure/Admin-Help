Set Form Properties

The Form Properties dialog is where you set default form settings that affect the appearance and protection level of the form. These settings are used by the form except for where they are overridden by individual field properties.

  1. On the navigation bar, click Design/Form Designer.
  2. On the upper left of the Form Designer page, select a form name in the list.

  3. On the upper left, click the Form Properties link and select the following form settings on the Form Properties dialog. To change a setting, click the current setting to open a list of options and then click the preferred option.

    Font Family sets the default font family for the form (for example, Arial).

    Font Size sets the default font size for the form (for example, 12 point).

    Print Color sets the font print color (for example, Blue).

    Protection sets the protection level and visibility for the form. Be sure to use the Protection feature to hide any agent or review forms from the client.

    • Public is visible to everyone. Reviewer dashboard forms must always be set to Public.
    • Private is visible only to the Form Designer.
    • Client is visible only to the client. This is used for the client’s private information, such as medical information. Protection was not implemented in FireLight.
    • Agent is visible only to the agent. This is used for agent information, such as commission splits. The agent protected indicates that it is filled out by the agent and not visible in external Client Fill and Sign requests. As such, it is treated as "Office Use Only," and all required fields must be filled out before the Request Client Fill & Sign option appears in the Other Actions menu.
    • Signer is visible only to the signer.
    • Reviewer is visible only to the reviewer.
  4. Click the Update button to save the selected form properties and lock the form for editing.
  5. Click the SAVE button on the upper left of the Form Designer page to save the changes to the form, which unlocks the form for editing by other users.
