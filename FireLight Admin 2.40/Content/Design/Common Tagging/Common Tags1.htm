Common Tags

IMPORTANT! To enable Common Tags, go to the Organization Configuration/Organization Settings/Activity tab and select the Enable Common Tags checkbox.

FireLight® provides a Common Tag Name field in most field types available in Form and Wizard designers.

The Field Name can be named anything that applies, which may be a common tag. Distributors often create field names distinct from what the carrier is using. The Common Tag Name field is used to connect fields together. For example:

Carrier Field Name is ClientFirstName.

Distributor Field Name is D_ClientFirstName.

Both fields have a Common Tag Name of Owner_FirstName.

There are two ways to locate a common tag: using the search function while entering a field's Common Tag Name field property or by searching through the CommonTags.xlsx spreadsheet below.

The search returns the most logical matches and displays them in list format, starting with the closest match. You can enter multiple words. For example, assume you are looking for phone number tags for the owner. Simply enter owner phone number. A list containing the entered text from which the user can select a common tag name from the list appears. The user continues typing to refine the list.

Prior to searching, when the list first opens, due to the large number of common tags only a small subset of common tags appears. By default, the full list is not shown in the Common Tag Name dropdown.

NOTE: If the list does not appear, ensure the spelling is correct.

NOTE: If the results are not returning a valid list of values, you can also search for a different portion of the common tag name.

Common Tag Name Field Property

In support of common tags, when you enter a common tag name in the Field Name field property, the system automatically enters the common tag name in the Common Tag Name field property when you save or tab off the field. If the Field Name property is not a common tag name, a search function is available in the Common Tag Name field property in both forms and wizards.

EXAMPLE: Combo Box Form Field for Owner Gender

Following is an example of adding a Combo Box form field with a Common Tag field name and Common Tag List Name values.

  1. On the left under Add Control to Form, click Combo Box.

  2. Drag the Combo Box from the upper left of the form to the applicable field on the form and re-size it to fit the field space.

  3. Double-click the field or on the left under Input Control, click Field Properties.

  4. Enter or change settings in Field Properties.

    1. In the Field Name field, enter the name of the data item associated with the field. This does not need to be unique among all fields in the form or wizard. When a user enters a Field Name that exactly matches a common tag name, the system updates the Common Tag Name property to match the Field Name when the user saves or tabs off the Field Name field. For the following form designer example, click in the Common Tag Name property, begin typing Owner in the box, and select Owner_Gender from the list of common tags to autocomplete the Common Tag Name property.

    3. In the List Name Values field, click the Edit List Name Values link to open the List Name Values dialog as shown below. The system automatically populates the list of options for the gender.

    4. Use the scroll bar on the right to move through the list.
    5. Do any of the following to customize the list.

      • Clear the checkbox to hide the option from the user's view.

      • Click the up or down buttons to move options to new locations in the list.

      • Press and hold Ctrl and click the up arrow icon on the right of the row to move the row to the top of the list.

      • Press and hold Ctrl and click the down arrow icon on the right of the row to move the row to the bottom of the list.

      • Select the checkbox at the end of the list to add a row for a new option. Enter the option name in the Caption/Rules Value column and optionally add a value in the Rules Override column. In the following example, the user added a Binary option. The user can click the trashcan icon to remove the user-created option from the list.

      • Click the OK button to save the changes to the list.

      NOTE: The OK button is disabled if no row is selected or the Caption is blank in a user-created row.

    6. Enter or change the remaining field properties.
  5. Click the Update button to update the field properties.
  6. Click the Close button to close the Field Properties dialog.
  7. Click the Save button on the upper left of the Form Designer page to save the form.

Common Tag Spreadsheets

Click the CommonTags.xlsx link to view common tags. The file is organized by group name (e.g., Owner, Agent). Click the CommonTags_ListName link to view common tag list name values for options within a list (e.g., state and country codes on the Address tab).

Example:

  1. Go to the Common Tags topic in Admin Help.

  2. Open the CommonTags.xlsx spreadsheet.

  3. Click the Owner tab at the bottom of the spreadsheet.

  4. In the Common Tag column, locate the Owner_Gender common tag.

  5. In the ListName column, locate the List_Gender list name. You need the List_Gender list name to add the common tag list name values in Edit List Name Values.

  6. Open the CommonTags_ListName.xlsx spreadsheet.

  7. Click the Party tab at the bottom of the spreadsheet.

  8. In the ListName column, locate List_Gender. To the right in the ListValue column of the spreadsheet, note the caption and numeric export values for each of the gender options (for example, ListValue 1 = Male, 2 = Female, 3 = Unisex, etc.).
