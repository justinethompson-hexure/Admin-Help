Search for Fields in Wizard Packages

Use the Search Fields in Wizard Packages tab on the Wizard Library page to search all wizards within a package and list the wizards (or fields) that contain a specific field.

  1. On the navigation bar, click Designer/Wizard Library.
  2. On the lower left of the Wizard Library page, click the Search Fields in Wizard Packages tab.

  3. Enter the name of a field in Field Name.

  4. In Match, select either Exact or Anywhere for the type of match for the search field.

  5. In the Of Control Type list (search field), select a control type or select All.

  6. Select a package name from the In Package list or select All to search all packages.

  7. In the Forms in Package list, select All or select a specific form name.

  8. Click Search to initiate the search.

  9. (Optional) Click Export to export the field information to a spreadsheet in CSV format.

    Below the grid, click a button (or click an arrow next to a button and then click an option) to open or save the spreadsheet.
