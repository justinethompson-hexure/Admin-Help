View Wizard History

  1. On the navigation bar, click Designer/Wizard Library.

  2. Select a wizard in the grid at the top of the Wizard Library page.

  3. On the lower left of the page, click the Wizard History tab to view the history of the selected wizard.

  4. In Utilities, Reports page, Admin Reports, run the Admin History Report for Wizard History Audit Report to view an audit history report for all wizard changes during a selected time frame for a single wizard or all wizards.

NOTE: The system purges all wizard deployment and audit history that is more than one year old.
