Import a Wizard

Import a previously exported wizard.

  1. On the Home page or navigation bar, click Designer/Wizard Library.
  2. On the Wizard Library page, select a wizard in the list, or create a new wizard to import to (see Create a Wizard). The imported wizard overwrites the selected wizard on the Wizard Library page.
  3. On the Wizard Detail tab, click Import Wizard.
  4. Click OK on the Import Wizard confirmation dialog.
  5. Click Add Files, browse to the location of your exported wizards, and select the wizard to import.
  6. Click Start Upload to upload the imported wizard to the selected wizard on the Wizard Library page.
  7. After the upload completes, click Back to return to the Wizard Library page and click Save.
