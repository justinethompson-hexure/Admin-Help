Wizard Library

Click Designer/Wizard Library on the Home page or the navigation bar to open the Wizard Library page. Use the Wizard Library page to create and manage wizards.

NOTE: Hexure provides a Template Wizard (Annuity) and associated Rule Set that you can download from Admin Help and use as a starting point for creating a wizard of your own. See Add a Template Wizard and Rule Set for instructions.
