Create a Wizard

TIP: Hexure provides a Template Wizard (Annuity) and associated Rule Set that you can download from Admin Help and use as a starting point for creating a wizard of your own. See Add a Template Wizard and Rule Set for instructions.

NOTE: A wizard does not have a PDF associated with it so does not require a PDF upload.

To create a wizard for a navigation stop in a wizard user interface, do the following:

  1. On the upper left of the Designer/Wizard Library page, click Create Wizard.
  2. On the Wizard Detail tab, enter a Wizard Number, Wizard Name, Wizard Description, and select the states that apply in the State Approved list.
  3. (Optional) If you want to designate begin and end dates for the wizard: click in This Wizard Begins On and select a begin date on the calendar, click in This Wizard Ends On and select an end date on the calendar, and select a time zone in Effective Time Zone.
  4. Click Save to save the wizard.
  5. Repeat for each wizard you are creating for each navigation stop in the wizard user interface.

NOTE: Be careful when editing wizard details.

See Also: Wizard Designer for detailed instructions about designing wizards and setting up a wizard user interface.
