Export a Wizard

Export (save) a wizard with all of its metadata from FireLight® Admin. Exported wizards can be imported to new or existing wizards in the Wizard Library if the wizard is used in other regions within your environment, such as Staging, Production, Quality Engineering, etc., or if planning to use the same wizard multiple times; for example, when requiring proof of citizenship for joint or additional insured.

  1. On the Designer/Wizard Library page, select a wizard in the list.
  2. On the Wizard Detail tab, click Export Wizard.
  3. On the notification bar of the Internet Explorer web browser, click Open or click an option on the Save button to save the exported wizard.
