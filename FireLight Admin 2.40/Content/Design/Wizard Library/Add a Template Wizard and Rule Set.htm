Add a Template Wizard and Rule Set

Hexure provides a Template Wizard (Annuity) and associated Rule Set that you can use as a starting point for creating a wizard of your own. Use the following instructions to download/copy and set up the template wizard and rule set.

NOTE: This example uses 7-Zip (free software) for creating and extracting .zip file types. You need to have a Zip application (7-Zip, WinZip, etc.) installed on the PC to extract the template wizard files.

  1. In the Admin Help in Designer/Wizard Library, go to the Add a Template Wizard and Rule Set topic. Click the TemplateWizardandRuleSet.zip link Click this TemplateWizardandRuleSet.zip link to download the file. Click the file on the lower left of the window to open the 7-Zip application.

    NOTE: For the Internet Explorer browser, right-click the link above, click Save target as, browse to the location on the PC or network to save the file, and click Save. On the information bar at the bottom of the window, click Open to open the 7-Zip application.

    1. On the 7-Zip window, click Extract.

    2. In the Copy dialog, browse to the location on the PC or network to where the file is copied and click OK.

  2. Create a wizard.

    1. Go to the Designer/Wizard Library page.
    2. On the upper left, click Create Wizard.
    3. On the Wizard Detail tab, enter a Wizard Number, Wizard Name, Wizard Description, and select the states that apply in the State Approved list. The wizard name and number in this example are "Template Wizard Annuity."
    4. At the bottom of the page, click Save.

  3. Import the downloaded Template Wizard (Annuity).txt file.

    1. Ensure the wizard created in step 2 is selected in the grid at the top of the page.
    2. On the Wizard Detail tab, click Import Wizard.
    3. Click OK on the Import Wizard confirmation dialog.

    4. Click Add Files, browse to the location of the downloaded template wizard, and select the Template Wizard (Annuity).txt file.

      The Template Wizard (Annuity).txt file appears in the grid.

    5. Click Start Upload to upload the imported wizard to the wizard you created in step 2 on the Wizard Library page.
    6. After the upload completes, click Back to return to the Wizard Library page and click Save to save the imported wizard file.

  4. Create a rule set for the Template Wizard Rule Set (Annuity).

    1. Go to the Product Config/Rule Sets page.
    2. On the upper left, click Add New Rule Set.
    3. On the Rule Set Detail tab, enter the Rule Set Name and Description for the Template Wizard Rule Set (Annuity). For this example, the name and description are "Template Wizard Annuity Rule Set."
    4. Select the applicable State Approved checkboxes.
    5. At the bottom of the page, click Save Rule Set.

  5. Copy the rules from the downloaded Template Wizard Rule Set (Annuity).xml file to the rule set created in step 4.

    1. Locate and open the Template Wizard Rule Set (Annuity).xml file that was downloaded in step 1 using Notepad, Notepad ++, or any other text editor.
    2. On the Rule Sets page, ensure the rule set created in step 4 is selected, click the View All Rule Definitions link to open the Rule Definitions Window, and delete all the text from the window.
    3. Copy the rules from the Template Wizard Rule Set (Annuity).xml file, paste the rules into the Rule Definitions Window for the new rule set file, and click Update. Close the Rule Definitions Window.
    4. At the bottom of the Rule Sets page, click Save Rule Set.

  6. Create a product for the template wizard.

    1. Go to Product Config/Products page.
    2. On the upper left, click Add New Product.
    3. Enter the Product Name. In this example, the name is "Template Wizard Annuity."
    4. Select a Product Type and Product Category in the lists.
    5. Select the State Approved checkboxes for the wizard.
    6. At the bottom of the page, click Save Product Config.

  7. Create a package for the template wizard.

    1. Go to Product Config/Packages page.
    2. On the upper left, click Add New Package.
    3. On the Package Detail tab, enter a Package Name and select an ACORD Type in the list. In this example, the name is "Template Wizard Annuity" and the type is "Application (103)."
    4. On the Forms/Wizards in Package tab in the All Forms/Wizards list, move the Template Wizard Annuity to the Forms/Wizards in Package list and select Required in the list.
    5. On the Rule Sets in Package tab in the All Rule Sets list, move the Template Wizard Annuity Rule Set to the Rule Sets in Package list.
    6. At the bottom of the page, click Save Package.

  8. Create a sales agreement for the template wizard.

    1. Go to Product Config/Sales Agreements page.
    2. Select the Template Wizard Annuity product in the grid at the top of the page.
    3. Select a Selling Org in the list.
    4. Select the applicable checkboxes on the State Approvals tab.
    5. In the Available Packages list, locate and move the Template Wizard Annuity package to the Assigned Packages list and set Package Duty to Required.
    6. At the bottom of the page, click Save Sales Agreement.

  9. Grant access to the Template Wizard Annuity product.

    1. Go to Organization Config/Groups page.
    2. Select a user role that has access to the Template Wizard Annuity product.
    3. On the Product Access tab, select the Template Wizard Annuity checkbox.
    4. At the bottom of the page, click Save.
