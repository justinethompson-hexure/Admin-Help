My Preferences [Reviewer Instructions]

Reviewers use the My Preferences dialog to set some basic reviewer preferences (defaults) for the Review Queue.

Set Review Queue Preferences

  1. On the top navigation bar of the Review Queue page, the reviewer clicks My Preferences.

    My Preferences dialog opens.

  2. In Page size, the reviewer selects the number of orders to display on a Queue page.

  3. Warn before lock sets whether the Lock and Review Order dialog, shown below, appears when the reviewer clicks the Review button to begin reviewing an order in one of the queues. The reviewer selects an option in the list.

    • Yes sets the Lock and Review Order warning dialog to appear when the reviewer clicks the Review button on an order.

    • No disables the Lock and Review Order warning dialog, so when the reviewer clicks the Review button FireLightÂ® automatically locks the order without displaying the Lock and Review Order dialog.

    NOTE: Changes take effect when the reviewer logs off and logs back on to FireLight.

  4. On the My Preferences dialog, the reviewer clicks Save to save the setting, or clicks Cancel to close the dialog without changing settings.
