Review Queue Navigation [Reviewer Instructions]

The Review Queue page contains a top navigation bar, a left side navigation bar, and page navigation numbers.

Top Navigation Bar

NOTE: Pre-submission reviewers who access the Review Queue via a web link and a passcode see only the Log Off button.

The top navigation bar contains buttons to take reviewers to the following areas of FireLight®.

Home – opens the FireLight Home page.

My Preferences – opens the My Preferences dialog to set user preferences for the Review Queue.

Log Off – logs off and closes FireLight.

– opens the My Messages page to view messages from the message center.

– opens the (optional) customer support Contact Us dialog.

Side Navigation Bar

NOTE: Pre-submission reviewers who access the Review Queue via a web link and a passcode will see only the PreSubmit queue in the Queues section.

The side navigation bar on the left contains the following buttons for accessing Review Queue features. Reviewers click a section name to expand or collapse the section on the side navigation bar.

Review Queue Navigation Bar
Navigation Bar Descriptions

Queues contains links to the queues that reviewers can access. This is where you review, approve, reject or request more information for orders within the queues. Automated queues have a separate Automated Queues section.

History opens the History page where reviewers view application history, queue history, audit report and optional documents for each order.

Summary contains links to various report charts for the review queues.

Reviewers contains links for viewing individual reviewer action history.

Page Navigation

Reviewers click the numbers below the selection lists to navigate through multiple pages in a Queue or in History.
