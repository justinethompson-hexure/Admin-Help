Review Queue Page [Reviewer Instructions]

NOTE: Pre-submission reviewers who access the Review Queue via a web link and a passcode see only the Log Off button on the top navigation bar and the order for review in the PreSubmit queue in the Queues section.

Post-submission reviewers use the Review Queue page to review orders, set review preferences, and access review messages, queues, history, summary report charts, and reviewers.

The left of the page contains a side navigation bar for moving among the Review Queue features. The right side of the page contains the information for each order package within the selected feature. If no items are in a queue, a (0) appears to the right of the dimmed queue name and "No item found in this queue" displays on the right of the page.

NOTE: Additional information could appear under the name of the order package. This optional information can be organization specific and may not be available for the reviewer’s organization.
