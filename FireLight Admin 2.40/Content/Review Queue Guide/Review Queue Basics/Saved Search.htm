Saved Search [Reviewer Instructions]

  1. On the side navigation bar, the reviewer clicks a section name (Queues, History, or Reviewers) and the applicable queue name (in Queues) or reviewer name (in Reviewers) where the search was saved from the Advanced Search.

  2. At the top of the page in the Saved Search list, the reviewer selects a saved search.

    Example – Saved Search list

  3. The reviewer clicks the Advanced Search button to open the selected search in the Advanced Search dialog.

    Example - Advanced Search dialog displays the saved search

  4. The reviewer makes any applicable changes to the search options.

  5. The reviewer clicks one of the following buttons:

    • Search – initiates the search process and displays the search results on the right.

    • Cancel – cancels the search and closes the Advanced Search dialog.

    • Save – opens a Save Search dialog. The reviewer clicks Save to save changes to the saved search Name or search options or clicks Cancel to cancel saving the changes.

    • Delete – opens a Delete Search dialog. The reviewer clicks Delete to delete the saved search from the review queue or clicks Cancel to cancel the saved search deletion.
