Search [Reviewer Instructions]

  1. On the side navigation bar, reviewers click a section name (Queues, History, or Reviewers).

  2. On the right, the reviewer selects search options in the drop-down lists to filter orders in the selected section.

    Queues section list options:

    History and Reviewers sections list options:

    NOTE: In History, the Actions list includes search options for All Actions, Approved, Auto Removed, and Rejected.

  3. The reviewer enters a partial or full name of an order in the search box and clicks to search for a specific order.

    Example – Queues section

    In the Queues section, the reviewer can click the Export button to download a .csv file that contains the search results for the selected queue (see Export Data).

  4. Example - History or Reviewers section

  5. The reviewer clicks X on the right of the search button to clear the selections and search text to begin a new search.

NOTE: Pre-submission reviewers who access the PreSubmit Review Queue via a web link and a passcode see only the Log Off button on the top navigation bar and do not see the Search and Advanced Search options.
