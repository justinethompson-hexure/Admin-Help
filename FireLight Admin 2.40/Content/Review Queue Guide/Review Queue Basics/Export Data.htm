Export Data [Reviewer Instructions]

Users can download data from a selected Queue into a Comma Separated File (CSV), which can be used for further research. Users can export data from the default view or after performing a search in the Queues section.

To the right of the search input boxes, click the Export button to download the data for all the orders in the selected Queue (or all the search results for orders in the selected Queue) to a .csv file.

Sample view of the downloaded .csv file from the information bar in the Chrome browser at the lower left of the screen:

The .csv file contains data in the following columns:

  • User

  • Review Queue

  • Activity Name

  • Product

  • Policy Number

  • Issue State

  • Locked By

  • Status

  • Created (date/time)

  • Carrier

  • Jurisdiction

  • Agent Organization
