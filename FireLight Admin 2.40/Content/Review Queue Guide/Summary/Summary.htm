Summary [Reviewer Instructions]

Use the Summary section to view, create, print and save queue summary reports in chart format.

View, Create, Print and Save Summary Reports

  1. On the left navigation bar, the reviewer clicks Summary to expand the list of reports.

  2. Under Summary, the reviewer clicks a report to create the associated chart.

    In the following example, the Overall Report displays activity charts for all queues.

  3. The reviewer clicks the Print Report or Save button on the upper right to print or save the summary report charts.

    • The reviewer clicks Print Report button to generate and open the report in a separate window.

    • (Chrome) The reviewer clicks an option in the list to open and view the summary report file.
