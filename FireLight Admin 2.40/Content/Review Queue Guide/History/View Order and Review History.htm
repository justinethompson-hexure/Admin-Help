View Order and Review History [Reviewer Instructions]

  1. On the left navigation bar, the reviewer clicks History.

  2. On the right in the lists, the reviewer can select a date range, queue, and action (All Action, Approved, Rejected, Auto Removed, Deadline Passed) to filter and limit the number of orders. The reviewer can also use the Search [Reviewer Instructions] or Advanced Search [Reviewer Instructions] options to filter and limit the number of orders.

  3. If the organization has Additional Info (e.g., Product Name, Jurisdiction, etc.) added to orders in the Review Queue, the Additional Info also displays below the name on the orders in review History.

  4. The reviewer finds an order in the list. Approval or rejection status appears on the upper right with a green check mark next to an approved order and a red X next to a rejected order.

    If the review deadline expired, “Deadline Passed by [Reviewer Name]” indicates the review deadline passed and the order is unavailable for review.

    If the order package includes additional documents the agent uploaded (e.g., an illustration, etc.), the order history includes the additional document name.

  5. The reviewer clicks one of the following links to view activity documents, history, queue history, audit information, reviewer documents, and reviewer uploaded documents.

    • Application PDF link opens the activity documents for the order.

    • Application History link opens the History dialog for viewing the order history.

    • Queue History link opens the Queue History dialog for viewing the order review history.

    • [Document Name] link opens the agent attached document (e.g., “JaneClientVA2” illustration).

    • Audit Report link opens the FireLight® Packet Audit Report for the order.

    • Reviewer Documents link opens the reviewer documents for the order.

    • Reviewer Attached Documents link opens the uploaded documents the reviewer attached for the order. The display name is the document PDF filename. Select the checkbox on the right and click the View button to view the uploaded document.

    NOTE: The Reviewer Documents and Attached Documents remain with the order if the reviewer moves the order to another queue or the order is recalled after being rejected.
