Step 7. Approve or Reject an Order [Reviewer Instructions]

Approve an Order

  1. After reviewing the application documentation, the reviewer clicks the Approve button to approve the order.

  2. The reviewer clicks the Approve button on the Approve Confirmation dialog to approve the order and send an email to the agent that the order was approved, or clicks Cancel to cancel the approval and return to the queue.

Reject an Order

  1. After reviewing the application documentation, the reviewer clicks Reject to reject the order.

  2. On the Reject Confirmation dialog, the reviewer enters a rejection reason and clicks Reject, or clicks Cancel to cancel the rejection and return to the queue.

    The system sends an email to the agent indicating the order was rejected.

    NOTE: See Recall a Rejected Order and Return It to the Queue [Reviewer Instructions] on the History page to find how to recall a rejected order to the queue.

 

Approve or Reject Multiple Orders

When the reviewer opens an order for review, the checkbox in the upper right is automatically selected if no More Info requests are pending.

  1. The reviewer unlocks and reviews several orders and ensures the checkbox is selected for each order to approve or reject.

  2. The reviewer clicks the Approve All button to approve all of the selected orders, or clicks the Reject All button to reject all of the selected orders.
