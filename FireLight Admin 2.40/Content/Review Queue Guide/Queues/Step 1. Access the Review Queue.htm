Step 1. Access the Review Queue [Reviewer Instructions]

Access the Pre-Submission (PreSubmit) Review Queue

  1. A pre-submission reviewer receives a review request email with a URL link to FireLight® and receives a separate email or text with a passcode.

    Example – URL link and instructions in review email

    Example – Passcode email (with passcode highlighted)

  2. In the review request email, the reviewer locates and clicks the URL link to FireLight.

  3. On the Welcome dialog, the reviewer enters the code from the passcode email into Passcode and clicks Enter.

    The FireLight Review Queue page opens and displays the order in the PreSubmit queue.

Access the Post-Submission and Automated Review Queues

  1. Post-submission reviewers do one of the following to log in to FireLight.

    • If leveraging a single-sign-on implementation, the FireLight URL from the reviewer’s firm automatically passes the reviewer’s known credentials to the FireLight application.

    • Reviewer opens a web browser, goes to the FireLight application URL, and uses the login information provided by the firm to log in to FireLight.

  2. After successful log on, FireLight opens to the Home page. On the right under Manage, the reviewer clicks Review Queue.

  3. The reviewer accesses a queue by clicking a link under Queues on the side navigation bar. Each queue has its own page.

  4. Automated review queues reside in a separate Automated Queues section below the Queues section on the side navigation bar as shown below. The Automated Queues section is visible only when the organization has one or more automated queues set up in FireLight Admin.
