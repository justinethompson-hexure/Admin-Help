Step 6. Move, Assign, or Unlock an Order [Reviewer Instructions]

Move an Order to another Queue

NOTE: Reviewers can move only activities that are not in the PreSubmit review queue, have not been approved or denied in any queue, and are not awaiting a More Info response. If the organization has the Enable Manual Queue Move option enabled, along with the Deadline Length option and the Deadline Date - Continue From Previous Queue option enabled in FireLightÂ® Admin, the expiration deadline continues the deadline date to the next queue when moving an order to another review queue.

If the organization has the Enable Manual Queue Move option enabled in FireLight Admin, the reviewer can move an order to another review queue.

  1. The reviewer opens an order in the review queue and decides to move the order to another queue that is appropriate for the order review.

  2. The reviewer selects the review queue to which to move the order for review and clicks the Move to Queue button to move the order to the selected queue.

    NOTE: The Reviewer Documents and Attached Documents remain with the order if the reviewer moves the order to another queue.

    In the following example, an order in the High Issue Value queue also involves a 1035 exchange and is multiple contract sales, so the reviewer can select 1035 Exchange or Multiple Contract Sales and click Move to Queue to move the order to the 1035 Exchange or Multiple Contract Sales queue.

Assign an Order to Another Reviewer

The reviewer can manually assign a case to another reviewer as follows.

  1. The reviewer logs into FireLight and clicks the Review Queue button on the Home page.

  2. The reviewer opens an order in the review queue and decides to assign the order to another reviewer.

  3. The reviewer selects the reviewer to whom to assign the order and clicks the Assign Reviewer button.

  4. The "assigned to" reviewer receives an email that the order has been assigned to them and can proceed with the review.

  5. The reviewer logs in to FireLight and clicks the Review Queue button on the Home page.

  6. To show only the cases that have been assigned to them in the queue, the reviewer selects the My Locked orders option in the status list.

Unlock an Order

After the reviewer locks an application, the Review button changes to the Unlock button.

  1. The reviewer clicks the Unlock button to close an order that the reviewer previously locked for review, which releases the order to the queue so other reviewers can access the order and continue the review.

    NOTE: After the reviewer clicks Unlock, the button changes to the Review button.

  2. On Unlock the order dialog, the reviewer clicks Unlock and Release to the queue to proceed with unlocking and releasing the order to the queue. Click Cancel to leave the order locked.
