Step 4. Request More Info [Reviewer Instructions]

If a reviewer requires more information from the user who submitted the application, the reviewer can use the More Info button to contact the agent. The More Info button is available in the "forms only" and the "wizards + forms" UI.

  1. The reviewer clicks the More Info button.

  2. In the Request More Info dialog, the reviewer types the specific request in the text box and clicks the Send button or clicks Cancel to delete the request without sending it to the agent.

  3. In the review queue, an information icon and “Request Sent, Pending Reply” in red text appear below the order name. The reviewer clicks the information icon to view the application history.

    FireLight® sends a “Request More Info” email to the agent (submitter) and posts a Request More Info message in the submitter’s My Messages.

  4. The submitter opens the application, clicks the Reply link on the Review Queue message, enters the requested information in a text box below the Reply link and clicks Send.

    When the submitter sends a reply, an “Info Received” status appears on the Review Queue message and “Info Received” appears under the application name on the order in the Review Queue.

  5. The reviewer clicks the Info Received link to view the information supplied by the submitter.

  6. A “Review – Info Received” email appears in the reviewer’s email. It contains the response from the agent for the requested information.
