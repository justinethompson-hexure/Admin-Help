Attach Documents [Reviewer Instructions]

The reviewer can add documents to the application package.

  1. The reviewer clicks the Attach Documents link.

  2. In the Reviewer Documents dialog, Add Supplemental Document section, the reviewer drags and drops a single file from a system or network location to the drop area. Or the reviewer can click in the drop area to choose a file to upload on the Open dialog and click the Open button.

  3. On the Reviewer Documents dialog, the reviewer clicks the Upload button to add the document to the application package.

    NOTE: If the file size of the PDF document exceeds the maximum file size limit set by the reviewer’s organization, FireLight® halts the document upload and displays a notification message that the file is too large.

    The document appears in the Reviewer Documents list.

  4. The reviewer clicks the View link to open the document. The reviewer can remove the document by clicking the Remove link.
