My Messages [Reviewer Instructions]

Reviewers use My Messages to view sent and received reviewer messages within FireLight®.

View Messages

  1. On the top navigation bar, the reviewer clicks the messages button to navigate to My Messages.

    Message titles are based on order name. The messages are arranged by descending date order with the most recent messages for orders at the top of the list.

  2. The reviewer uses the scroll bar to move through the list of messages, or uses the search feature to locate a message by typing the order name in the search box and clicking Search.

  3. On the left, the reviewer clicks the down arrows to expand the list of messages for an order. The message title bar turns a different color when pointing to the message.

  4. In the list of messages for the order, the reviewer clicks the Inbox or Sent button to view the message contents.

    NOTE: Reviewers may be included in an optional Review Queue notification email distribution list that notifies reviewers when an order reaches a review queue. This option is set at the organization level and may not be available.

  5. When finished with all of the order messages, the reviewer clicks the up arrows on the upper left to collapse the list of messages for the order.

  6. The reviewer clicks the Back button on the top navigation bar to return to the Review Queue page.
