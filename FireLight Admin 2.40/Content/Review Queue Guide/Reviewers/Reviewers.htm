Reviewers [Reviewer Instructions]

The reviewers section displays orders for each reviewer. Reviewers use the Reviewers section to view reviewer order action history.

View Reviewer Order Action History

  1. On the left navigation bar, the user clicks Reviewers to expand the list of reviewers.

  2. On the right, the user uses the Search [Reviewer Instructions] and Advanced Search [Reviewer Instructions] features to filter the number of reviews.

  3. Under Reviewers, the user clicks a reviewer ID to access the selected reviewer’s orders.

    The list on the right displays the queue name, approval status, order name and date of the order status action.

  4. On the right, the user clicks an order in the list to expand and show the review history for the order.

  5. The user clicks the links to view the order documents including, Application PDF, Application History, Queue History, agent attached documents (e.g., JaneClientVA2), Audit Report, and any reviewer Attached Documents.

  6. The user clicks the order row again to hide the review history for the order.
