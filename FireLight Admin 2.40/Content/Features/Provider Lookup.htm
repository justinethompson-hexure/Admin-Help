Provider Lookup - NPIS Integration
Description, Setup, User Experience and Additional Information

Description

This feature integrates FireLight® with the National Provider Identifier Standard (NPIS) database for searching for a specific health care provider or office inside FireLight via a button or custom action using the existing CRM functionality to pre-fill the provider data into the associated fields.

Admin Setup

(See Add Multiple Copies of a Form for use on multiple-instance forms.)

  1. Designer/Form Designer or Designer/Wizard Designer:

    1. Add a Button (forms) or a Custom Action (wizards) to a form or wizard.

    2. Set the following Button or Custom Action Properties:

      • Data Group - select Provider or Provider 2 as applicable.
      • Custom Action - select CRM.
      • CRM Type - select NPIS.
    3. Click OK to save the Button or Custom Action.

    4. In the field Properties for all of the associated fields, set Data Group to Provider or Provider 2 as applicable and set the Data Group Property to the applicable field (e.g., BusinessName, FirstName, LastName, BusinessName, Occupation, etc.).

      Common tags for mapping on the forms and wizards are:

      Provider_FirstName

      Provider_MiddleName

      Provider_LastName

      Provider_BusinessName

      Provider_Gender

      Provider_MailingAddress1

      Provider_MailingAddress2

      Provider_MailingAddress_City

      Provider_MailingAddress_State

      Provider_MailingAddress_Zipcode

      Provider_Business_PhoneNumber

      Provider_Occupation

  2. Deploy: Deploy the forms/wizards changes to the applicable environments.

User Experience

  1. Create a new activity that includes the provider lookup button or custom action for the NPIS CRM.
  2. Click the provider lookup button, enter a full or partial provider last name or business name in My Contacts, select a State in the list, and click the search button.
  3. In the results list on the left, click a provider name to view the provider information on the right.
  4. Click the Select button to pre-fill the data into the associated provider information fields on the form or wizard.
