Comm100 Chat Service Integration
Description, Setup, User Experience, and Additional Information

Description

FireLight® can integrate with Comm100 Chat Service.

Comm100 provides a live chat service for customer support. Their online portal allows for real-time communication and email between the User and Agent. Users is able to open the Comm100 dialog and ask questions when help is needed.

Comm100 integration does not require any changes to the Provider for your organization.

Spotlight Video: See the FireLight Comm100 -Chat- Integration spotlight feature video for training instructions.

Comm100 Chat Service Setup

  1. FireLight clients contract directly with Comm100 to set up an account (https://www.comm100.com).

  2. Comm100 provides setup instructions and customization instructions for the chat button.

Admin Setup

In Organization Configuration on the Organization Settings page, Admin tab, set up the Comm100 configuration by adding the configuration properties and values in the Third-Party Service Config dialog.

  1. Click the Click to Edit link to the right of Third-Party Service Config.
  2. Click the Add New link at the top of the dialog. In Config Name, type SiteId. In Config Value, type the Site ID provided by Comm100. In Service Type, select Comm100 and click the Add Config button.
  3. Click the Add New link at the top of the dialog. In Config Name, type PlanCode. In Config Value, type the Main Code Plan ID provided by Comm100. In Service Type, select Comm100 and click the Add Config button.
  4. Click the Add New link at the top of the dialog. In Config Name, type PrimaryServer. In Config Value, type the primary Chat Server for your Comm100 service plan provided by Comm100. In Service Type, select Comm100 and click the Add Config button.

  5. Click the Add New link at the top of the dialog. In Config Name, type SecondaryServer. In Config Value, type the secondary Chat Server for your Comm100 service plan provided by Comm100. In Service Type, select Comm100 and click the Add Config button.

    NOTE: Config Group is an optional configuration used by the DocuSign Integration integration.

    Examples:

  6. Close the dialog.
  7. Click the Save Organization button on the Organization Settings page.
  8. In Utilities on the Deployments page, deploy the Organization Configuration to the applicable environments. See Deploy Data.

User Experience

  1. Begin an application or other activity.

  2. Click the Chat button on the lower right of the FireLight web page to begin the chat session.
