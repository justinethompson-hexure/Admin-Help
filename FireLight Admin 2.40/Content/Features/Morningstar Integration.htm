Morningstar Integration

Morningstar Integration Feature
Description, Setup, User Experience, and Additional Information

Description

FireLight can integrate with Morningstar to pull the Annuity Intelligence (AI) report and the contents back into Firelight from Morningstar. This workflow includes the old and new contract fees and can help firms remain compliant with FINRA requirements to ensure surrendering a contract is a suitable decision.

  • FireLight clients contract directly with Morningstar to obtain their user name and password.
  • Common tags are given to the data points from Morningstar. This allows the Morningstar AI data to populate any forms by leveraging common tags. See the Design Document under "Additional Information" below for a list of standard common tags used with Morningstar.
  • You may want to force the upload of the Annuity Intelligence Report before reaching 100%.

Admin Setup

  1. Set up the Morningstar credentials in Organization Configuration on the Organization Settings page, Admin tab.

    The following inputs are case sensitive.

    1. Click the Click to Edit link next to Third-Party Service Config.

    2. Click the Add New link at the top of the dialog.

    3. In Config Name, type Username. In Config Value, type the Username Morningstar provided for your organization. In Service Type, select Morningstar and click the Add Config button.

    4. Click the Save Organization button on the Organization Configuration/Organization Settings page.

  2. In the Form Designer, add a custom action button control to the surrender contract comparison forms. You will need to configure when the button appears.

    1. Open the form.
    2. Add a Button control to the form.
    3. In the Field Properties, type Launch Morningstar in Text, select Custom Action in Action Type, and ensure Morningstar is the Action Parameter.

  3. In Product Configuration on the Packages page, add the forms with the new button and data mapping to the applicable product packages.

 

User Experience

  1. Begin an application.
  2. On the surrender contract comparison form, click the Launch Morningstar button.

  3. In the Morningstar dialog, fill in the applicable information and click Generate XML.

  4. Morningstar generates the report and sends the details back to FireLight by populating the common tag mapped fields on the surrender contract comparison form.
