Description, Preparation, Setup, User Experience, and Additional Information

Description

Short Life Quote Demo Video: Quote to e-Application

NOTE: Quote for annuity products is not yet fully functional.

IXN provides a Life Quoting tool that is embedded in FireLight, which includes the ability to print, email and text the proposal to the advisor, and to launch into an E-Ticket or a FireLight® e-Application or Illustration.

  • IXN is not expected to be WCAG level A compliant for this release. We will work with IXN to meet compliance in a future release.

  • IXN does not currently support Localization.

  • IXN Life Quotes can be saved, shared and transferred.

  • A Quote Activity Report is available. See Detailed Activity Reports in Admin Help for a description and sample report.

  • FireLight is embedded in IXN, which allows IXN users access to FireLight.

The ability to view quotes depends on the type of organization:

  • A carrier organization can see only their own annuity and life quotes, which means they cannot see other carriers’ quotes.
  • A distributor organization can see all carrier annuity and life quotes.

There is a licensing component for the Quote activity. Contact your Strategic Account Manager for details.

NOTE: Life products are Guaranteed Universal Life, Term Life, Whole Life, and Final Expense Insurance.

Admin Setup

When you license the Quote activity, Hexure adds the Quote Activity to the grid at the top of the Activities page in Product Configuration. Also have an additional DocType of IXNQuote with a Document Tag of IXNQuote added to your organization's List Profile.

Do the following to set up FireLight and Life Quotes and E-Ticket.

Organization Configuration/Organization Settings/Security tab (FireLight Embedded in IXN):

With the enhancement to IXN for embedding the FireLight e-Application, IXN certificate/service record enhancements are required within FireLight to easily support the connection between the two systems.

Enhanced certificate support allows ease of certificate security management for IXN access and updates to the IXN certificate expiration that eliminates the need for user interaction. This further expands the client access to FireLight and IXN clients alike for both systems. See Security for more information.

Add a new integration service record for enhanced certificate support to set up a relationship for IXN.

  1. Below the supported services table, click the Add New Integration Service Record link to open the Integrated Services Configuration dialog.

  2. In the Name list, select IXN Integrated Service. Select the applicable Supported Services checkboxes (Embedded, Illustration API, API Access, Token SSO).

  3. In Client Secret, click the New button. On the Confirm dialog, click Update. Next, click Save.

  4. FireLight adds the IXN Integrated Service to the table.

Organization Configuration/Organization Settings/Admin tab (IXN Embedded in FireLight):

Set up the IXN service credentials. IXN supplies the configuration ALQ_ID and value.

  1. Click the Click to Edit link next to Third-Party Service Config.

  2. Click the Add New link at the top of the dialog. In Config Name, type ALQ_ID. In Config Value, type the ALQ_ID value provided by IXN. In Service Type, select IXN and click the Add Config button.

NOTE: Config Group is an optional configuration used by the DocuSign Integration integration.

Examples:

Organization Configuration/Groups page:

  1. Select a user group in the grid at the top of the Groups page.

  2. On the Activity Access tab, select the Quote checkbox to enable the Quote activity.

    Change the Quote Email Expiration number of days setting as applicable for your organization. The default is 3 days, which means the email will expire at the same time on the third day (72 hours) from the time it is sent to the client.

  3. On the App tab:

    • Select the checkbox for the type of Quote to disable if both quote types are not sold by the user group:

      • Disable Annuity Quotes disables the Quote activity for annuity quotes. Annuity product quotes are not yet functional.
      • Disable Life Quotes disables the Quote activity for Life quotes
    • Select the Sales Agreement Products Only checkbox to filter quotes based on whether a sales agreement exists for a product with a package that has an ACORD Type of Application 103. When selected, only the quotes with a sales agreement and a package with an ACORD Type of Application 103 are shown on the Quote page if the logged-in user has access to the products. This applies to carrier sponsored and InsTech Managed products.
  4. Click Save.

Enable Quote Activity in FireLight

Prerequisite Checklist

Did the request come from a JIRA ticket?

  • If YES, confirm with the appropriate SAM that the client has a subscription for these services before proceeding, and check to see if a WO is needed. An Active subscription for ALQ/AAQ is required before enabling in FireLight.

  • If NO, this should be implemented via a signed WO provided to Client Delivery – in this case, you can assume there is an active subscription.

Has the Distributor/Agency been set up already as a firm/agency in the IXN platform? This must be completed prior to enabling the Quote functionality in FireLight.

Get the information for the Config Value needed for both ALQ and AAQ prior to starting your setup. It will be a unique ID that is a combination of numbers and letters separated by dashes from the IXN Admin Dashboard.

  1. Verify the Quote Activity is configured.

  2. In the Admin Tool, under Product Configuration, click the Activities button.

  3. Locate the Quote activity; this should be configured with the default settings, and there is no need to change anything. This step just confirms that the Quote activity is configured with the defaults.

  4. Enable Quote Activity for the appropriate Groups (Role Codes).

  5. From the main dashboard, under Organization Configuration, click the Groups button.

  6. On the Groups screen, select a role code for which to enable the Quote Activity.

  7. Once the Group is selected, click the Activity Access tab and select the Quote Activity Option check box.

  8. On the App tab in the Groups window, ensure that the option to Disable certain quote types is not selected–there is an option to Disable Annuity Quotes and an option to Disable Life Quotes. Review these settings based on the services the client has contracted for and update accordingly.

  9. Click the Save button at the bottom of the screen to save the updated settings.

  10. Repeat steps 6-8 for each Group (role code) that needs access to the Quote activity.

  11. Add a New Integrated Service Record to connect to IXN

  12. Click the Organization Settings button under Organization Configuration.

  13. Click the Security tab on this screen.

  14. Click the Add New Integrated Services Record at the bottom of the screen.

    15. The Integration Service Records are available for known integrated services. Only one record can exist per integrated service type.

    Note: Upon expiration, FireLight will automatically update the IXN certificate, so clients do not have to update the certificate in the Organization settings.

  15. In the Name list, select IXN Integrated Service.

  16. Select the applicable Supported Services checkboxes.

  17. In Client Secret, click the New button.

  18. On the Confirm dialog, click Update, then click Save.

    19. FireLight adds the IXN Integrated Service to the table.

  3. On this screen, next to the Third Party Service Config, click Click to Edit.

  4. This opens a new window. If the client has other third-party configs already set for other FireLight integrations, they appear here (i.e., Salesforce, Redtail, etc).

  5. At the top of the screen, click Add New at the top.

    6. New fields will be displayed.

  6. You will need to add a new record for both ALQ and AAQ if both services are in scope:

  7. For ALQ: Config Name: ‘ALQ_ID’ Config Value: This can be obtained from the Admin Dashboard for the client in the IXN platform; Service Type: IXN.

    1. Click Save Organization when done.

User Experience - Life Products

  1. On the Home page under Start New or on the New Activity menu, click Quote to open the Quote wizard.
  2. On the Quote toolbar, click Life.

  3. In the left column Proposed Insured section, enter the information for the proposed insured.

    • First Name

    • Last Name

    • Date of Birth or Age

    • Gender

    • Tobacco Use

    • State

    • Health Options - The number of health options may need to be reduced if other options are selected. For example, you cannot select all health options with all carriers, but all health options can be selected with a single carrier.

  4. In the center column Product section, select the product types (Term Periods, To Age, Term ROP, Whole Life, and Final Expense) and carriers to filter the quotes by product and carrier. Up to five product types can be selected when the user selects all carriers. When the number of carriers is limited, more than five product types can be selected.

  5. In the right column Coverage and Agent sections, enter or select the coverage and agent information. Up to three Face Amounts can be entered. Optionally, click the Calculate Needs link to calculate the face amount based on the Debts and Expenses, Income, and Assets.

  6. On the lower right of the wizard, click Run Quote.

    The Quote Results contain the client information, Coverage, and Agent information followed by the number of results and a search field. Term Quote results display on separate tabs for each Term duration. Click Show Details at the bottom of a product quote to show product details. Click a tab to show product Features, Forms, Carrier Overview, Premiums, and Important Information on separate tabs. Click Hide Details to hide product details.

  7. (Optional) Click the button to the right of the search field to expand the dialog. Enter a Carrier name if applicable. Select search options to Sort By, Group By, Filter By, and select Product Features to filter the quote results.

  8. On one or more tabs for the desired term period, select the checkbox for one or more product quotes. The Share and Compare buttons become visible.

  9. Click the Compare button to open the Comparing Quotes dialog. Use the scroll bar to scroll left and right through the columns of the comparison. On the lower right of the comparison, click the Share button to select a delivery method and share the Quote.

  10. Click the Share button above the product selection or click the Share button on the Comparing Quotes dialog to open the Select a Delivery dialog.

    On the Select a Delivery Method dialog, click a Download, Email, or Text button for the delivery method and click the Next button. Download is the default method.

    On the Select a PDF Layout dialog, click a PDF layout for the quotes: Detailed Comparison, Condensed Premium Comparison, Premium Comparison, or Feature Comparison and click the Next button.

    On the Enter Information dialog, enter Agent Information and ensure Recipient Information is correct. Click the applicable Download, Email, or Text button for the delivery method.

    On the Success dialog:

    • Download delivery method: Click View PDF to open and view the quote PDF document.

    • Email delivery method: Click Send to deliver the email to the client.

    • Text delivery method: Click Send to deliver the text to the client.

    Click the X on the upper right to close the dialog.

  11. Save the quote: on the navigation bar, click Save. In the Save Quote dialog, enter a Name for the quote and click Save. Save the quote under a new name: in the Other Actions menu on the navigation bar, click Copy Activity. In the Copy Activity dialog, enter a new name for the quote and click OK.

  12. On the Quote page, to the right of the Quote, pause the mouse (hover) over the premium amounts of a quote and click the Get Started button.

  13. On the Pick An Action dialog, click one of the following buttons:

    • Apply – begins an application in FireLight (if E-Application is enabled)

    • E-Ticket – begins an E-Ticket in FireLight (if E-Ticket is enabled)

    • Illustrate – begins an illustration in FireLight (if Illustration is enabled)

    • Download PDF – download PDF delivery method

    • Text PDF – text PDF delivery method

    • Email PDF – email PDF delivery method

  14. Complete the application, e-ticket, or illustration in FireLight.

    Illustration: complete the illustration and click the Apply button to create an application.

    Application: create an application. When data entry is completed, click Continue to move to the next step in the process until signatures and any applicable review are complete.

    E-Ticket: complete the Quote Ticket App. When data entry is completed, click Continue to submit.

    NOTE: In the application or e-ticket on the Other Actions menu, click Display/Print PDF, select the documents to print including the Quote, and click Print Selected Documents to create a PDF file with the selected Application or E-Ticket and quote documents.

  15. On the upper right of the Life Quote page, click the New Quote button to start a new quote.
