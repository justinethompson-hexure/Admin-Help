Quote
Description, Preparation, Setup, User Experience, and Additional Information

Description

There are two types of quotes in FireLight®. Users are able to use the Annuity quoting provided by FireLight and the embedded IXN tool for Life quoting capabilities. Data is transferable between the two quote product types.

  • The FireLight Annuity Quote activity includes a standard wizard for data entry along with the ability to upload product rates and rules through the Admin. The Annuity Quote layout displays information on each quote, which includes the ability to launch product documents, such as a product guide, to print a proposal and email the proposal to the advisor or client, and to launch into an Illustration or e-Application. The Quote activity supports multi-product selection, column sorting, and filtering to easily find the appropriate quotes in the list. Once the quotes are selected, you can create a PDF proposal and send the quote to the advisor or client in an email. Save logic is included for the activity. An audit will show an entry in the Illustration or e-Application indicating it was created from a quote along with the PDF Proposal output for the quote in Other Action, Documents.

  • IXN provides a Life Quoting tool that is embedded in FireLight, which includes the ability to print, email, and text the proposal to the advisor and to launch into a FireLight e-Application.

    IXN is not expected to be WCAG level A-compliant for this release. We will work with IXN to meet compliance in a future release.

    IXN does not currently support Localization.

    FireLight is embedded in IXN, which allows IXN users access to FireLight.

The ability to view quotes depends on the type of organization:

  • A carrier organization can see only their own annuity and life quotes, which means they cannot see other carriers’ quotes.
  • A distributor organization can see all carrier annuity and life quotes.

There is a licensing component for the Quote activity. Contact your Strategic Account Manager for details.

NOTE: Life products are Guaranteed Universal Life, Term Life, Whole Life, and Final Expense Insurance. Annuity products are Fixed, Fixed Indexed, and Multi-Year Guaranteed Annuity products.

NOTE: There are different rates and rules for fixed and fixed indexed annuity products.

Annuity Proposal Templates

Hexure provides two proposal templates for annuity product quotes. If it exists, the client logo is added automatically to all proposal templates. The templates are standard and cannot be modified by clients. Provide your logo in .png format to the Strategic Account Manager for adding it to your organization so it appears in the Quote proposals.

  • Annuity Basic Proposal displays quote information for each selected product in list format.
  • Annuity Comparison Proposal displays quote information for each selected product, up to three products per page, in side-by-side columnar format. If more than three products are selected, the next three selected products display after the first three selected products on a new set of pages and so on until all selected products are displayed.

Annuity Products

Annuity Rates

Prepare the rate files for the fixed, fixed indexed, and multi-year guaranteed (MYGA) Annuity products.

  • Rate files can be in Excel .xlsx or .csv file formats.
  • You can supply rates for multiple products in a single rate file or in multiple rates files.
  • You can supply multiple rates for a single product to support different product rates by firm.
  • Ensure the rate files for the products are saved in a secure location with a meaningful name.

  • Rate files need to be in the following format for Fixed and Fixed Indexed Annuity Products.

    NOTE: Hexure can convert existing rate files into the FireLight format if desired.

Rates - Fixed and Multi-Year Guaranteed Annuity Products

There are 19 columns in the following order:

  • Product Type for Fixed Annuity products in FireLight is 5 for Fixed and 16 for Multi-Year Guaranteed. This is a required field.
  • Product Name is the name as it should appear in FireLight. Note that the carrier name appears on the Quote page and the PDF Proposal output so you do not need to include the carrier name in the product name. This is a required field.
  • CUSIP is a unique value for each annuity product. It is alphanumeric format and must be nine characters. This is a required field.

  • Rider Name is entered when rates vary based on rider. If applicable, add one row for each rider variance in addition to the base policy rates. NOTE: The Rider Name and Rider Product Code are both required when entered in a specific row. This is not a required field.

  • Rider Product Code is entered when rates vary based on rider. Rider Product Code that matches PPfA for rider listed in "Rider Name" column. PPfA path = /TXLife/TXLifeRequest/OLifE/PolicyProduct/AnnuityProduct/FeatureProduct/FeatureOptProduct/
    ProductCode. If applicable, add one row for each rider variance in addition to the base policy rates. NOTE: The Rider Name and Rider Product Code are both required when entered in a specific row. This is not a required field.

  • Jurisdiction is formatted as a unique number to group jurisdictions for the applicable rates. This is a required field. You can calculate the number following these steps:

    • Select the jurisdictions that apply. For assistance, use the Select All and Clear All buttons in cells K1 and L1.

    • When finished, copy the number in cell J1 and paste it into the Jurisdiction column on the Rate File tab using the Paste Values option in Excel.

    • You can include rates for multiple jurisdiction groups in a single rate file or break them out into separate rate files.

  • Distributor ID is the Firm ID that applies to the rates and is used to filter which firms will see the rates for the product. The format is typically four characters (but may be longer) and alphanumeric. This is not a required field because blank is allowed.

    • If the rates apply to all firms, leave the distributor ID blank.
    • If there are rates for all firms but one, provide two sets of rates, one with the distributor ID blank and a second set with the distributor ID for the specific firm. Users logging in with the specific firm ID will see these rates and all other users will see the rates with the blank distributor ID.
    • If a product is specific to one firm, set the distributor ID for the specific firm. Users logging in with the firm ID will see only the product.

    • Distributor ID is equal to the organization level DTCC Member ID. If you are not a DTCC Member, an ID is assigned by Hexure.

  • Minimum Premium Band and Maximum Premium Band are the premium bands for the rates. These values must be positive numeric values with two decimals. These are required fields.

  • Free Withdrawal Year 1 is the year one free withdrawal option. Valid options are a percentage, such as 0.00 for 0% or 0.10 for 10%, Interest, or None. You can specify the percentage as any numeric value up to 1.0 or 100%. This is an alphanumeric format. This is a required field.
  • Free Withdrawal Years 2+ is the year two and greater free withdrawal option. Valid options are a percentage, such as 0.00 for 0% or 0.10 for 10%. You can specify the percentage as any numeric value up to 1.0 or 100%. This is a numeric format. This is a required field.
  • Interest Guaranteed Years is the number of years the rate is guaranteed, such as 5. This is a required field.
  • Minimum Rate is the guaranteed minimum interest rate (GMIR) on a fixed annuity and the guaranteed interest rate on a multi-year guaranteed annuity. The format is numeric, such as 0.04 for 4.00%. This is a required field.
  • Current Rate is the current interest rate on the product. The format is numeric, such as 0.0425 for 4.25%. This is a required field.
  • Base Rate is the base interest rate on a fixed annuity. Beginning in year 2 and each year thereafter the projected interest rate projected is the base rate. The format is numeric, such as 0.03 for 3.00%. This is a required field.
  • Current Yield to Surrender Rate is the annualized yield of the annuity calculated over the time the surrender charge exists on a fixed annuity; it is calculated using the guaranteed interest rate, if guaranteed, then it uses the base rate. The format is numeric, such as 0.0525 for 5.25%. This is not a required field.
  • Guaranteed Yield to Surrender Rate is the annualized yield of the annuity calculated over the time the surrender charge exists on a fixed annuity; it is calculated using the guaranteed interest rate. The format is numeric, such as 0.0325 for 3.25%. This is not a required field.
  • Premium Bonus Rate is the rate used in upfront premium bonuses, which are the first year interest rate bonuses for fixed annuities. The format is numeric, such as 0.07 for 7.00%. This is not a required field.

  • Effective Date is the date the rates become effective for the product. The format is mm/dd/yyyy. This is a required field.

    NOTE: If you need to replace an existing rate file with the same effective date, simply upload a new rate file, which will completely replace the existing rates loaded for the product. This must be a complete rate file. If you need to replace rates with a more recent effective date, upload a new rate file with a complete set of rates. This will replace all previous rates, including additions and deletions in the replacement file. You can use an effective date in the future. These rates will not become effective until that date is reached. In this example, previous rates is shown until the effective date of the new rates is reached.

Annuity Admin Setup and User Experience

Admin Setup

When you license the Quote activity, Hexure adds the Quote Activity to the grid at the top of the Activities page in Product Configuration. Do the following to set up FireLight Admin for Quote.

Organization Configuration/Groups page:

  1. Select a user group in the grid at the top of the Groups page.

  2. On the Activity Access tab, select the Quote checkbox to enable the Quote activity.

    Change the Quote Email Expiration number of days setting as applicable for your organization. The default is 3 days, which means the email will expire at the same time on the third day (72 hours) from the time it is sent to the client.

  3. On the App tab:

    • Select the checkbox for the type of Quote to disable if both quote types are not sold by the user group:

      • Disable Annuity Quotes disables the Quote activity for annuity quotes
      • Disable Life Quotes disables the Quote activity for Life quotes
    • Select the Sales Agreement Products Only checkbox to filter quotes based on whether a sales agreement exists for a product with a package that has an ACORD Type of Application 103. When selected, only the quotes with a sales agreement and a package with an ACORD Type of Application 103 are shown on the Quote page if the logged-in user has access to the products. This applies to carrier sponsored and InsTech Managed products.
  4. Click Save.

Organization Configuration/Email page:

  1. In Email Category, select QuoteQuote Email Category.
  2. Make any desired modifications to the Client Quote Proposal and Annuity Rate Quote email templates that reside in the Quote email category in Organization Configuration on the Email page.

  3. Click Save Email Template.

Add Company Ratings

You must add company ratings for annuity quotes at the Organization level.

  1. On the Organization Configuration/Organization Settings/Organization tab, enter the Company Ratings for A.M. Best.

    NOTE: Only the A.M. Best rating is shown in Annuity quotes. The A.M. Best rating is required for annuity quotes since the rating field is located on the annuity quote wizard and used to filter the quote results.

  2. Click Save Organization.

Add a Product

Add your carrier annuity products for the Quote activity in FireLight. If a product, application and/or illustration package, and sales agreement are set up in FireLight for the product, this enables the ability for users to start an application or illustration from the Quote activity.

1. On the upper left of the Products page, click Add New Product.

2. Enter the Product Name, Product Category, and CUSIP.

3. Select the applicable State Approved checkboxes.

4. At the end of the page, click Save Product Config.

NOTE: Add the Annuity products that is used for the Quote activity for licensed carriers in FireLight. See "Add a Product" in Products for more information.

Add a Product Package

  1. On the upper left of the Product Configuration/Packages page, click Add New Package.
  2. In Package Name, enter the product name.
  3. In ACORD Type, select Application (103) for an application package or select Illustration (111) for an illustration package.

  4. Ensure Area Type is set to Activity.

  5. Click the Forms/Wizards in Package tab. Add the forms and wizards to be included in the package to the Forms/Wizards in Package list.
  6. Click the Rule Sets In Package tab. Add the Rule Sets to be included in the package to the Rule Sets In Package list.
  7. At the end of the page, click the Save Package button.

Add a Sales Agreement.

  1. In the top section of the Product Configuration/Sales Agreements page in the grid, select an annuity product.
  2. Move the packages created for the product (e.g., product Illustration package and product Application package) from the Available Packages to the Assigned Packages list.
  3. Select the applicable State Approvals checkboxes.
  4. At the end of the page, click Save Sales Agreement.

User Experience

  1. On the Home page under Start New or on the New Activity menu, click Quote to open the Quote wizard.
  2. On the Quote toolbar, click Annuity.
  3. In Jurisdiction, select a jurisdiction in the list.
  4. In Product Type, select the type of product (Fixed, Fixed Indexed, Multi-Year Guaranteed) in the list.
  5. Enter the Age of the client.
  6. Select the A.M. Best Rating to filter quotes.
  7. In Product Information select a Market Type (Non-Qualified, Qualified) in the list and enter an Initial Premium Amount.
  8. In the Agent section, optionally select the Enter Agent Name checkbox to add the agent First Name and Last Name to the quote proposal. It defaults to the agent first and last name from the Agent section in the Preferences page.

  9. To filter the Annuity quotes, select the applicable Carriers, Features and Riders, and Surrender Charge Years (Fixed Indexed) or Guarantee Periods (Fixed or MYGA), or select the Select All checkbox above each column to select all.

  10. When data entry is complete, the tool bar above the Quote displays Alerts with a green check mark.
  11. On the lower right of the wizard, click Get Quote to display the quote results. The Quote Results grid columns contain the carrier name, product name and A.M. Best financial rating, various product rates, and an Actions column with links to available product documents. If the product is carrier sponsored, the Actions column can also contain an Apply link to start an application and an Illustrate link to start an illustration. If the product is not carrier sponsored, the Actions column can also contain a Print link to start an application, which can be completed and printed.

  12. In the column title row of the Quote page, do any of the following for viewing and sorting quote results:

    • On the left in the Select All column, select the checkbox to select all Quote Results. Clear the checkbox to select individual Quote Results in the column. When at least one quote is selected the Print and Send Email options are enabled.
    • Expand the grid to view more columns in the scrollable column section. When additional columns are available to view to the right, click the right directional arrow button. When additional columns are available to view to the left, click the left directional arrow button. The arrow buttons appear only if there are columns to view in the applicable direction.
    • Click on a column to make it the primary sort column. Click on the column header to switch between ascending and descending order.

    • For Fixed Indexed Annuity quotes, in the left column title row, click the button to expand all the product rows in the grid to view the indexed funds for the products and click the button to collapse the rows. In a product row, click the button in the left column to expand the row vertically to view the indexed funds for the product. Click the button to collapse the row.

      NOTES:

      • When a row is collapsed and only the product row is visible, the row contains the maximum value for Fixed Rate, Premium Bonus, and Index Rate Cap from all related indexed funds.
      • When a row is expanded, the index fund rows display the actual values associated to the index fund. These are Fixed Rate, Premium Bonus, Index Rate Cap, Cap Type, and Free Withdrawal Yr 1 | Yrs 2+.

      • Columns with sort capability may be sorted when rows are collapsed or expanded. When rows are expanded the values for the selected column are sorted by the indexed funds within each product.

  13. Optionally, return to the Quote wizard data entry to change inputs: on the lower right, click Back to Inputs.
  14. View and download product documents: in the Actions column, click the Documents link and click a document type (Product Guide, etc.) to open the product document in PDF format in a separate browser tab or window. Use the pdf reader controls to save or print the document.

  15. View available riders for a product: in the Available Riders column, pause the mouse over the Riders link to view the available riders for the product.

  16. Save the quote results: on the top navigation bar, click Save, enter a Name for the quote and click Save on the Save Quote dialog.
  17. Print the quote results: in the grid, select the quotes to print. On the toolbar above the Quote grid, click Print. In the Print dialog, select the Quote Documents (Basic Proposal, Comparison Proposal) and Optional Documents (Product Disclosures, Product Guide, Specimen Policy) to print and click the Print button. The quote proposal opens in a separate browser tab or window. The Basic Proposal displays the quotes in list format. The Comparison Proposal compares up to three product quotes at a time in a columnar format. If more than three products are selected for a Comparison Proposal, the next three selected products display after the first three selected products on a new set of pages and so on until all selected products are displayed. Use the pdf reader controls to save or print the document. NOTE: The optional documents generate only one copy per product in the document set provided either via print or email.

  18. Email the quote proposal to the advisor: on the toolbar, click Send Email. Enter the advisor’s name and address in To Name and To Address, select the Quote Documents and Optional Documents and click Send.

    The advisor receives the email and clicks the URL link within the email to open the quote file. The advisor can use the pdf reader controls to save and print the file.

  19. View summary information, audit history and sent emails or share the quote with another user: on the top navigation bar, click Other Actions and click Summary to view summary information, click History to view audit history, click Sent Emails to view emails sent to the client, or click Share to begin the process of sharing the quote with another user. The Share and Transfer options are available after you save the quote.
  20. (Optional - only for products that are set up with a product, package and sales agreement for Application or Illustration in FireLight) Apply or Illustrate an annuity product: in the Actions column of the grid, click Apply to begin the application process or click Illustrate to begin the illustration process for the selected annuity product quote. FireLight attaches the basic and comparison PDF proposal output to the application or illustration, which can be accessed from Other Actions > Documents within the application or illustration activity. If the product is not carrier sponsored, the Actions column can also contain a Print link to start an application, which can be completed and printed.

Life Admin Setup and User Experience

IXN provides a Life Quoting tool that is embedded in FireLight, which includes the ability to print, email and text the proposal to the advisor, and to launch into a FireLight e-Application. IXN Life Quotes can be saved, shared and transferred. A Quote Activity Report is available. The report includes activity for both IXN Life quotes and Annuity quotes. See Detailed Activity Reports in Admin Help for a description and sample report.

Admin Setup

When you license the Quote activity, Hexure adds the Quote Activity to the grid at the top of the Activities page in Product Configuration. Also have an additional DocType of IXNQuote with a Document Tag of IXNQuote added to your organization's List Profile.

Do the following to set up FireLight and Life Quotes.

Organization Configuration/Organization Settings/Security tab (FireLight Embedded in IXN):

With the enhancement to IXN for embedding the FireLight e-Application, IXN certificate/service record enhancements are required within FireLight to easily support the connection between the two systems.

Enhanced certificate support allows ease of certificate security management for IXN access and updates to the IXN certificate expiration that eliminates the need for user interaction. This further expands the client access to FireLight and IXN clients alike for both systems. See Security for more information.

Add a new integration service record for enhanced certificate support to set up a relationship for IXN.

  1. Below the supported services table, click the Add New Integration Service Record link to open the Integrated Services Configuration dialog.

  2. In the Name list, select IXN Integrated Service. Select the applicable Supported Services checkboxes (Embedded, Illustration API, API Access, Token SSO).

  3. In Client Secret, click the New button. On the Confirm dialog, click Update. Next, click Save.

  4. FireLight adds the IXN Integrated Service to the table.

Organization Configuration/Organization Settings/Admin tab (IXN Embedded in FireLight):

Set up the IXN service credentials. IXN supplies the configuration ALQ_ID and value.

  1. Click the Click to Edit link next to Third-Party Service Config.

  2. Click the Add New link at the top of the dialog. In Config Name, type ALQ_ID. In Config Value, type the ALQ_ID value provided by IXN. In Service Type, select IXN and click the Add Config button.

NOTE: Config Group is an optional configuration used by the DocuSign Integration integration.

    Change the Quote Email Expiration number of days setting as applicable for your organization. The default is 3 days, which means the email will expire at the same time on the third day (72 hours) from the time it is sent to the client.

User Experience - Life Products

  1. On the Home page under Start New or on the New Activity menu, click Quote to open the Quote wizard.
  2. On the Quote toolbar, click Life.

  3. In the left column Proposed Insured section, enter the information for the proposed insured.

    • First Name

    • Last Name

    • Date of Birth or Age

    • Gender

    • Tobacco Use

    • State

    • Health Options - The number of health options may need to be reduced if other options are selected. For example, you cannot select all health options with all carriers, but all health options can be selected with a single carrier.

  4. In the center column Product section, select the product types (Term Periods, To Age, Term ROP, Whole Life, and Final Expense) and carriers to filter the quotes by product and carrier. Up to five product types can be selected when the user selects all carriers. When the number of carriers is limited, more than five product types can be selected.

  5. In the right column Coverage and Agent sections, enter or select the coverage and agent information. Up to three Face Amounts can be entered. Optionally, click the Calculate Needs link to calculate the face amount based on the Debts and Expenses, Income, and Assets.

  6. On the lower right of the wizard, click Run Quote.

    The Quote Results contain the client information, Coverage, and Agent information followed by the number of results and a search field. Term Quote results display on separate tabs for each Term duration. Click Show Details at the bottom of a product quote to show product details. Click a tab to show product Features, Forms, Carrier Overview, Premiums, and Important Information on separate tabs. Click Hide Details to hide product details.

  7. (Optional) Click the button to the right of the search field to expand the dialog. Enter a Carrier name if applicable. Select search options to Sort By, Group By, Filter By, and select Product Features to filter the quote results.

  8. On one or more tabs for the desired term period, select the checkbox for one or more product quotes. The Share and Compare buttons become visible.

  9. Click the Compare button to open the Comparing Quotes dialog. Use the scroll bar to scroll left and right through the columns of the comparison. On the lower right of the comparison, click the Share button to select a delivery method and share the Quote.

  10. Click the Share button above the product selection or click the Share button on the Comparing Quotes dialog to open the Select a Delivery dialog.

  11. On the Select a Delivery Method dialog, click a Download, Email or Text button for the delivery method and click the Next button. Download is the default method.

  12. On the Select a PDF Layout dialog, click a PDF layout for the quotes: Detailed Comparison, Condensed Premium Comparison, Premium Comparison, or Feature Comparison and click the Next button.

  13. On the Enter Information dialog, enter Agent Information and ensure Recipient Information is correct. Click the applicableDownload, Email or Text button for the delivery method.

  14. On the Success dialog:

    Download delivery method: Click View PDF to open and view the quote PDF document.

    Email delivery method: Click Send to deliver the email to the client.

    Text delivery method: Click Send to deliver the text to the client.

  15. Click the X on the upper right to close the dialog.

  16. Save the quote: on the navigation bar, click Save. In the Save Quote dialog, enter a Name for the quote and click Save. Save the quote under a new name: on the navigation bar, click the Save down arrow and click Save As. In the Save As dialog, enter a new name for the quote and click OK.

  17. After saving a quote, the Share and Transfer options become available on the Other Actions menu.

  18. On the Quote page, to the right of the Quote, pause the mouse (hover) over the premium amounts of a quote and click the Get Started button.

  19. On the Pick An Action dialog, click the Apply button to begin an application in FireLight. Product availability in FireLight varies, dependent on Carrier participation. If a product is not available in FireLight, the Apply button will not be displayed.

  20. Complete the application in FireLight.

    NOTE: In the application on the Other Actions menu, click Display/Print PDF, select the documents to print including the Quote, and click Print Selected Documents to create a PDF file with the selected application and quote documents.

  21. On the upper right of the Life Quote page, click the New Quote button to start a new quote.

  22. The current design of IXN Quote in FireLight does not allow client detail entered into Life quotes to be Saved to the FireLight dashboard/home page. This enhancement is targeted for a future release.
