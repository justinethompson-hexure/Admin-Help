DTCC Money Settlement Support Feature
Description, Setup, User Experience, and Additional Information

Description

The DTCC Money Settlement feature implements the database and DTCC file changes required to support DTCC money settlement, including adding indicators in the Admin Selling Agreement to denote who is a DTCC settling firm and adding checks to ensure the submitting firm and the settling carrier both are a DTCC NSCC Settling firm. This feature can be used for initial applications and for subsequent premium payments.

DTCC NSCC / FireLight Setup

  1. Carriers and submitting firms must first be approved as Full Settling Members by the DTCC.

  2. Contact your FireLight® Strategic Account Manager to submit a Work Order to ensure the necessary file changes are made to your outbound feeds.

Admin Setup

Organization Configuration/Organization Settings page: enter/verify your DTCC Member ID is correct and select the Full Settling DTCC Member checkbox. This indicates you are a full settling member firm. FireLight validates the DTCC information for the carrier and for the distributor before sending money settlement detail to the DTCC.
