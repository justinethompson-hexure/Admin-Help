Support

Use the Support page to access the following support tools for FireLight®.

API Insights Find instructions for using the API Insights tool to analyze and evaluate requests and responses in the FireLight REST APIs.

Application Data Find instructions for using the Application Data tool to view information on each case that is being sent to the DTCC and their App/Sub and attachment status.

Rule Tracing Find instructions for using the Rule Tracing support tool to provide visibility into the rules engine for debugging and troubleshooting rules.
