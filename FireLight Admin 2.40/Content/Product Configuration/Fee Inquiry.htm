Fee Inquiry

On the navigation bar, click Product Config and click Fee Inquiry to go to the Fees & Expenses page. Carriers can use the Fees & Expenses page to manually upload PPfA Lite and CST files and view or download their uploaded files and/or files that were submitted through DTCC.

See Fees, Expenses and Commissions Repository in Features for information about implementing the repository.

Upload a Fees & Expenses File

Upload a Fees & Expenses File
  1. On the navigation bar, click Product Config and click Fee Inquiry to go to the Fees & Expenses page.
  2. Click the Upload File button.

  3. Enter the Distributor DTCC Member Codes, browse for the PPfA Lite or CST file and click Upload.

    Leave the Distributor DTCC Member Codes field blank to apply the fee to all distributors, or enter one or more DTCC member codes (separated by commas) to upload just for specific distributors.

    NOTE: When you upload an updated version of a file, it overwrites the existing same-named file.

Download a Fees & Expenses File

Download a Fees & Expenses File
  1. On the navigation bar, click Product Config and click Fee Inquiry to go to the Fees & Expenses page.

  2. In the Download column of the grid, select the checkbox for the file to download. Or on the right, select the Check/Uncheck All checkbox to download all of the files.

  3. Click the Download Files button. FireLight® creates a Fee Files.zip download file.

  4. On the notification bar, select a Save option from the list to save the downloaded file.

Delete a Fees & Expenses File

Delete a Fees & Expenses File
  1. On the navigation bar, click Product Config and click Fee Inquiry to go to the Fees & Expenses page.

  2. In the Download column of the grid, select the checkbox for the file to delete. Or on the right, select the Check/Uncheck All checkbox to delete all of the files.

  3. Click the Delete Files button.

  4. On the Confirm Delete dialog, click OK.
