SuranceBay API Key

Below are the steps to require users to enter the SuranceBay API Key in the Third Party Configuration to validate that agencies are ready to use the SuranceBay integration.

  1. Log in to the Admin App.

  2. Select Organizational Settings.

  3. Select the Admin tab.

  4. Click Edit next to Third Party Configuration Settings.

    5. The Third Party Configuration Settings window appears.

  5. Click Add New.

  6. Browse Service Type and select SuranceBay.

  7. For Config Name enter Apikey and for Config Value enter Provided by SuranceBay.

  8. Click the Add Configuration button.

    9. The system displays the new configuration in alphabetical order.
