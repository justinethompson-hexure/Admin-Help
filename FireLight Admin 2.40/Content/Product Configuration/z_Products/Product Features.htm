Product Features

On the Product Configuration/Products page, click the Product Features link to open the Product Features dialog. Use the dialog add product features to be displayed in an informational pop-up message during product selection when creating an activity in . The informational icon next to a product becomes available when a product feature is added. The information is also available for term products quoting (see Quote Detail).
NOTE: The list of features can be edited by using the Branding feature, if enabled.

Add, Edit, or Delete Product Features

  1. On the Products page, click the Product Features link to open the Product Feature dialog.

  2. Select a feature in the list; for example, select Company A.M. Best Rating.

  3. Type the feature description (rating) in the text box below the feature name and click the Save button.

    A feature preview appears on the right. Product information is cached to improve response times. The Display Preview will update the preview of the feature information after a 20-minute delay.

  4. (Optional) Edit a selected feature: Click in the text box below a feature name and type the corrected text.

  5. (Optional) Delete a selected feature: Click the red X on the right of the text box for the feature and click the Save button.

  6. When finished, click the X on the upper right to close the dialog
