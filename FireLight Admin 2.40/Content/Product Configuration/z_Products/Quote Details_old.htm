Quote Details

On the navigation bar, click Product Config and click Products to open the Products page. On the lower left, click the Quote Details link to open the Quote Details dialog.

NOTE: The Quote Details dialog is not available in Admin when the Disable View Quoting group option is selected on the Admin tab in Organization Configuration/Groups for your role code so you can access the Quote Details dialog from the Product Configuration/Products page. When Quote is set up in Admin, be sure to select the Quote feature on the Activity Access tab on the Groups page in Organization Configuration so users can access the Quote feature.

Use the Quote Details dialog to upload rates and product rules and set up product configuration for the Quoting feature in , which provides comparison quotes for Term Life products.

Prepare the rate files and then upload rates and add product and rate information for Term Life products in Admin.

Step 1 - Prepare the Rate File

  1. Gather the rates and put them in the following format for each Term Life product for quoting.

    You can have multiple rates in a row, but will use only the first rate.

  2. Save the file as a .csv or .xlsx file type.

Step 2 - Upload a New or Replacement Rate File

  1. On the navigation bar, click Product Config and then click Quoting to open the Quoting page on the Rate Info tab.

  2. Click the Upload Rate button to open the Upload File dialog.

  3. Enter the following information in the Upload File dialog.

    Rate Name Type a unique Rate Name for the rates.

    Replace Current Rate Select the checkbox if this is a replacement for an existing rate file.

    Mapping Type the mapping numbers in the rate spreadsheet that match the listed risk class designations. Type -1 if you have no mapping, but the spreadsheet risk classes can map to multiple risk classes.

  4. Click the Browse button and select a rate file to upload.

  5. Click the Upload button to upload the rate file.

  6. Click the Save button.

Step 3 - Add Product and Rate Information

  1. On the upper right of the Quoting page, click the Product Info Class tab.

  2. Click a Term Product name in the grid.

  3. Click the Rate tab and enter the following information.

    Rate Set - Select a Rate Set for the selected product.

    Rate Start - Enter the date when the rates start.

    Rate End - Enter the date when the rates end.

    NOTE: To clear rate start and end dates, click the Add New Rate for Product button.

    State Approved grid - Select the checkboxes for the states where the rates are approved. Or select the Check/Uncheck All checkbox to select all states or clear all states.

    Save Product Rate Info - Click the Save Product Rate Info button to save your entries.

  4. Click the Product Rules tab, click the Add New Rule button, and enter the following information.

    Risk Class - Select a Risk Class in the list.

    Min Age - Type the minimum age for the selected Risk Class.

    Max Age - Type the maximum age for the selected Risk Class.

    Min Face Amount - Type the minimum face amount for the selected Risk Class.

    Max Face Amount - Type the maximum face amount for the selected Risk Class.

    List Periods - Press and hold the Ctrl key and click each applicable time period in the list.

    State Approved grid - Select the checkboxes for the states where the product rates are approved. Or select the Check/Uncheck All checkbox to select all states or clear all states.

    Save Filter Info - Click the Save Filter Info button to save your entries.

  5. Click the Product Config tab and do the following.

    1. Click a Config Type in the grid.
    2. Enter updates in Config Value below the grid.
    3. Click the Save Config Info button to save the updates.
