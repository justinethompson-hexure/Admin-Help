View Rule Set Audit History

  1. On the Rule Sets page, select a rule set in the Rule Sets grid at the top of the page.
  2. Click the Rule Set History tab.
  3. View the selected rule set history in the applicable columns of the history grid: Action, Old Value, New Value, Action Date, and Login.
  4. In Utilities, Reports page, Admin Reports, run the Admin History Report for Rule Set History Audit Report to view an audit history report for one or all rule sets during a selected time frame.

NOTE: The system purges all rule set deployment and audit history that is more than one year old.
