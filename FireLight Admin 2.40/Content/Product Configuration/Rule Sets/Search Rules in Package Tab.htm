Search Rules in Package Tab

Use the Search Rules in Package tab to search all rules in a package and list the rule set names that contain a specific rule.

  1. On the navigation bar, click Product Config and click Rule Sets to go to the Rule Sets page.
  2. Click the Search Rules in Package tab.

  3. Enter the name of a data item in Data Item and/or enter a string in AND/OR String.

  4. In Match, select either Exact or Anywhere for the type of match for the search.

  5. Select a package name in the In Package list or select All to search all packages.

  6. In the Rule Sets in Package list, select All or select a specific rule set name.

  7. Click the Search button to initiate the search.

  8. (Optional) Click the Export button to export the searched rule information to a spreadsheet in CSV format.

    Depending on the browser being used, in the notification bar click a button (or click an arrow next to a button and then click an option) to open or save the spreadsheet.
