Copy Rules

  1. On the navigation bar, click Product Config and click Rule Sets to go to the Rule Sets page.
  2. In the upper grid, select a target rule set to receive the copied rules.
  3. On the Rule Set Detail tab, click Copy Rules.

  4. Click the Copy From field and select the source rule set to copy the rules from. Or type a rule set name to filter the list of rule sets and then select a rule set from the filtered list.

  5. In the grid, ensure all of the rules to be copied are selected and then clear the checkboxes for the rules that will not be copied.

    • Select all rules - select the checkbox in the Rule Name column of the list header. Clear the checkbox to clear all rules.
    • Search for a specific rule - type the rule name in the Search field and click Go to display rules that match the search text.
    • Clear the search results - clear the text in the Search field and click Go to clear the search results.
    • Sort the grid columns - click a column title name to sort the grid by the selected column. Click the same column title to sort the grid in the reverse (descending or ascending) order.
  6. Click Copy Selected Rules to copy the rules to the rule set that is active on the Rule Sets page. Click the X on the upper right to close the Copy Rules window.
  7. Click Save Rule Set on the Rule Set Detail tab to save the copied rules to the target rule set.
