Export Rule Set XML

From the Rule Sets page, you can export all rules within a rule set to an XML file.

  1. On the navigation bar, click Product Config and click Rule Sets to go to the Rule Sets page.

  2. Select a rule set in the upper grid.
  3. On the Rule Set Detail tab, click Export Rule Set XML.
  4. Choose whether to Open or Save the exported file.

NOTE: Use the Import/View All Rule Definitions Window to copy rules from an exported rule set to a new rule set.
