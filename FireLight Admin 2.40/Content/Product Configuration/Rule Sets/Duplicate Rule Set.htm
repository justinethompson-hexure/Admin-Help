Duplicate a Rule Set

This feature provides the ability to duplicate an existing rule set and copy all the selected States Approved from the original rule set to the copied rule set in FireLight®.

  1. On the navigation bar, click Product Config and click Rule Sets to go to the Rule Sets page.
  2. In the upper grid, select a rule set to duplicate. For example, select the "Owner Rules" rule set.

  3. On the Rule Set Detail tab, click the Duplicate Rule Set link.

  4. FireLight copies the selected rule set and all selected States Approved from the original rule set to a new rule set and appends "Copy -" to the end of the new rule set name. The Duplicate Rule Set will display in the grid at the top with a suffix of the original rule set name and “- Copy”, for example, "Owner Rules - Copy" rule set name.
  5. If applicable, change the new rule set name and click Save Rule Set.

NOTES:

  • The "Copy" suffix will repeat based on the number of times the rule set is copied.

  • If the rule set is locked by the current user or by another user, the "Duplicate Rule Set" link remains enabled.

  • If the original rule set is locked, the locked property will not be transferred to the copied version of the rule set. The duplicate rule set is copied as unlocked.

  • If the user has read-only permissions on the Rule Set window, FireLight does not allow the user to duplicate the rule set.

  • Rule Set History will contain an entry indicating the duplication of the original rule set name.
