Import/View All Rule Definitions

Use the Rule Definitions Window to display, import, and update the rule definitions for a rule set.

Import or Update Rule Definitions

  1. In the upper grid of the Rule Sets page, select a rule set in the list.

  2. On the Rule Set Detail tab, click Import/View All Rule Definitions link to open the Rule Definitions Window.

  3. Import a new rule set:

    1. Next to Import Rule Set XML, click Choose File.

    2. In the dialog, select a rule set file and click Open. Click the Load XML File button.

    3. If the imported rule set XML is for a new rule set, select the new rule set and click the Load XML File button. No warning message appears.

    4. Imported rule set XML files will overwrite existing rule set definitions. If the imported XML file will overwrite an existing rule set, an Import Rule Set warning message displays: "The Rule Set XML you are about to import will overwrite an existing Rule Set. Do you want to continue?"

      Click OK to import the rule set that will overwrite the existing rule set. Click Cancel to stop the import.

    5. Upon completion of the XML file load, make any updates to the imported rule set and click Update on the lower right of the Rule Definitions Window.

    6. FireLight writes an audit record to the Rule Set History tab, e.g., "Rule set XML was uploaded in rule set [RuleSetName]." For more information about rule set audit history, see View Rule Set Audit History.

  4. Update an existing rule set: on the Rule Definitions Window, make updates to the rule set and click Update. FireLight writes an audit record to the Rule Set History tab.

  5. Click the X on the upper right of the Rule Definitions Window to close the window.

  6. Click Save Rule Set on the Rule Sets page to save the imported rule set or updates to an existing rule set.

Copy Rules from an Exported Rule Set to a New Rule Set

  1. Locate and open the exported rule set XML file using Notepad, Notepad ++ or any other text editor.

  2. On the Product Configuration/Rule Sets page, click the Import/View All Rule Definitions link to open the Rule Definitions Window for the new rule set and delete the text from the file.

  3. Copy the rules from the exported rule set file, paste the rules into the Rule Definitions Window for the new rule set file, and click Update.

  4. Click Save Rule Set on the Rule Sets page to save the rule set.
