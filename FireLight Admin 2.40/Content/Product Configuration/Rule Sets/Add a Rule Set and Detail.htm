Add a Rule Set and Detail

Add a Rule Set and Detail:

  1. On the navigation bar, click Product Config and click Rule Sets to go to the Rule Sets page.

  2. On the upper left, click Add New Rule Set.

  3. On the Rule Set Detail tab, add or update the Rule Set Detail information for the selected rule set.

  4. Click the SAVE RULE SET button to save rule set detail entries, and unlock the rule set for editing by other users.
  5. Go to the Rules Window to add rules from pre-built templates and create your own custom rules, or open the Import/View All Rule Definitions Window for the new rule set and import or add rules from an exported rule set XML file.
