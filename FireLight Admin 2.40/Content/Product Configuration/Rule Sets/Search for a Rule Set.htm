Search for a Rule Set

  1. On the navigation bar, click Product Config and click Rule Sets to go to the Rule Sets page.

  2. On the upper left, begin typing the name of a rule set in the text box and click the search button.

    The rule set grid displays the filtered list.

  3. In the grid, click a name in the Rule Set Name column to select the rule set for editing.

  4. When the user clears the search bar, the search results refresh and reset back to what they were before searching.
