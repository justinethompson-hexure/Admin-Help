Navigation Buttons for SSO Users [BETA]

NOTE: This feature is currently in beta and will be implemented only for select clients.

A new button on both the Firelight App Home Page and the Firelight Admin Home Page allows users who log in to Firelight through SSO to navigate between the Firelight App and the Admin app without having to re-authenticate through their SSO provider.

For users for whom this enhancement has been enabled, the Firelight App home page has in the Manage section a button that says To FireLight Admin, and the FireLight Admin home page has in the Support section a button that says To FireLight App.

Clicking this button from either Home Page takes the user to the desired page without having to reauthenticate through their SSO provider.
