Branding

WARNING! Because this is an advanced feature, use of this option should have approval from your FireLight® Strategic Account Manager.

On the navigation bar, click Utilities and click Branding to go to the Branding page.

NOTE: This page is available when: (1) the Allow Branding of UI option is enabled on the Organization tab in Organization Configuration/Organization Settings page and (2) the Enable Change UI Branding checkbox is selected for the user role on the Admin tab of the Groups page.

Use the Branding page to view and modify all user interface items (UI Labels, string constants (Enums), Audit Messages, drop-down options (List Profile), Email Templates, and Validation Error Messages) and add or update sub/firm branding. When the Branding page is enabled, a Branding button is available on the Home page.

View, Add, or Update Sub Firm Branding

View, Add, or Update Sub Firm Branding

NOTE: Please consult with our Configuration Specialists or Solutions Consultants about the usage of this tool.

  1. On the Home page or navigation bar in Utilities, click Branding.

  2. Click the Add or Update Sub/Firm Brand tab to view the list of Sub/Firm Brands, if any.

  3. On the right, type the Branding Company Name in the box.

  4. Type the associated Branding Code.

  5. Click the Browse button and select the associated Logo Image.

  6. Click the Upload button to upload the sub/firm branding logo.

  7. Make any adjustments to the branding company name, branding code, logo or text.

  8. Click the Save Branding button to save the sub/firm branding.

  9. Click OK in the confirmation dialog.

    The new sub/firm brand is added to the List of Sub/Firm Brands on the left.

  10. On the left, click the View on link for the new sub/firm brand to view the new branding.
