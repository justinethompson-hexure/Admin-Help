Utilities

Use the Utilities pages to access branding, deployments, reports, transfer user activities, and common tag conversion utilities in FireLight® Admin.

Branding Find instructions for modifying English text to localized text for User Interface elements.

Download Tag List Find instructions for downloading a .csv formatted spreadsheet for the existing dataitem tags (names) within forms, wizards, and rules in a package. Then add new dataitem tags in the "to" columns of the spreadsheet to create a mapping spreadsheet from old tags to new tags. This utility is the first step in converting old tags to common tags or other new tags.

Common Tag Utility Find instructions for adding common tag names to existing forms and wizards using the Common Tag Utility.

Package Conversion Find instructions for using the mapping spreadsheet to create new forms, wizards and rule sets with new dataitem tags in a package. This utility is useful for converting old tags to common tags or other new tags.

Deployments Find instructions for deploying data from UAT to Staging and Production environments and viewing deployment history along with deployment terminology, testing process guidelines, and deployment workflow.

Reports Find descriptions and samples of Admin, Detailed Activity, and Summary Activity reports, along with typical Activity report setup and how to generate and view Admin reports.

Activity Reports Scheduler Find instructions for scheduling any available Detailed or Summary activity reports from the Admin App and configuring the sending of required reports at a specific frequency, format, and send mode.

Transfer User Activities Find instructions for permanently transferring all activities from one user ID to another user ID.
