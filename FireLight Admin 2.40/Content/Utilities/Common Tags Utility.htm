Common Tags Utility

IMPORTANT! Be sure to enable the Common Tags Utility. Go to the Organization Configuration/Groups/Admin tab and select the Enable Common Tags Utility checkbox.

The Common Tags Utility provides a way for carriers and distributors to map their existing Field Names in field properties to Common Tags to make client implementations easier and faster by eliminating mapping rules between carriers and distributors.

See the Common Tags Utility User Guide Common Tags Utility User Guide in Admin Help for detailed instructions.
