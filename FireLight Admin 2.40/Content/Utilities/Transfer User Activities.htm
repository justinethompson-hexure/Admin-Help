Transfer User Activities

NOTE: To enable the Transfer User Activities feature, go to the Organization Configuration/Groups/Admin tab and select the Enable View Utilities checkbox.

On the Home page or navigation bar, click Utilities/Transfer User Activities to go to the Transfer Activity Utility page.

Use this utility to permanently transfer all activities from one user ID to another user ID.

Following are some scenarios for using the Transfer All Activities utility:

  • A user changes name and is issued a new user ID. Transfer all activities from the old user ID to the new user ID.
  • An advisor leaves the company. Transfer all the advisor’s activities to their replacement.

  • A sales desk person changes jobs. Transfer all the sales desk person's activities to another sales desk person.

Transfer All Activities

  1. In the Transfer from User ID field, enter the user ID from which to transfer all activities.
  2. In the Transfer to User ID field, enter the user ID to receive all activities from the other user.

  3. Click the TRANSFER ACTIVITIES button.

  4. If both user IDs are valid, FireLight® displays a confirmation dialog. Click OK.

    FireLight permanently transfers all of the activities to the recipient user ID and creates an Audit History entry in each transferred activity with a display status of Admin Transfer.
