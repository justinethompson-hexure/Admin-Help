Localization

NOTE: This page is available only for Hexure FireLight® team access.

On the navigation bar, click Utilities and click Localization to go to the Localization page.

 

NOTE: The client contacts their Strategic Account Manager to have the FireLight team provide a spreadsheet of the FireLight UI labels, lists and messages in the localized language using Google Translate. Clients can update the spreadsheet with their preferred translations and return the spreadsheet to the FireLight team for implementation. The client's Provider may also require changes for localized date and currency formats, etc. The Jurisdiction list needs to be modified for the localized language.

In , users can select a language/locale in the Locale list on the Preferences page.

Update a Translation

  1. On the Home page in Utilities, click the Localization button. Or on the navigation bar, click Utilities and click Localization.

  2. Above the grid in the list on the left, select the type of UI element to translate. If applicable, select a Page/View in the second list.
  3. In the grid header, click the DISCLAIMER button to view the disclaimer information for Google Translate.

  4. In the Translation column, type the updated translation text.

    The updated translation is highlighted.

    In the following example, the user selected to update a Validation Error Message.

  5. Click the Save Changes button to save the updated translation text.
