FireLight v2.35 Release History

May 2024 Release—5/30/2024 Production

New Functionality

Remove Ability to Manage Attachments When Application Is in Post-Submit Review: DIST-2663

With this enhancement, an agent will not be able to add or remove documents to an activity once it is sent to post-submit review. Because the activity is already signed when sent for post-submit review, agent should not be allowed to add/remove documents to/from it.

Organizations can enable this feature from the group settings in the Admin tool based on their workflow.

Impact

  • Admin: Add new group setting to enable the feature to enforce product availability alert - FL Admin > Organization Config > Groups > App (tab) > Submit (section) > “Disallow Document Upload After Submission” (check box)

  • App: Upload/Remove of documents will not be available to the agent if the application is sent to post-submit review.

Modify Sales Agreement Report to Display Enabled as of Today: DIST-3030

With this enhancement admin users can generate list of Sales Agreements enabled for any selling organization as of current date. The filter for existing Sales Agreement Admin report will be modified.

Impact

Admin Report “Sales Agreement Report” will be modified for both PDF and CSV formats - FL Admin > Utilities > Reports.

Modify Admin Report - Package Forms Report: DIST-3026

With this enhancement, the existing admin report “Package Forms Report” will also display the below additional information.

  1. Duty type selected for the form/wizard in the package

  2. Number of copies selected for the form in the package

  3. Effective start and end dates for the form

Along with above information, the order of the form listed in the report will be kept the same as the order set into the package.

Impact

Admin Report “Package Forms Report” will be modified for both PDF and CSV formats - FL Admin > Utilities > Reports.

Ability to Copy Fields from Wizard: DIST-2597

With this enhancement, admin user will be able to copy fields from a source wizard to destination wizard. Currently, fields from an existing form can be copied to a wizard. This enhancement will extend that same functionality to copy fields from a wizard.

Impact

FL Admin > Designer > Wizard Designer > Copy Wizard Fields (new link under Page Options)

Modify Package Rules Reports to Include Effective Dates: DIST-3194

With this enhancement, the existing admin report “Package Rules Report” will also display effective start and end dates of the rules sets listed under the package(s).

Impact

Admin Report “Package Rules Report” will be modified for both PDF and CSV formats - FL Admin > Utilities > Reports.

Audit and Display History for Organization Settings in Admin Tool: DIST-3328

With this enhancement, FireLight will now audit Organization Settings data in the admin tool. Edits made to any of the fields on existing tabs will be recorded and displayed on History tab.

A new admin report to list organization setting history will be available under Admin History Reports.

Organization and group settings data will be included as a part of the purge process.

Impact

  • A new tab will be added to display the history of changes made.

    • FL Admin > Organization Settings > History (new tab).

  • A new admin report to org settings history will be added to:

  • FL Admin > Utilities > Reports > Admin History Report > Organization Settings History (new report)

• Organization and group settings history data will be purged based on admin data purge policy.

Allow to Bold Label for Text Box Control on Wizard: DIST-3109

With this enhancement, users can bold the labels for text box controls on a wizard.

Impact

Add a new field property “Make Label Bold” to the text box control on the wizard designer.

  • FL Admin > Designer > Wizard Designer

Validate Search Items in Emails If Blank or Null: DIST-3238

Currently, if search items used in an email template have a blank or null value, they are not replaced in the emails sent to the users. This exposes the search items in the emails sent to users. With this enhancement, the search items will be tackled to not be included in the emails if they have blank or null values while email is sent.

Impact

Below area will be impacted by this feature:

  • FL Admin > Organization Config > Email

Maintain Producer Profile Section When Changing any Product Filters: CR-2133 & CR-2485

This enhancement keeps the Product Profile section static when user changes Jurisdiction or Product Type or Carrier or Locale or Override Date in Product Selection Page when SuranceBay Integration is enabled.

Impact

User will have a better user experience when changing any product filters, because the Producer Profile from SuranceBay will remain static regardless changes in the Jurisdiction, Product Type, Carrier, Locale, or Override Date.

Add Indexed Universal Life for a Product Type: PC-2468

This enhancement adds the Indexed Universal Life product type in the product details section of the administration tool so that carriers setting up new products have the ability to designate a product as Indexed Universal Life.

Impact

Products can be identified as Indexed Universal Life allowing IUL products to be distinguished from other Life Product types.
