Groups

On the navigation bar, click Organization Config/Groups to go to the Groups page.

Use the Groups page to create and manage user group specific role codes. The role codes are used to enable and disable features within the FireLight® system and to define what activities, products and cases are available to the logged-in users within a group.

Step through the following to find instructions for creating and editing a group, comparing case access by firm ID, group role code and user share, setting up needs determination product filtering, and detailed information about each group access option that can be enabled, disabled or allowed on each tab of the Groups page. Also view group history for each action taken on the Groups page.
