Tele-Sign Scripts

NOTE: To enable the Tele-Sign feature, go to Organization Config/Groups, App tab and select the Enable Tele-Sign checkbox. See Tele-Sign in Features for setup and user information.

On the navigation bar, click Organization Config/Tele-Sign Scripts to go to the Tele-Sign Scripts page.

Use the Tele-Sign Scripts page to add or edit Tele-Sign scripts for the applicable products with the Tele-Sign feature.
