Review Queue

  1. At the top of the Organization Configuration > Groups page, select a group name/user role code in the grid.

  2. At the top of the Review Queue tab, select the checkboxes for the global review queue features that can be accessed by the selected group user role.

    Disable Action Email Notifications

    Enable Manual Queue Move

    Enable Assign Reviewer Button

    Application PDF Link When Application is Locked by Other User

  3. In the lower portion of the Review Queue tab, select checkboxes to enable review queues. Select the Check/Uncheck All checkbox to grant or remove the group's access to all review queues.

    NOTES:

    • Be sure to select the PreSubmit checkbox for the Pre-Submit Review Queue reviewers.
    • If no queue options are selected, the Review Queue button on the FireLight Home page and on the navigation bar will not be visible.

  4. Within each queue, select the features to enable for the queue.

    Read Only

    Can Unlock All

    Unlock Only

    Recall Rejection

  5. At the bottom of the page, click Save. (Reviewer instructions)
