Admin

On the Organization Configuration/Groups page Admin tab, select the checkboxes for the following options to disable or enable for the Admin user role code in ® Admin. The options are listed in alphabetical order.

NOTE: If the admin user is a Firm Org admin, they will not be able to disable (uncheck) the following options, either at the time of creation of the group or in subsequent visits: Disable View Organization, Disable View Product, and Disable View Sales Agreement. When the Firm Org in the Firm Org list is changed from a Firm Org to blank, the default options listed above will default to disabled (unchecked). Those options can then be enabled (checked) if applicable for the user role.
