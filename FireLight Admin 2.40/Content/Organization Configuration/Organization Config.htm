Organization Config

Use the Organization Configuration pages to add and update custom lists, edit email templates, add and edit groups, edit mapping data items, add and edit organization settings, and add and edit post submission review queues.

Custom Lists Find instructions for adding and updating custom lists for your organization that are accessed from button controls.

Email Find instructions for editing the default email templates for your organization. Individual email templates are grouped by Email Category in this section.

Groups Find instructions for adding user access role codes and enabling and disabling features based on user access role codes.

Mapping Find instructions for editing data IDs used in mapping form and wizard party data to ® data.

Organization Settings Find instructions for adding and editing your organization's settings.

Review Queue Find instructions for adding, editing, and deleting an automated or standard post-submission review queue prior to the orders being transmitted to the carrier's back office.

Tele-Sign Scripts Find instructions for adding and editing Tele-Sign scripts.
