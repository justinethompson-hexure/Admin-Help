Download Completed E-Delivery

Download Completed E-Delivery
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

This email is sent for the child cases when they are complete for Fill and Sign and Sign Only workflows for E-delivery. This email template is not sent for Delivery Only workflow because they already received the Download email. The Download Completed e-Delivery is sent two times if there is an agent signer on the child package.

  • Leave behind – send Download Completed e-Delivery immediately so the insured or owner gets their completed forms immediately upon signing completion. This will not have the agent signatures because they are not done yet. The agent signs one time for all packages.

  • Completed email – send to the child recipients after agent signs, which is when the entire package is complete. Send the Download Completed e-Delivery email again only if there are agent signatures. That means we send this email when the owner and insured and any agent signing are done and the entire package is marked eDelivery Complete. Suppress this email if there are no agent signatures on the child package.
