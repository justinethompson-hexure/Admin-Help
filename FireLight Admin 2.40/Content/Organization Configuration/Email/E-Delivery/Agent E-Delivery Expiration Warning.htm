Agent E-Delivery Expiration Warning
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Notifies the Agent when an E-Delivery is nearing expiration1
Template Name: Agent E-Delivery Expiration Warning

Email Subject Line: Agent E-Delivery about to Expire

Message:

Dear <AGENT_NAME>,

Your pending E-Delivery to <CLIENT_NAME> in <ACTIVITY_NAME> <APPLICATION_NAME> is about to expire in <EXPIRY_DAYS> days. Please take all necessary actions that this requires.

This template is available for sending when the E-Delivery checkbox is selected in Organization Configuration/Groups page, Activity Access tab and the E-Delivery Expiration Warning Threshold and Requests Timeout options are set in Organization Configuration/Organization Settings on the Activity tab.
