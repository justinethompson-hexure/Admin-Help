Revoke User Share - All Activities
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Revokes agent share of All Activities with another user

Template Name: Revoke User Share - All Activities

Email Subject Line: <AGENT_NAME> has revoked the electronic application (<SHARETYPE>) share request

Message:

{0} has revoked your ({1}) application access request originally sent on {2}.

The Revoke User Share - All Activities template is available for sending when the Enable All Activities User Share checkbox is selected on the in Organization Configuration/Groups page on the App tab.

To Name / Email

User preference Name/Email for owning agent

From Name / Email

User preference for agent rejecting share request

CC Address

No cc Sender
