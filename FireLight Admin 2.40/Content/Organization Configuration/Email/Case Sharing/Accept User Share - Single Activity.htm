Accept User Share - Single Activity

Accept User Share - Single Activity
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Provides confirmation of user acceptance to share a Single Activity from another user

Template Name: Accept User Share - Single Activity

Email Subject Line: <USERSHARE_NAME> Accept Single Activity Share Request

Message:

<USERSHARE_NAME> accepted your <ACTIVITY_NAME> share request named <CASE_NAME>.

The User Share templates for sharing a single activity are available for sending when the Allow Activity Share checkbox is selected in Organization Configuration/Groups page on the App tab.

To Name / Email

User preference Name/Email for owning agent

From Name / Email

User preference for agent accepting share request

CC Address

No cc Sender

Available Tags3

<AGENT_NAME>
<APPLICATION_ID>
<CASE_NAME> (name of the activity)
<DATE_TIME> (current datetime in UTC)
<SHARETYPE> (Read Only, Full Control)
<STATUS>
<TRANSACTION_TYPE>
<USERSHARE_NAME> (shared "to" user name)
ApplicationSearchItems4 (e.g., add <CLIENT_NAME>)
