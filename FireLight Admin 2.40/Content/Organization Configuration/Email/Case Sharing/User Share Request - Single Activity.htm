User Share Request - Single Activity
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Provides link to allow share of a Single Activity with another user
Template Name: User Share Request - Single Activity

Email Subject Line: <USERSHARE_NAME> Accept Single Activity Share Request

Message:

<USERSHARE_NAME> accepted your <ACTIVITY_NAME> share request named <CASE_NAME>.

The User Share Request - Single Activity template is available for sending when the Allow Activity Share checkbox is selected in Organization Configuration/Groups page, App tab.

To Name / Email

User preference for agent receiving share request

From Name / Email

Default: Session User’s Preference Name/ Email2
Use Agent: Owning Agent’s Preference Name/Email

CC Address

cc email sent to From Email address
