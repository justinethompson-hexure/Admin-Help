Revoke User Share - Single Activity

Revoke User Share - Single Activity
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Revokes agent share of a Single Activity with another user
Template Name: Revoke User Share - Single Activity

Email Subject Line: <AGENT_NAME> has revoked the <ACTIVITY_NAME> (<SHARETYPE>) share request

Message:

<AGENT_NAME> has revoked your (<SHARETYPE>) access on the <ACTIVITY_NAME> named <CASE_NAME> for the user share request originally sent on <SHAREDATE>.

The Revoke User Share - Single Activity template is available for sending when the Allow Activity Share checkbox is selected in Organization Configuration/Groups page, App tab.

To Name / Email

User preference Name/Email for owning agent

From Name / Email

User preference for agent rejecting share request

CC Address

No cc Sender

Available Tags3

<APPLICATION_ID>
<CASE_NAME>
<DATE_TIME> (current datetime in UTC)
<SHAREDATE>
<SHARETYPE> (Read Only, Full Control)
<STATUS>

<USERSHARE_NAME> shared "to" user name
