User Share Request - All Activities
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Provides link to allow share of All Activities with another user
Template Name: User Share Request - All Activities

Email Subject Line: Share Request: <USERSHARE_NAME> - Applications

Message:

I would like to grant you ({1}) access to view my electronic applications.

Click on this link to accept this share request.

{2}

Sincerely,

{3}

The User Share Request - All Activities template is available for sending when the Enable All Activities User Share checkbox is selected in Organization Configuration/Groups page, App tab.

To Name / Email

User preference for user receiving share request

From Name / Email

Default: Session User’s Preference Name/ Email2
Use Agent: Owning Agent’s Preference Name/Email

CC Address

cc email sent to From Email address
