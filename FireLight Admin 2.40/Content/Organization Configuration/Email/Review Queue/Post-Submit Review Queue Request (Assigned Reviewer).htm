Post-Submit Review Queue Request (Assigned Reviewer)
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Notifies a particular assigned reviewer that a case is ready for their review. This notification goes to the reviewer upon submission of the activity. The system detects if an email address has been added to the Preferences section for that reviewer, and if so, it sends the email to the assigned reviewer.

Template Name: Post-Submit Review Queue Request (Assigned Reviewer)

Email Subject Line: A New Case has been Assigned

Message:

Dear Reviewer,

The following case <CASE_NAME>, has been auto-assigned to <QUEUE_NAME> for <REVIEWER_NAME> and is now available for review on queue. Please login to FireLight at your earliest convenience to review.
