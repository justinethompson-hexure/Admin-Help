Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Notifies the agent that the application has been processed and approved by the reviewer and has been released to the carrier.1

Template Name: Post-Submit Review Queue Approve Message

Email Subject Line: Review Approval

Message:

This application has been processed and approved by the reviewer and has been released to the carrier. Please contact the carrier if you have any further questions about the status of this application.

<MESSAGE>

Review action templates are available for sending when the Disable Action Email Notifications checkbox is cleared on the Organization Configuration/Groups page, Review Queue tab.
