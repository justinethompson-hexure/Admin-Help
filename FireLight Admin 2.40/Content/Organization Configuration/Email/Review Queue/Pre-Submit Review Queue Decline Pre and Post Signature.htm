Pre-Submit Review Queue Decline (Pre and Post Signature)
Action, To Name/Email, From Name/Email, CC Address, Available Tags and Notes

Action

Notifies the agent that the reviewer declined (rejected) the pre-submission review request for the activity.

Template Name: Pre-Submit Review Queue Decline (Pre and Post Signature)

Email Subject Line: <APPLICATION_NAME> Status Update - Reviewer Signature

Message:

Reviewer <REVIEWER_NAME> has declined the application <APPLICATION_NAME> at <DATE_TIME>.

This template is available for sending when the Allow Pre-Signature Review or Allow Post-Signature Review checkbox is selected in Product Configuration/on the Activities page for each activity type that includes pre-submission review. Be sure to set the reviewer access options in Organization Configuration/Groups page, Review Queue tab.

See Note7 below.
