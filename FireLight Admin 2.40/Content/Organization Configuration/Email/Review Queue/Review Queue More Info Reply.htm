Review Queue More Info Reply

Review Queue More Info Reply
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

An email notification is sent to the reviewer when the agent replies to a More Info request within an Activity sent from the pre-submission or post-submission review queue reviewer.

  • Email is sent whenever a user clicks Reply in the More Info dialog that appears when a user opens an activity with a pending More Info Review Queue request.
  • Email is sent each time Reply is clicked. Note that multiple replies can be sent in the same activity request.

Template Name: Review Queue More Info Reply

  • This template is available for sending for the pre-submission review queue when you select the Allow Pre-Signature Review or the Allow Post-Signature Review checkbox in Product Configuration/Activities page on the Activity Detail tab.
  • This template is available for sending for the post-submission review queue when a Review Queue is set up in Organization Configuration on the Review Queue page. Be sure to set the applicable reviewer access options on the Review Queue and Review Queue Access tabs in Organization Configuration on the Groups page.

Email Subject Line: Reviewer – More Info Received: <APPLICATION_NAME>

Message:

Dear <REVIEWER_NAME>,

<AGENT_NAME> has replied to your request for More Info for the following case, <APPLICATION_NAME>. Please log into Firelight at your earliest convenience to review.

<MESSAGE>

To Name / Email

  • Post-submission review queue: Uses the current preference email address of the reviewer that locked the order in the review queue. If the reviewer changes their User Email Address in Preferences after the order is locked for review, FireLight uses the most current address—not what was set at the time of the lock. If an email address is entered in Organization Configuration/Review Queue page in the Email Group field, the email is sent to both the reviewer address and to the Email Group address.
  • Pre-submission review queue: Uses the email address of the reviewer that locked the order in the review queue.

From Name / Email

Owning Agent’s Full Name and User Email Address in Preferences2

CC Address

If Enable Cc Sender In All Requests is set in Organization Configuration/Groups page on the App tab, cc email is sent to the ‘From Email’ address

Available Tags1

<AGENT_NAME>
<APPLICATION_NAME>
<APPLICATION_ID>
<DATE_TIME> (current datetime in UTC)
<MESSAGE>2
<REVIEWER_NAME> (name of first reviewer)
<STATUS>
ApplicationSearchItems4 (e.g., add <CLIENT_NAME>)

Notes:

2 The <MESSAGE> tag populates the email with the agent's additional information for the reviewer. To receive the email, agents need to have the Send Message Center Emails checkbox selected on the My Preferences page in FireLight.
