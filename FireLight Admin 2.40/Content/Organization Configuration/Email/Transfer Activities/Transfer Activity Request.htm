Transfer Activity Request
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Notifies the agent or advisor when ownership of an application in FireLight has been transferred to them.

Template Name: Transfer Activity Request

Email Subject Line: <APPLICATION_NAME> - Activity Transferred

Message:

Dear <AGENT_NAME>,

Ownership of this <ACTIVITY_NAME> named <APPLICATION_NAME> has been transferred to you. To view this <ACTIVITY_NAME>, open FireLight. The activity will appear in your Recent Activity list on the Home page.

Thank you,

<USER_NAME>

This email is typically sent when a back office/case manager user keys in a paper application in FireLight, and if the application is NIGO, FireLight sends this email to notify the agent or advisor to log in to FireLight and make the necessary corrections. See the Create User, Transfer, Email Custom Action (Concierge Paper Workflow) topic in Features for more information.
