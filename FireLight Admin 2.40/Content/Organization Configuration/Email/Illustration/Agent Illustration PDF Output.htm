Agent Illustration PDF Output
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Sends the agent a URL link to access the illustration PDF output. The agent needs to take action before the expiration date.
Template Name: Agent Illustration PDF Output

Email Subject Line: Illustration PDF Output for <CLIENT_NAME>

Message:

Dear Agent,

Per your request, below is a link to open the illustration PDF output for <CLIENT_NAME> on the <PRODUCT_NAME> of <COMPANY_NAME>.

This email link expires in <EXPIRY_DAYS> days. Please take action prior to the expiration.

Thank you,

To review the illustration output, click on <URL_LINK>. If a new window does not automatically appear, you may have to copy the link and paste it into the address bar of a new browser window.

The Agent Illustration PDF Output email is available for sending when the Illustration checkbox is selected and the Enable Illustration Send Email checkbox is selected in Organization Configuration/Groups page on the Activity Access tab. Also select an expiration number of days in the Illustration Email Expiration list.
