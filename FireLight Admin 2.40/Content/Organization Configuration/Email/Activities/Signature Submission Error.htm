Action

This email template is used with auto generated emails when the status is changed during the signing ceremony for an invalid signature submission.

Email Subject Line: <APPLICATION_NAME> Status Update: Signing Error

Message:

Invalid jurisdictions were selected during the signing ceremony for one or more of the signing roles for <APPLICATION_NAME> as of <DATE_TIME>. The activity is incomplete and no longer valid.

This template is available for sending when the State Signing Error checkbox is selected in Signature Box controls.
