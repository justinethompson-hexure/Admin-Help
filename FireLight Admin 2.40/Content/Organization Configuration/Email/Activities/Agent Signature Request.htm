Agent Signature Request

Agent Signature Request
Action, To Name/Email, From Name/Email, CC Address, Available Tags and Notes

Action

Sends a request to the primary or additional agent to e-sign the application

Template Name: Agent Signature Request1

Email Subject Line: <APPLICATION_NAME> - Please complete your signature

Message:

Dear <RECIPIENT_NAME>,

<APPLICATION_NAME> is ready for your signature.

Use the link included at the bottom of the email to start the signing process. You is asked to acknowledge your acceptance of the disclosure terms and consents. The instruction for completing your "Electronic Signature" is provided as well.

Please use passcode to login.

If you have any questions please feel free to contact me.

If you need help signing, please watch this short guide:

https://vimeopro.com/user47344059/firelight-training/video/263943487

Sincerely,

<AGENT_NAME>

To sign your application, click on <URL_LINK> and enter the passcode. If a new window does not automatically appear, you may have to copy the link and paste it into the address bar of a new browser window.

This template is available for sending when the Allow E-Signature checkbox is selected in Organization Configuration/Organization Settings page on the Signature tab.

 

To Name / Email

Agent Name/Email typed in email fields or pre-filled from mapped dataitems

From Name / Email

Default: Session User’s Preference Name/ Email2
Use Agent: Owning Agent’s Preference Name/Email

Use DMARC1: Organization's Support Email

CC Address

If Enable Cc Sender In All Requests is set in Organization Configuration/Groups page on the App tab, cc email sent to "From Email" address
