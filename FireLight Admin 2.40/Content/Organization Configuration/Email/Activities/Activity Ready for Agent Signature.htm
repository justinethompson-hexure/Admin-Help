Activity Ready for Agent Signature
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Notifies the agent that the activity is ready for the agent's signature. Add the email link to the portal for clients to log in to sign the application, and use the <HYPR_LINK> tag to make the link clickable in the email.

Template Name: Activity Ready for Agent Signature

Email Subject Line: <APPLICATION_NAME> Status Update: Ready for your signature

Message:

All required parties have signed and approved application <APPLICATION_NAME> as of <DATE_TIME>. The application is ready for your signature.

This email is available for sending when the following options are set in Organization Configuration/Groups page on the App tab: Agent Signing Order is set to Last and the Notify Agent When Ready to Sign checkbox is also selected.
