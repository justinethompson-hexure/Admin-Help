Email Delivery Failure Notification (SendGrid)

Email Delivery Failure Notification
Action, To Name/Email, From Name/Email, CC Address, Available Tags and Notes

Action

When the SendGrid email system is activated, it sends email delivery failure notifications to the agent via the Message Center. On screen notification appears if the agent is online.

Template Name: Email Delivery Failure Notification

Email Subject Line: Failure delivering email

Message:

The following had an issue with email delivery. Please review and update as necessary.

<MESSAGE>.

This email is available for sending when the user selects the Send Message Center Emails checkbox on the Preferences page in FireLight. For user instructions, see "Preferences" Preferences [User Instructions] in the FireLight Platform User Guide.

To Name / Email

Agent or Reviewer Name/Email used in original request

From Name / Email

Organization Name/Email

CC Address

No cc Sender

Available Tags

<MESSAGE>

  • When an activity is associated with the email, <MESSAGE> is defined as: <CASE_NAME> <CLIENT_NAME> <CLIENT_EMAIL> <EMAIL_FAILURE_REASON>.

  • When there is no activity associated with the email, <MESSAGE> is defined as: There was an error sending an email to <CLIENT_EMAIL> with the subject <EMAIL_SUBJECT>. The reason for the failure was <EMAIL_FAILURE_REASON>.
