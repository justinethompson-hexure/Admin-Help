Post Submission Pending Case Status
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Notifies the agent or reviewer user when a post submission status message posts to the FireLight Message Center. This email is sent once the polling response is received. The email includes additional details from the provider post-submission call, if any, and displays the agent name, case name, status, and carrier.

Template Name: Post Submission Pending Case Status

Email Subject Line: <CASE_NAME>- <STATUS>

Message:

Dear <AGENT_NAME>,

<CASE_NAME> has received the status of <STATUS> from <COMPANY_NAME>.

<MESSAGE>

This email is available for receiving when the user selects the Send Message Center Emails checkbox on the Preferences page in FireLight. The user can also access the message in the Message Center by clicking the post submission status "envelope" icon in the Recent Activity grid on the Home page or by clicking the icon within the activity on the All Activities page.
