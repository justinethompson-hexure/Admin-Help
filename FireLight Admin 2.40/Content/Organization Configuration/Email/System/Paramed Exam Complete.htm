Paramed Exam Complete
Action, To Name/Email, From Name/Email, CC Address, Available Tags, and Notes

Action

Notifies the agent when a paramed exam has been completed. The message posts to the FireLight Message Center.

Template Name: Paramed Exam Complete

Email Subject Line: <CASE_NAME> - Paramed Exam Complete

Message:

<CASE_NAME> has received a status update from <PARAMED_VENDOR_NAME> on behalf of <COMPANY_NAME>. The status received from <PARAMED_VENDOR_NAME> indicates that this paramedical exam order has been completed.

<MESSAGE>

This email is available for sending when the user selects the Send Message Center Emails checkbox on the Preferences page in FireLight.

To Name / Email

Agent Name/Email used in original request

From Name / Email

Paramed Vendor/Email

CC Address

No cc Sender

Available Tags

<CASE_NAME>
<COMPANY_NAME>
<MESSAGE>
<PARAMED_VENDOR_NAME>
