Custom Lists

On the navigation bar, click Organization Config and click Custom Lists to view the Custom Lists page.

Use this page to manage your custom lists in ®.

Add and update custom lists for your organization that are accessed from a button control.

In , custom lists can contain data for up to nine fields and need to be in Tab delimited text format (*.txt).

NOTE: See the Button control in the Form Designer or the Custom Action field in the Wizard Designer for instructions on adding a button with a custom list and associated fields to a form or wizard.

Add a Custom List to FireLight

  1. Use Microsoft Excel to define your custom list. The spreadsheet should include a single sheet.
  2. The first nine columns within the first sheet of the Excel spreadsheet are used for your custom list. Any additional columns are ignored.

  3. In Excel, save the custom list as a Text (*.txt) file.

    NOTE: The custom list upload file must be tab-delimited, and each column (text between tab characters) is limited to 4000 characters.

    1. Open the spreadsheet in Excel.
    2. On the File tab, click Save As and select a folder in which to save the file.
    3. On the Save As dialog, type a File name.
    4. In Save as type, select Text (Tab delimited) (*.txt).

    NOTE: FireLight supports commas in custom lists and will not replace them with spaces. FireLight also supports Unicode characters (e.g., registered trademark, etc.) in custom lists. If Unicode characters are included in the file, save the file as a Unicode Text (*.txt) file so the Unicode characters will appear in the custom list and wizard Data Grid controls.

  4. On the navigation bar in FireLight Admin, click Organization Config/Custom Lists to open the Custom Lists page.

  5. At the top of page, click the Add New List link.

  6. Type a List Name and an optional List Description if applicable.

  7. Click the Choose File button, browse to the folder, click the name of the .txt file for the custom list, and click the Open button.

  8. Click the Upload button.

  9. Click the OK button to close the confirmation dialog.

  10. The new custom list name is added to the Custom Lists. On the left, click the list name to view the contents on the right. In this example, the Peak Life - Occupations custom list contains the Letter and Occupation columns. Click About This List link to view a description of the list.

  11. Deploy the custom list to the applicable environments. See Deploy Data.

Update a Custom List

  1. On the navigation bar, click Organization Config/Custom Lists to open the Custom Lists page.

  2. In Custom Lists, click a list name and click the Update This List link.

  3. Click the Choose File button, browse to the folder, click the name of the .txt file update, and click the Open button.

  4. Click the Upload button to upload the updated custom list file.

  5. On the left, click the list name to view the contents of the updated list.

  6. Deploy the custom list update to the applicable environments. See Deploy Data.

Delete a Custom List

  1. On the navigation bar, click Organization Config/Custom Lists to open the Custom Lists page.

  2. In Custom Lists, click a Custom List Name and click the red trash can icon to the left of the name.

  3. Click the OK button on the Confirmation Delete dialog. Click the Cancel button to not delete the list.
