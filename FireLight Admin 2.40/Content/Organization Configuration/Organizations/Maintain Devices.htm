Maintain Devices

  1. On the Organization Settings page, click the Maintain Devices link to open the Device Configuration dialog for your organization.

  2. At the top of the dialog in the By Device Name field, type a ® console Device Name or type a User in the By User field and click the Go button to search by device or user name.

  3. The dialog contains the following tracking information for each downloaded and installed console instance.

    Device Name displays the name the agent assigned to the device during the initial installation. The device name must be unique by User ID.

    User displays the User ID of the agent who downloaded and installed the console.

    Install Date displays the date the console installation was successfully initialized.

    Last Active displays the last time the console application contacted the server.

    Device Type displays the type of device targeted (iOS, Android, Windows, etc.).

    Install Status displays whether the console is Activated or Deactivated.

  4. (Optional) Select the Wipe Device checkbox to signal a remote wipe of the device’s data and click Save. Then click Yes on the Save confirmation dialog to wipe all data from the user’s device.

  5. (Optional) Select the Reset Password checkbox to signal a remote reset of the Password data and click Save. Then click Yes on the Save confirmation dialog to force the user to reset their Password on next launch of the console.

  6. (Optional) In the right column of the grid on the Device Configuration dialog, click the red X to delete the device row from the grid. Next click Yes on the Delete Confirmation dialog.

  7. Click the Save button to save your entries.
  8. Click the X on the upper right to close the dialog.

  9. On the Organization Settings page, click the Save Organization button to save your entries.
