Enable Work Groups

  1. In the Admin tool, in the Admin tab under Organization Settings, select Allow Work Groups.
  2. From the toolbar menu, select Preferences, then click Manage Work Groups.

    • If Allow Creation of New Work Group is selected in the Groups view of the Admin tool, users can click Create New to create a new work group.

    • Users without the ability to create work groups can view their group access and change their notification settings.

    • When a work group is created, the user is added to the group as the Admin.

  3. Rename the work group and add a producer. The producer can be added by typing the name in the Producer field and clicking the Search icon.

    • The selected name from the User list populates the Producer field.

    • Once the Producer field has been filled, the Activities and Product tabs is populated.

  4. Add members to the group following the same steps.
  5. Select Read Only and Disable Notifications for each member and the activities and product types you want to share for this group.
  6. Click Save Work Group to save your changes.

See Preferences [User Instructions] for more information.
