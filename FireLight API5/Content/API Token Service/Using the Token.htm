Using the Token

For FireLight® API service endpoints that use the JWT token security scheme, the token string needs to be placed on any requests made to that service in the ‘Authorization’ header. The header needs to contain the ‘token_type’ value received from the response, followed by a space, followed by the ‘access_token’ value received from the response:

token_type + ‘ ‘ + access_token
