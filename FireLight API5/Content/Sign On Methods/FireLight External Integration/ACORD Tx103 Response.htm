ACORD Tx103 Response

An ACORD <TXLifeResponse> aggregate will be returned in response to the 103 request from the external system. Stored within the <HoldingKey> element will be the identifier to the application that will be created in FireLight® after step 7 in the sequence listed in Create New Application. This identifier can later be used to retrieve status information about the application, or to launch back into FireLight to edit the existing application at a later date. Refer to Edit Existing Application in FireLight on how to launch back into FireLight to edit an existing application.
