FireLight External Integration

Summary

FireLight® supports five possible integration scenarios from an external system:

  1. The creation of a new application from data submitted from the external system.

  2. The ability to update an existing application with new data from the external system.

  3. The ability to launch directly into FireLight to edit an existing application.

  4. The ability to launch directly into the FireLight dashboard.

  5. The ability to launch directly into the FireLight dashboard with retained use of 103 data.

The topics in this section describe in detail the steps an external system must take for each scenario.
