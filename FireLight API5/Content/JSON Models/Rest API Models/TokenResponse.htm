TokenResponse

Type: object

Properties:

  • access_token string

    This is the string text value of the token. It will need to be passed along when used, as defined in ‘Using the Token’ below.

  • expires_in integer

    This is the amount of time, in seconds, that the token will remain active. After this amount of time, the token is considered expired and no longer valid to be used.

  • token_type string

    This is the type of token. It needs to be passed along when the token is used, as defined in ‘Using the Token’ below. It is expected to return as “Bearer”.
