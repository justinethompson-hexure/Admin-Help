AvailableProductResponse

Type: object

Properties:

  • CUSIP: string

    String used to identify product.

  • ID: Guid

    ID Related to Request

  • ProductName: string

    Name of the Product

  • ProductOrganizationName: string

    The org name defined at the product level (if defined) or the org name in Organization profile

  • CarrierCode: string

    The Carrier associated with the product

  • Jurisdictions: List<long>

    List of Approved states

  • Locale: String

    The Locale of the product, i.e., en-US

  • Type: Int32

    The Type of the Product, i.e., Term, Life, etc.
