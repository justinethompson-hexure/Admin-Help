CompleteSignatureRequest

Type: object

Properties:

  • TeleSignAppId: String

    The ID of the Tele-Sign application.

  • ANI: String

    The “calling from” phone number.

  • DNIS: String

    The “calling to” phone number.

  • IVRId: String

    The IVR call record ID OR the contact ID.

  • FireLightAppId: Guid

    The ID of the application in FireLight. See Return Status list.

  • CarrierCode: String

    The Carrier Code of the application in FireLight.

  • CompletionDate: DateTime

    The date/time in UTC the signature process was completed (in ISO format).

  • Declined: bool

    Whether or not the client declined the signature.
