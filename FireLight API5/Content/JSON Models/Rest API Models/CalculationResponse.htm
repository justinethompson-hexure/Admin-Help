CalculationResponse

Type: Object

Properties:

  • Id string

    The unique ID of the request that was used for this response.

  • ResultCode integer

    A value from the Return Status list to indicate if the call was a success or not.

  • ExceptionError string

    The exception message from the server (null if no exception)

  • Messages ResponseMessage model object array

    A collection of ForeSight UI messages that is part of the response.

  • FireLightCalcOutputs array of OutputSet models

    The output set for FireLight specific outputs.

  • SystemVersion string

    Current version of the system
