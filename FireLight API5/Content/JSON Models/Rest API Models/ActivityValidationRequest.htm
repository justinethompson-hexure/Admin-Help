ActivityValidationRequest

Required:

  • CUSIP
  • Jurisdiction
  • AcordTypeCode
  • AcordSubTypeCode
  • CarrierCode

Optional

  • ReturnUpdatedDataItems

Properties:

  • CUSIP string

    Matches the CUSIP that is configured in the FireLight admin tool

  • Jurisdiction integer – view Jurisdictions list

    Identifies the jurisdiction type of the activity you are creating

  • AcordTypeCode integer – view AcordTypeCode & AcordSubTypeCode list

    Identifies the type of activity to create. Note that some activities require AcordSubTypeCode as well as AcordTypeCode.

  • AcordSubTypeCode integer – view AcordTypeCode & AcordSubTypeCode list

    Identifies the type of activity to create. Used in conjunction with AcordTypeCode to identify some activities.

  • CarrierCode string

    The three letter carrier code for the product specified by the CUSIP

  • ReturnUpdatedDataItems boolean

    Determines if the updated DataItems should be returned

  • DataItems array of ActivityDataItem

    A collection of data items that can be used to populate the wizard that is created during the request. The name of the data item should match the name of the element in the wizard.
