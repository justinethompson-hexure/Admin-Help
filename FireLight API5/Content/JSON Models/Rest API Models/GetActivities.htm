GetActivities

Type: object

Properties:

  • ActivityId string

    The unique identifier for the activity

  • TransactionType integer

    The type of transaction - view Transaction Type

  • Status integer

    The Status Types that the activity is in

  • Activity Name string

    The name of the activity

  • AuditDate string, format date

    Date of last activity update

  • DisplayStatusType Display Status

    The Display Status Types of the activity

  • DataItems array of IDataItemCollection

    Dictionary of DataItems

DataItem

Properties:

  • DataItemId string

    String identifi er for the dataitem

  • Value string

    Current value of the dataitem

  • IsEnabled boolean

    Whether dataitem is enabled for use and validation

  • IsReadonly boolean

    Whether control is locked

  • IsVisible boolean

    Whether dataitem is visible

  • CanBeDeleted boolean

    Whether dataitem can be removed due to rule execution

  • DataItemSource Source

    Source of dataitem creation

  • Unused

    Whether dataitem is available or no longer in use

  • Context

    Where in the workflow the dataitem was last updated by user

  • Version

    Version of FireLight
