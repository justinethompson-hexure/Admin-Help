ProductAvailabilityRequestModel

Type: object

Properties:

  • AcordType: string (AcordTypeCode & AcordSubTypeCode list)

    Filters the results to product types that match the given AcordTypeCode (optional). The value must be either the AcordTypeCode (i.e., “103”) or the AcordTypeCode and AcordSubTypeCode concatenated with a hyphen (i.e., “111-11101”).

  • ProductCarrierCode: string

    Limits the product results to those available for the given Product Carrier Code (optional)

  • JurisdictionIds: List<integer>

    Filters the results to products available for the supplied Jurisdictions IDs (optional)

  • RoleCode: string

    Filters the results to products available to the supplied Role (optional)
