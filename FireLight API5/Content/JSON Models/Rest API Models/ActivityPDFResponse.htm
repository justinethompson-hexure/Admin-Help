ActivityPDFResponse

Type: object

Properties:

  • ApplicationId string

    The activityId for the document

  • DocumentId guid

    The unique id of the document

  • DocumentType int (ActivityAttachmentDocumentType)

  • FormNumber string

    The form number for the document, if applicable

  • FormName string

    The form name for the document, if applicable

  • Parameter: DutyType

    The Duty Type for the document (Required/Optional)

  • PDFStream stream

    The stream containing the PDF document
