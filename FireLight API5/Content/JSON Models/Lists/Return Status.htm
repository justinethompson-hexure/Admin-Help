Return Status

Return Status
Return Status ID Description

RESULT_SUCCESS

1

The transaction was a success.

RESULT_SUCCESSINFO

2

The transaction was a success but some condition caused information to be returned. This information is usually informational or a warning.

RESULT_RECDPEND

3

Not used.

RESULT_RECDPENDINFO

4

Not used.

RESULT_FAILURE

5

The transaction was not successful and the reason for the failure was returned.
