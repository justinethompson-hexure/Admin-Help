ActivityBundle

Type: object

Properties:

  • FormGuid: string

    The unique ID of the underlying form or wizard.

  • FormIndex: int

    The index of the underlying form/wizard within the package.

  • BundleName: string

    The name of the bundle. Corresponds to the name of the form/wizard.

  • Pages: Array of BundlePage model

    The pages within the form/wizard

  • IsSelected: boolean

    Indicates if the bundle is currently selected. This is true if one of bundle’s pages is selected.

  • IsEnabled: boolean

    Indicates if the bundle is available to be selected. This is true unless the form/wizard is marked as excluded or hidden from data entry.

  • IsVisible: boolean

    Indicates if the bundle should be displayed to the user. This is true unless the form/wizard is marked as excluded or hidden from data entry.

  • HasErrors: boolean

    Indicates if any of the pages within the bundle have error messages associated with them.
