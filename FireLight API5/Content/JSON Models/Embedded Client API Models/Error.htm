Error

Type: object

Properties:

  • ApiSource: string

    The API method that threw this error

  • CorrelationID: string

    The ID of the item this error is correlated with

  • Environment: string

    The name of the environment (Token Service Endpoint) that this error was thrown in

  • ErrorMessage: string

    The descriptive text of the error message that was thrown

  • ErrorSource: string

    The API method name where this error was generated from

  • ErrorType: int

    The enum value of the error message that was generated
