SignatureStatusEvent

Type: object

Properties:

  • succeeded: boolean

    True if the application is complete and can proceed to gathering signatures

  • message: string

    Information regarding the attempt to proceed to gather signatures

  • redirectUrl: string

    URL of the address to gather signatures.

    NOTE: This URL is only usable when enableAuthProxy is set to false when calling Common API init function. When enableAuthProxy is true use the Token SSO sign on method to redirect to FireLight E-Signature.

The API method call that generated this event.
