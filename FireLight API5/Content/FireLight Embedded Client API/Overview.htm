Overview

The FireLight® embedded client API can be used to integrate FireLight wizards directly into your application. This functionality allows you to re-use FireLight wizards that you build in the FireLight Admin tool within your own application outside of the traditional FireLight frame. With our embedded client API, you are able to control how and where the wizards are embedded within your application. This is not a class iFrame solution. Our embedded client API uses our client side view engine to natively render the FireLight wizard for you into your user’s browser's document object model (DOM). This approach gives you the ability to brand the wizard using CSS and subscribe to events so that your application can react to actions within the wizard.
