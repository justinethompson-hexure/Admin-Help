Font Awesome Installation

In order for icons to show in embedded FireLight®, the Font Awesome icon library must be present on the hosting page. FireLight uses the Pro version of Font Awesome 5. Embedded users may load the Font Awesome 5 Pro stylesheet directly from FireLight, but the icons may only be used in the FireLight embedded application – hosting applications are prohibited from using Font Awesome 5 Pro icons without purchasing their own license.

If the hosting application already owns Font Awesome 5 Pro, you may continue to use your existing stylesheet and fonts. As long as they are loaded on the hosting page, FireLight can make use of them. If a hosting application would like to use Font Awesome 5 Pro icons outside of the FireLight embedded application, the hosting application must purchase a license directly from Font Awesome at https://fontawesome.com/plans.

In order to show Font Awesome icons in FireLight embedded, please utilize the following stylesheet (font files will be loaded automatically as needed):

Stylesheet URL: {Base URL}/Content/FontAwesome/css/all.css

Example: https://firelight.insurancetechnologies.com/EGApp/Content/FontAwesome/css/all.css
