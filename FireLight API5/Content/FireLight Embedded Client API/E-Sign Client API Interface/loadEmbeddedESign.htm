loadEmbeddedESign

Initializes embedded mode for a given activity for E-Signature. This method assumes the specified activity has already been created and is ready for E-Signature. loadEmbeddedESign is only supported when the eSign resource is loaded. See Getting Started and loadModule to see how to load multiple module resources.

NOTE: If resources for multiple modules are loaded, loadModule must be called pior in order to load the correct module.

Parameters

  • activityId: string

    The GUID of the activity to be loaded.

Returns: void
