navigateToPageNext

This API loads up the client's application and navigates to the next page if one exists. If one does not exist, the button directs to the last possible page in the application. Navigates to the specified page within the current activity.

Parameters

Returns void

The navigateToPageNext and navigateToPagePrevious API calls are intended to be used with client's own Previous/Next navigation buttons. To gain a feel of how these buttons will operate, the client can use the regular FireLight® process and experiment with the FireLight existing Previous/Next buttons.

NOTE:

  • “last possible page” means the last page on the application that has a control on it.

  • “first possible page” means the first page on the application that has a control on it.

If the client would like to navigate to a specific page, use the existing navigateToPage API and simply pass in the specific page the users would like to navigate to.
