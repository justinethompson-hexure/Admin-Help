loadModule

Initializes the specified module. This method is only needed when multiple module resources are loaded. It must be called prior to calling init or loadEmbeddedESign.

Parameters

  • module: string

    The string returned from the API Token Service that uniquely identifies the current call to the client API (for tracking purposes). The string that uniquely identifies the current call to the client API (for tracking purposes).

    Valid values: EApp, illustration, eSign
