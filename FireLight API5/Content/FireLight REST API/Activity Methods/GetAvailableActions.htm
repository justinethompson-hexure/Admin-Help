GetAvailableActions

HTTP GET to query for a list of available actions for an application for the user associated to the auth token (IS4) that was passed in the header.

Requires an API Token with an associated user.

URL

{rootURL}/Activity/GetAvailableActions

Parameters

  • activityId: string *required

    The unique ID of the activity to get the actions for

Response

The GetAvailableActions method returns an AvailableActionsReturn model
