RemoveDocument

 

Endpoint:

api/Activity/RemoveDocument

Parameters:

Id: Required. The activity ID.

Documents: Required. A comma-separated list of DocumentQuery items to specify which documents should be removed.

DocumentQuery Structure

DocumentId: Optional. The internal ID of the document (as retrieved from GetDocuments)

DocumentName: Optional. The Document Type Name. E.G. Illustration, Other

DocumentType: Optional. The Document Type as ID.

DocumentTag: Optional. The Document Tag.

All DocumentQuery parameters are optional, and only documents that match all specified attributes will be removed. For example, DocumentType: “Optional”, DocumentTag: “Additional” will remove only documents of type Other with the tag “Additional”.

Example JSON:

{'Id': '3d2f0a0a-3b0a-4889-82d1-7f0e429984bb', Documents: [{DocumentName: 'Illustration'}, {DocumentTag: 'MyTag'}]}

This will remove all documents of type Illustration and all documents with the tag ‘MyTag’ from the application.
