GetIllustrationPDF

/Illustration/GetIllustrationPDF

HTTP post that generates the illustration output for the specified JSON input.

Requires API Token, but does not need an API Token with an associated user.

Request body GetIllustrationRequest Model *required

A list of data items used for the calculations along with a unique ID to identify the request

Response

The GetIllustrationPDF method returns a ReportResponse model. Model is returned. If the request could not generate calculation output due to errors from running FireLight rules, you will receive a CalculationErrorResult Model back.
